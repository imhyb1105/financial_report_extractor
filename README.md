# 财税小工具

> 中国现行18个税种在线计算器

## 项目简介

财税小工具是一个免费、专业、便捷的在线税务计算工具集合，涵盖中国现行全部18个税种的计算器，帮助企业和个人快速准确地计算各类税费。

## 功能特点

- **18个税种全覆盖**：增值税、企业所得税、个人所得税、消费税、印花税等
- **即时计算**：无需安装，打开即用，结果即时呈现
- **知识库配套**：每个税种配备完整的知识库文档
- **多端适配**：支持电脑、平板、手机等设备
- **政策更新**：及时跟进最新税收政策

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/your-username/tax-calculators.git
```

2. 打开项目目录
```bash
cd tax-calculators
```

3. 启动本地服务器（可选）
```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js 的 serve
npx serve
```

4. 浏览器访问 `http://localhost:8080`

## 部署说明

本项目使用 Vercel 自动部署，连接 GitHub 仓库后，每次推送到主分支都会自动部署。

### 部署步骤

1. Fork 本仓库到你的 GitHub 账户
2. 登录 [Vercel](https://vercel.com)
3. 点击 "Import Project"
4. 选择你 Fork 的仓库
5. 点击 "Deploy" 完成部署

## 更新维护

### 税率更新

每个计算器都有独立的配置文件 `{税种}.config.json`，修改税率时只需更新配置文件中的相关数值。

配置文件包含以下字段：
- `version`: 版本号
- `effective_date`: 生效日期
- `last_verified_date`: 最后验证日期
- `needs_verification`: 是否需要验证
- `policy_evidence`: 政策来源信息

### 版本规范

版本号格式：`YYYY.MM-序号`

例如：`2026.03-001` 表示2026年3月第1个版本
