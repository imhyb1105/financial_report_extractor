# 个人所得税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向个人和企业的个人所得税计算器网页应用，帮助用户快速计算各类所得应缴纳的个人所得税。

## 2. 用户画像 (Persona)

**主要用户**：企业HR、财务人员、个人纳税人、税务代理人员

**核心痛点**：
- 个人所得税税目多，计算方式各异
- 综合所得年度汇算复杂
- 专项附加扣除项目繁多
- 全年一次性奖金计税方式选择

## 3. 功能需求

### 3.1 核心功能

#### 所得类型分类（Tab切换）
- 综合所得（工资薪金）
- 全年一次性奖金
- 经营所得
- 劳务报酬/稿酬/特许权使用费
- 其他所得

### 3.4 数据输入/输出

**输入数据**：
- 所得类型（必选）
- 收入金额（必填）
- 五险一金（如适用）
- 专项附加扣除（如适用）

**输出数据**：
- 应纳税所得额
- 适用税率
- 应纳税额
- 计算明细

## 5. 计算公式

### 综合所得（工资薪金）
```
应纳税所得额 = 年收入 - 60000 - 五险一金 - 专项附加扣除 - 其他扣除
应纳税额 = 应纳税所得额 × 适用税率 - 速算扣除数
```

### 全年一次性奖金
```
应纳税额 = 全年一次性奖金 × 适用税率 - 速算扣除数
（或并入综合所得计算）
```

### 劳务报酬
```
预扣预缴应纳税所得额 = 收入 × (1 - 20%)
应纳税额 = 应纳税所得额 × 适用税率 - 速算扣除数
```
