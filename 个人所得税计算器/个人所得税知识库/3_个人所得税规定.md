# 个人所得税规定

**版本：2026.03-001**

---

## 一、基本规定

### 1.1 纳税义务发生时间

| 所得类型 | 纳税义务发生时间 |
|----------|------------------|
| 工资薪金 | 实际支付日期 |
| 劳务报酬 | 实际取得日期 |
| 稿酬 | 实际取得日期 |
| 特许权使用费 | 实际取得日期 |
| 利息股息红利 | 实际取得日期 |
| 财产租赁 | 实际取得日期 |
| 财产转让 | 交易完成日期 |
| 偶然所得 | 实际取得日期 |

### 1.2 纳税期限

| 类型 | 期限要求 |
|------|----------|
| 扣缴申报 | 次月15日内 |
| 经营所得预缴 | 季度终了后15日内 |
| 综合所得汇算清缴 | 次年3月1日至6月30日 |
| 经营所得汇算清缴 | 次年3月31日前 |

## 二、专项扣除

### 2.1 专项扣除范围

专项扣除包括居民个人按照国家规定的范围和标准缴纳的基本养老保险、基本医疗保险、失业保险等社会保险费和住房公积金等。

### 2.2 扣除标准

| 扣除项目 | 扣除标准 | 备注 |
|----------|----------|------|
| 基本养老保险 | 个人缴纳部分 | 按实际缴纳金额 |
| 基本医疗保险 | 个人缴纳部分 | 按实际缴纳金额 |
| 失业保险 | 个人缴纳部分 | 按实际缴纳金额 |
| 住房公积金 | 个人缴纳部分 | 按实际缴纳金额 |

### 2.3 缴纳比例上限

| 保险类型 | 单位缴纳比例上限 | 个人缴纳比例上限 |
|----------|------------------|------------------|
| 养老保险 | 16% | 8% |
| 医疗保险 | 10% | 2% |
| 失业保险 | 0.5-2% | 0.5-1% |
| 公积金 | 12% | 12% |

**注**：具体比例以各地规定为准，表中为一般上限

## 三、专项附加扣除

### 3.1 专项附加扣除项目（7项）

| 序号 | 扣除项目 | 扣除标准/月 | 扣除主体 |
|------|----------|-------------|----------|
| 1 | 子女教育 | 2,000元/孩 | 父母各50%或一方100% |
| 2 | 继续教育 | 400元或3,600元/年 | 本人 |
| 3 | 大病医疗 | 最高80,000元/年 | 本人或配偶 |
| 4 | 住房贷款利息 | 1,000元 | 本人或配偶 |
| 5 | 住房租金 | 800-1,500元 | 本人 |
| 6 | 赡养老人 | 3,000元 | 独生子女全额/非独生分摊 |
| 7 | 3岁以下婴幼儿照护 | 2,000元/婴 | 父母各50%或一方100% |

### 3.2 各项扣除详细规定

#### 3.2.1 子女教育

| 条件 | 标准 |
|------|------|
| 适用范围 | 学前教育（3岁起）、学历教育（小学至博士） |
| 扣除标准 | 2,000元/月/孩 |
| 扣除方式 | 父母各扣50%或一方扣除100% |
| 时间范围 | 接受教育期间 |

#### 3.2.2 继续教育

| 类型 | 标准 | 期限 |
|------|------|------|
| 学历继续教育 | 400元/月 | 最长48个月 |
| 职业资格继续教育 | 3,600元/年 | 取得证书当年 |

#### 3.2.3 大病医疗

| 条件 | 标准 |
|------|------|
| 起扣线 | 医保目录范围内自付部分超过15,000元 |
| 扣除限额 | 最高80,000元/年 |
| 扣除范围 | 本人、配偶、未成年子女 |
| 医保范围 | 需在医保目录范围内 |

#### 3.2.4 住房贷款利息

| 条件 | 标准 |
|------|------|
| 适用范围 | 首套住房贷款 |
| 扣除标准 | 1,000元/月 |
| 扣除期限 | 最长240个月 |
| 扣除主体 | 夫妻双方约定由一方扣除 |

#### 3.2.5 住房租金

| 城市 type | 扣除标准 |
|-----------|----------|
| 直辖市、省会（首府）城市、计划单列市以及国务院确定的其他城市 | 1,500元/月 |
| 市辖区户籍人口超过100万的城市 | 1,100元/月 |
| 市辖区户籍人口不超过100万的城市 | 800元/月 |

**注意**：
- 纳税人及配偶在主要工作城市无自有住房
- 不得与住房贷款利息扣除同时享受

#### 3.2.6 赡养老人

| 条件 | 标准 |
|------|------|
| 适用范围 | 赡养60岁（含）以上父母，或子女均已去世的祖父母、外祖父母 |
| 扣除标准 | 3,000元/月 |
| 独生子女 | 全额扣除 |
| 非独生子女 | 兄弟姐妹分摊，每人最高1,500元/月 |

#### 3.2.7 3岁以下婴幼儿照护

| 条件 | 标准 |
|------|------|
| 适用范围 | 照护3岁以下婴幼儿 |
| 扣除标准 | 2,000元/月/婴 |
| 扣除方式 | 父母各扣50%或一方扣除100% |

## 四、其他扣除

### 4.1 企业年金和职业年金

| 项目 | 扣除标准 |
|------|----------|
| 企业年金 | 个人缴纳部分，不超过本人缴费工资计税基数的4% |
| 职业年金 | 个人缴纳部分，不超过本人缴费工资计税基数的4% |

### 4.2 商业健康保险

| 条件 | 标准 |
|------|------|
| 适用产品 | 符合规定的商业健康保险产品 |
| 扣除标准 | 2,400元/年（200元/月） |

### 4.3 税延型商业养老保险

| 条件 | 标准 |
|------|------|
| 适用范围 | 试点地区 |
| 扣除标准 | 当月工资薪金、连续性劳务报酬收入的6%和1,000元孰低 |

### 4.4 个人养老金

| 条件 | 标准 |
|------|------|
| 适用范围 | 参加人 |
| 缴费限额 | 12,000元/年 |
| 税收优惠 | 缴费环节税前扣除，投资环节免税，领取环节按3%计税 |

### 4.5 公益慈善捐赠

| 条件 | 标准 |
|------|------|
| 扣除限额 | 不超过应纳税所得额的30% |
| 扣除凭证 | 符合规定的公益捐赠票据 |

## 五、免税规定

### 5.1 法定免税项目

下列各项个人所得，免征个人所得税：

| 序号 | 免税项目 |
|------|----------|
| 1 | 省级人民政府、国务院部委、中国人民解放军军以上单位，以及外国组织、国际组织颁发的科学、教育、技术、文化、卫生、体育、环境保护等方面的奖金 |
| 2 | 国债和国家发行的金融债券利息 |
| 3 | 按照国家统一规定发给的补贴、津贴 |
| 4 | 福利费、抚恤金、救济金 |
| 5 | 保险赔款 |
| 6 | 军人的转业费、复员费、退役金 |
| 7 | 按照国家统一规定发给干部、职工的安家费、退职费、基本养老金或者退休费、离休费、离休生活补助费 |
| 8 | 依照有关法律规定应予免税的各国驻华使馆、领事馆的外交代表、领事官员和其他人员的所得 |
| 9 | 中国政府参加的国际公约、签订的协议中规定免税的所得 |
| 10 | 国务院规定的其他免税所得 |

### 5.2 暂免征税项目

| 序号 | 暂免项目 |
|------|----------|
| 1 | 外籍个人以非现金形式或实报实销形式取得的住房补贴、伙食补贴、搬迁费、洗衣费 |
| 2 | 外籍个人按合理标准取得的境内、外出差补贴 |
| 3 | 外籍个人取得的探亲费、语言训练费、子女教育费等 |
| 4 | 个人举报、协查各种违法、犯罪行为而获得的奖金 |
| 5 | 个人办理代扣代缴税款手续，按规定取得的扣缴手续费 |
| 6 | 个人转让自用达5年以上、并且是唯一的家庭居住用房取得的所得 |
| 7 | 外籍个人从外商投资企业取得的股息、红利所得 |
| 8 | 股票转让所得 |
| 9 | 储蓄存款利息所得（2008年10月9日后） |

## 六、减税规定

### 6.1 法定减征项目

| 情形 | 减征幅度 |
|------|----------|
| 残疾、孤老人员和烈属的所得 | 具体幅度由省级人民政府规定 |
| 因自然灾害遭受重大损失的 | 具体幅度由省级人民政府规定 |

### 6.2 地方性减征规定

各省级人民政府可根据本地实际情况，在法定幅度内制定具体减征标准。

---

## 七、申报规定

### 7.1 代扣代缴

| 项目 | 规定 |
|------|------|
| 扣缴义务人 | 支付所得的单位或个人 |
| 扣缴时限 | 支付所得时代扣 |
| 申报时限 | 次月15日内 |
| 扣缴手续费 | 按扣缴税款的2%支付 |

### 7.2 自行申报

需要自行申报的情形：

| 序号 | 情形 |
|------|------|
| 1 | 取得综合所得需要办理汇算清缴 |
| 2 | 取得经营所得 |
| 3 | 取得应税所得没有扣缴义务人 |
| 4 | 取得应税所得，扣缴义务人未扣缴税款 |
| 5 | 取得境外所得 |
| 6 | 因移居境外注销中国户籍 |
| 7 | 非居民个人在中国境内从两处以上取得工资、薪金所得 |

### 7.3 综合所得汇算清缴

| 项目 | 规定 |
|------|------|
| 办理时间 | 次年3月1日至6月30日 |
| 办理方式 | 自行办理、单位办理、委托办理 |
| 免办条件 | 年收入≤12万元且补税金额≤400元 |

## 八、法律责任

### 8.1 逾期申报

| 情形 | 处罚标准 |
|------|----------|
| 逾期申报 | 责令限期改正，可处2,000元以下罚款 |
| 情节严重 | 处2,000元以上10,000元以下罚款 |

### 8.2 偷税漏税

| 情形 | 处罚标准 |
|------|----------|
| 偷税 | 不缴或少缴税款50%以上5倍以下罚款 |
| 构成犯罪 | 依法追究刑事责任 |

### 8.3 扣缴义务人责任

| 情形 | 处罚标准 |
|------|----------|
| 应扣未扣 | 应扣未扣税款50%以上3倍以下罚款 |
