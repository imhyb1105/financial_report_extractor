# 个人所得税计算方法

**版本：2026.03-001**

---

## 一、综合所得计算

### 1.1 年度汇算计算公式

```
应纳税额 = (综合所得收入额 - 60,000 - 专项扣除 - 专项附加扣除 - 其他扣除) × 税率 - 速算扣除数
```

### 1.2 综合所得收入额计算

| 所得类型 | 收入额计算 |
|----------|------------|
| 工资薪金 | 全额计入 |
| 劳务报酬 | 收入 × (1-20%)（收入>4000时）；收入 - 800（收入≤4000时） |
| 稿酬 | 收入 × (1-20%) × 70% |
| 特许权使用费 | 收入 × (1-20%)（收入>4000时）；收入 - 800（收入≤4000时） |

### 1.3 计算步骤

**第一步**：计算各项所得的收入额
```
综合所得收入额 = 工资薪金 + 劳务报酬收入额 + 稿酬收入额 + 特许权使用费收入额
```

**第二步**：计算应纳税所得额
```
应纳税所得额 = 综合所得收入额 - 60,000 - 专项扣除 - 专项附加扣除 - 其他扣除
```

**第三步**：计算应纳税额
```
应纳税额 = 应纳税所得额 × 适用税率 - 速算扣除数
```

**第四步**：计算应补/退税额
```
应补/退税额 = 应纳税额 - 已预缴税额
```

## 二、工资薪金预扣预缴计算

### 2.1 累计预扣法公式

```
本期应扣缴税额 = (累计收入 - 累计免税收入 - 累计减除费用 - 累计专项扣除 - 累计专项附加扣除 - 累计其他扣除) × 税率 - 速算扣除数 - 累计已预缴税额
```

## 三、劳务报酬预扣预缴计算

### 3.1 预扣预缴公式

**收入>4,000元时**：
```
应预扣预缴税额 = 收入 × (1-20%) × 预扣率 - 速算扣除数
```

**收入≤4,000元时**：
```
应预扣预缴税额 = (收入 - 800) × 20%
```

### 3.2 预扣预缴税率表

| 预扣预缴应纳税所得额 | 预扣率 | 速算扣除数 |
|----------------------|--------|------------|
| 不超过20,000元 | 20% | 0 |
| 超过20,000元至50,000元 | 30% | 2,000 |
| 超过50,000元 | 40% | 7,000 |

### 3.3 计算示例

**例1**：劳务报酬5,000元
```
应纳税所得额 = 5,000 × (1-20%) = 4,000元
预扣预缴税额 = 4,000 × 20% = 800元
```

**例2**：劳务报酬30,000元
```
应纳税所得额 = 30,000 × (1-20%) = 24,000元
预扣预缴税额 = 24,000 × 30% - 2,000 = 5,200元
```

**例3**：劳务报酬100,000元
```
应纳税所得额 = 100,000 × (1-20%) = 80,000元
预扣预缴税额 = 80,000 × 40% - 7,000 = 25,000元
```

```

**例2**：稿酬20,000元
```
应预扣预缴税额 = 20,000 × (1-20%) × 70% × 20% = 2,240元

**例3**：稿酬100,000元

## 六、经营所得计算

### 6.1 计算公式

```
应纳税额 = (收入总额 - 成本 - 费用 - 损失) × 税率 - 速算扣除数
```

### 6.2 经营所得税率表

| 全年应纳税所得额 | 税率 | 速算扣除数 |
|------------------|------|------------|
| 不超过30,000元 | 5% | 0 |
| 超过30,000元至90,000元 | 10% | 1,500 |
| 超过90,000元至300,000元 | 20% | 10,500 |
| 超过300,000元至500,000元 | 30% | 40,500 |
| 超过500,000元 | 35% | 65,500 |

### 6.3 计算示例

**例**：某个体工商户年度收入500,000元，成本费用合计300,000元
```
应纳税所得额 = 500,000 - 300,000 = 200,000元
应纳税额 = 200,000 × 20% - 10,500 = 29,500元
```

## 七、分类所得计算

### 7.1 利息、股息、红利所得

```
应纳税额 = 收入 × 20%
```

**例**：股息收入10,000元
```
应纳税额 = 10,000 × 20% = 2,000元
```

### 7.2 财产租赁所得

**收入>4,000元时**：
```
应纳税额 = 收入 × (1-20%) × 20%（出租住房减按10%）
```

**收入≤4,000元时**：
```
应纳税额 = (收入 - 800) × 20%（出租住房减按10%）
```

**例1**：出租住房月租金3,000元
```
应纳税额 = (3,000 - 800) × 10% = 220元
```

**例2**：出租商铺月租金10,000元
```
应纳税额 = 10,000 × (1-20%) × 20% = 1,600元
```

### 7.3 财产转让所得

```
应纳税额 = (转让收入 - 财产原值 - 合理费用) × 20%
```

**例**：转让股权收入200,000元，原值100,000元，费用5,000元
```
应纳税额 = (200,000 - 100,000 - 5,000) × 20% = 19,000元
```

### 7.4 偶然所得

```
应纳税额 = 收入 × 20%
```

**例**：中奖收入50,000元
```
应纳税额 = 50,000 × 20% = 10,000元
```

## 八、综合所得汇算清缴计算实例

### 8.1 基本情况

- 纳税人：李四
- 2025年度综合所得构成：
  - 工资薪金：240,000元
  - 劳务报酬：50,000元
  - 稿酬：20,000元
- 专项扣除（三险一金）：36,000元
- 专项附加扣除：48,000元
  - 子女教育：12,000元
  - 住房贷款利息：12,000元
  - 赡养老人：24,000元
- 已预缴税款：15,800元

### 8.2 计算过程

**第一步：计算综合所得收入额**
```
工资薪金收入额 = 240,000元
劳务报酬收入额 = 50,000 × (1-20%) = 40,000元
稿酬收入额 = 20,000 × (1-20%) × 70% = 11,200元
综合所得收入额 = 240,000 + 40,000 + 11,200 = 291,200元
```

**第二步：计算应纳税所得额**
```
应纳税所得额 = 291,200 - 60,000 - 36,000 - 48,000 = 147,200元
```

**第三步：计算应纳税额**
```
适用税率：第3级（超过144,000元至300,000元的部分），税率20%
应纳税额 = 147,200 × 20% - 16,920 = 12,520元
```

**第四步：计算应退税额**
```
应退税额 = 15,800 - 12,520 = 3,280元
```

**结论**：李四年度汇算可退税3,280元
