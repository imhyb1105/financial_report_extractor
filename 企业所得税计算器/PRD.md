# 企业所得税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向财务人员的企业所得税计算器网页应用，**核心解决应纳税所得额的计算难点**，通过分步向导模式引导用户完成纳税调整，提供限额扣除项目的自动计算功能，帮助财务人员快速准确地进行税额计算。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、税务专员、会计师

**核心痛点**：
- 应纳税所得额 ≠ 会计利润，需要复杂的纳税调整
- 限额扣除项目计算规则多，容易遗漏或计算错误
- 不同扣除项目的超限处理方式不同（可结转 vs 不可结转）

## 3. 功能需求

### 3.1 核心功能：5步向导模式

#### 第1步：基础数据
- 销售(营业)收入
- 会计利润总额
- 工资薪金总额
- 从业人数（用于判断小微企业）
- 资产总额（用于判断小微企业）

#### 第2步：限额扣除（自动计算）
| 项目 | 计算规则 | 超限处理 |
|------|---------|---------|
| 业务招待费 | Min(60%, 5‰) | 不结转 |
| 广告费 | 15%（特殊行业30%） | 可结转 |
| 职工福利费 | 14% | 不结转 |
| 工会经费 | 2% | 不结转 |
| 职工教育经费 | 8% | 可结转 |
| 公益性捐赠 | 12% | 结转3年 |

#### 第3步：纳税调整
- **不得扣除项目**：罚款、滞纳金、赞助支出等（全额调增）
- **免税/不征税收入**：国债利息、股息红利、财政拨款等（全额调减）

#### 第4步：优惠扣除
- 研发费用加计扣除（100%）
- 残疾人工资加计扣除（100%）
- 以前年度亏损弥补
- 企业类型选择

#### 第5步：计算结果
- 应纳税所得额
- 应纳企业所得税
- 纳税调整明细表
- 计算公式展示

### 3.2 企业类型支持

| 企业类型 | 税率 | 特点 |
|---------|------|------|
| 一般企业 | 25% | 法定税率 |
| 小型微利企业 | 5%（实际税负） | 需满足条件，有效期至2027年 |
| 高新技术企业 | 15% | 需经认定 |

### 3.3 数据输入/输出

**输入数据**：
- 销售(营业)收入（必填）
- 会计利润总额（必填）
- 工资薪金总额（必填）
- 各类扣除项目实际发生额（选填）
- 企业类型（必选）

**输出数据**：
- 应纳税所得额（含计算明细）
- 应纳企业所得税
- 纳税调整汇总表
- 政策依据说明

## 5. 界面设计

### 5.1 主要界面元素
- 5步进度导航条
- 分步表单区域
- 限额扣除实时计算卡片
- 结果展示区域（含明细表）

### 5.2 界面特点
- 分步引导，降低认知负担
- 限额扣除自动计算，减少手动计算错误
- 清晰的调增/调减区分（红色/绿色）
- 支持打印结果

## 6. 计算公式

### 应纳税所得额计算
```
应纳税所得额 = 会计利润总额
            + 限额扣除超限调增
            + 不得扣除项目调增
            - 免税/不征税收入调减
            - 加计扣除
            - 亏损弥补
```

### 业务招待费限额
```
扣除限额 = Min(实际发生额 × 60%, 销售(营业)收入 × 5‰)
```

### 小型微利企业（2023-2027）
```
应纳税额 = 应纳税所得额 × 5%（统一实际税负）
```

## 7. 政策依据

- 《中华人民共和国企业所得税法》
- 《中华人民共和国企业所得税法实施条例》
- 财政部 税务总局公告2023年第6号（小微企业所得税优惠）
- 财政部 税务总局公告2023年第7号（研发费用加计扣除100%）

## 8. 测试用例

### 测试场景1：业务招待费限额扣除
- 销售收入：1000万元
- 业务招待费：10万元
- 预期：限额5万元（取60%=6万与5‰=5万中较低值）

### 测试场景2：小型微利企业判断
- 应纳税所得额：250万元
- 从业人数：50人
- 资产总额：1000万元
- 预期：符合小微企业条件，税额 = 250 × 5% = 12.5万元

### 测试场景3：超过小微企业临界点
- 应纳税所得额：301万元
- 预期：不符合小微企业条件，全部按25%计算 = 75.25万元
