# 企业所得税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向财务人员的企业所得税计算器网页应用，提供简洁实用的界面，支持一般企业、小型微利企业、高新技术企业等不同类型的企业所得税计算，帮助财务人员快速准确地进行税额计算和筹划。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、税务专员、会计师，需要快速计算企业所得税应纳税额，了解不同企业类型的税负差异。

## 3. 功能需求

### 3.1 核心功能

1. **一般企业所得税计算**
   - 应纳税所得额计算
   - 应纳税额计算
   - 适用税率：25%

2. **小型微利企业所得税计算**
   - 年应纳税所得额 ≤ 100万元：实际税负 **2.5%**
   - 100万元 < 年应纳税所得额 ≤ 300万元：实际税负 **5%**
   - 自动判断是否符合小微企业条件
   - 政策有效期至2027年12月31日

3. **高新技术企业所得税计算**
   - 适用税率：15%
   - 亏损弥补年限延长至10年
   - 需经认定的高新技术企业

4. **研发费用加计扣除**
   - 研发费用100%加计扣除
   - 自动计算加计扣除金额

5. **亏损弥补**
   - 一般企业：5年内弥补
   - 高新技术企业：10年内弥补

### 3.2 数据输入/输出

**输入数据**：
- 收入总额（必填）
- 成本费用总额（必填）
- 企业类型（必选：一般企业/小型微利/高新技术企业）
- 研发费用（选填，用于加计扣除）
- 公益性捐赠（选填）
- 以前年度亏损（选填）

**输出数据**：
- 应纳税所得额
- 适用税率
- 应纳税额
- 税负率
- 计算依据说明

## 6. 计算公式参考

### 基本计算
```
应纳税所得额 = 收入总额 - 成本费用总额 ± 纳税调整
应纳税额 = 应纳税所得额 × 适用税率
```

### 小型微利企业（2023-2027）
```
≤100万元：应纳税额 = 应纳税所得额 × 25% × 20% × 50% = 应纳税所得额 × 2.5%
100万-300万元：应纳税额 = 应纳税所得额 × 25% × 20% = 应纳税所得额 × 5%
```

### 研发费用加计扣除
```
加计扣除金额 = 研发费用 × 100%
调整后应纳税所得额 = 应纳税所得额 - 加计扣除金额
```

## 7. 政策依据

- 《中华人民共和国企业所得税法》
- 《中华人民共和国企业所得税法实施条例》（2024年修订）
- 财政部 税务总局公告2023年第6号（小微企业所得税优惠）
- 财政部 税务总局公告2023年第7号（研发费用加计扣除）
