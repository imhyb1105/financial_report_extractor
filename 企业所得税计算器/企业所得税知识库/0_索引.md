## ⚠️ 重要提示

企业所得税计算的核心难点是**应纳税所得额的计算**，而非简单的税率应用。

```
应纳税所得额 = 会计利润
            + 限额扣除超限调增
            + 不得扣除项目调增
            - 免税/不征税收入调减
            - 加计扣除
            - 亏损弥补
```

建议按照计算器分步向导依次填写，确保纳税调整完整准确。
