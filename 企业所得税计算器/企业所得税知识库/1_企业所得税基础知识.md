# 企业所得税基础知识

**版本号：2026.03-001**

---

## 一、企业所得税的定义

企业所得税是对中国境内企业和其他取得收入的组织的生产经营所得和其他所得征收的一种所得税。

**法律依据**：《中华人民共和国企业所得税法》第一条

> 在中华人民共和国境内，企业和其他取得收入的组织（以下统称企业）为企业所得税的纳税人，依照本法的规定缴纳企业所得税。

---

## 二、纳税义务人

### 1. 居民企业

根据《企业所得税法》第二条规定，居民企业包括：

| 类型 | 定义 |
|------|------|
| 境内注册企业 | 依法在中国境内成立的企业 |
| 境外注册但实际管理在境内 | 依照外国（地区）法律成立但实际管理机构在中国境内的企业 |

**纳税义务**：就来源于中国**境内、境外**的全部所得缴纳企业所得税

### 2. 非居民企业

| 类型 | 定义 | 纳税义务 |
|------|------|----------|
| 设立机构场所 | 在中国境内设立机构、场所的企业 | 就其所设机构、场所取得的所得，以及发生在中国境外但与其所设机构、场所有实际联系的所得纳税 |
| 未设立机构场所 | 在中国境内未设立机构、场所，但有来源于中国境内所得的企业 | 就来源于中国境内的所得纳税 |

### 3. 不适用企业所得税的主体

根据《企业所得税法》第一条第二款：

- **个人独资企业**
- **合伙企业**

上述企业不适用《企业所得税法》，其投资者按"经营所得"缴纳个人所得税。

## 三、征税对象与范围

### 1. 所得类型及来源地判定

根据《企业所得税法实施条例》第六条至第十条：

| 所得类型 | 来源地确定原则 |
|----------|----------------|
| 销售货物所得 | 交易活动发生地 |
| 提供劳务所得 | 劳务发生地 |
| 转让财产所得（不动产） | 不动产所在地 |
| 转让财产所得（动产） | 转让动产的企业所在地 |
| 转让财产所得（权益性投资资产） | 被投资企业所在地 |
| 股息、红利等权益性投资所得 | 分配所得的企业所在地 |
| 利息所得 | 负担、支付所得的企业所在地 |
| 租金所得 | 负担、支付所得的企业所在地 |
| 特许权使用费所得 | 负担、支付所得的企业所在地 |

### 2. 不征税收入

根据《企业所得税法》第七条：

| 类型 | 说明 |
|------|------|
| 财政拨款 | 各级人民政府对纳入预算管理的事业单位、社会团体等组织拨付的财政资金 |
| 行政事业性收费 | 依法收取并纳入财政管理的行政事业性收费 |
| 政府性基金 | 依法收取并纳入财政管理的政府性基金 |
| 其他 | 国务院规定的其他不征税收入 |

### 3. 免税收入

根据《企业所得税法》第二十六条：

| 类型 | 说明 |
|------|------|
| 国债利息收入 | 持有国务院财政部门发行的国债取得的利息收入 |
| 居民企业间股息红利 | 符合条件的居民企业之间的股息、红利等权益性投资收益 |
| 非居民企业股息红利 | 在中国境内设立机构、场所的非居民企业从居民企业取得与该机构、场所有实际联系的股息、红利等权益性投资收益 |
| 非营利组织收入 | 符合条件的非营利组织的收入 |

### 4. 不征税收入与免税收入的区别

| 项目 | 不征税收入 | 免税收入 |
|------|------------|----------|
| 性质 | 不属于征税范围 | 属于征税范围但给予免税 |
| 对应支出 | **不得扣除** | 可以扣除 |
| 形成资产的折旧摊销 | 不得扣除 | 可以扣除 |

## 四、纳税义务发生时间

根据《企业所得税法》第五十三条、第五十四条：

| 情形 | 纳税年度 |
|------|----------|
| 正常经营 | 公历1月1日起至12月31日止 |
| 年度中间开业 | 以其实际经营期为一个纳税年度 |
| 终止经营活动 | 以其实际经营期为一个纳税年度 |
| 依法清算 | 以清算期间作为一个纳税年度 |

---

## 五、纳税地点

根据《企业所得税法》第五十条至第五十二条：

| 企业类型 | 纳税地点 |
|----------|----------|
| 居民企业 | 企业登记注册地（登记注册地在境外的，为实际管理机构所在地） |
| 居民企业设立不具有法人资格的营业机构 | 汇总计算并缴纳企业所得税 |
| 非居民企业在中国境内设立机构、场所 | 机构、场所所在地 |
| 非居民企业在中国境内设立两个以上机构、场所 | 经批准可选择由其主要机构、场所汇总缴纳 |
| 非居民企业未在中国境内设立机构、场所 | 扣缴义务人所在地 |

## 六、征收方式

### 1. 查账征收

适用于财务制度健全、能准确核算收入和成本费用的企业。

### 2. 核定征收

适用于账簿不健全、无法准确核算的企业，包括：

- **定额征收**：直接核定应纳税额
- **核定应税所得率征收**：核定应税所得率，再计算应纳税所得额

## 七、常见问题

### Q1：分公司是否需要单独缴纳企业所得税？

**答**：不需要。根据《企业所得税法》规定，居民企业在中国境内设立不具有法人资格的营业机构的，应当汇总计算并缴纳企业所得税。

### Q2：企业取得的财政补贴是否需要缴纳企业所得税？

**答**：需要区分情况：
- 属于不征税收入的财政拨款，不缴纳企业所得税
- 属于免税收入的，不缴纳企业所得税
- 其他财政补贴，一般需要计入收入总额缴纳企业所得税

### Q3：个人独资企业和合伙企业的投资者如何纳税？

**答**：个人独资企业和合伙企业不缴纳企业所得税，其投资者按"经营所得"缴纳个人所得税，适用5%-35%的超额累进税率。
