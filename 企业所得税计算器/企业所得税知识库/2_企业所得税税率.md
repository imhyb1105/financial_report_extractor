# 企业所得税税率

**版本号：2026.03-001**

---

## 一、法定税率

### 一般企业：25%

**法律依据**：《中华人民共和国企业所得税法》第四条

> 企业所得税的税率为25%。

**适用范围**：
- 无法定优惠政策的居民企业
- 年应纳税所得额超过300万元的企业

---

## 二、优惠税率

### 1. 小型微利企业

**政策依据**：财政部 税务总局公告2023年第6号

**有效期**：2023年1月1日 至 2027年12月31日

#### 认定条件（需同时满足）

| 条件 | 标准 |
|------|------|
| 从事行业 | 国家非限制和禁止行业 |
| 年应纳税所得额 | 不超过 **300万元** |
| 从业人数 | 不超过 **300人** |
| 资产总额 | 不超过 **5000万元** |

#### 分档税率

| 应纳税所得额 | 税率计算方式 | 实际税负 |
|--------------|--------------|----------|
| 不超过100万元 | 应纳税所得额 x 25% x 20% x 50% | **2.5%** |
| 超过100万元至300万元 | 应纳税所得额 x 25% x 20% | **5%** |

#### 计算示例

**例1**：年应纳税所得额80万元（不超过100万）

```
应纳税额 = 80 x 25% x 20% x 50% = 2万元
实际税负率 = 2 / 80 = 2.5%
```

**例2**：年应纳税所得额200万元（100万-300万区间）

```
不超过100万部分：100 x 25% x 20% x 50% = 2.5万元
超过100万部分：(200-100) x 25% x 20% = 5万元
应纳税额合计 = 2.5 + 5 = 7.5万元
实际税负率 = 7.5 / 200 = 3.75%
```

**例3**：年应纳税所得额320万元（超过300万临界点）

```
警告：超过300万元，全部所得按25%税率计算
应纳税额 = 320 x 25% = 80万元
实际税负率 = 25%
```

### 2. 高新技术企业：15%

**法律依据**：《企业所得税法》第二十八条

> 国家需要重点扶持的高新技术企业，减按15%的税率征收企业所得税。

#### 认定条件

根据《高新技术企业认定管理办法》：

| 条件 | 要求 |
|------|------|
| 注册时间 | 注册满一年以上 |
| 知识产权 | 拥有核心自主知识产权 |
| 技术领域 | 属于国家重点支持的高新技术领域（八大领域） |
| 科技人员占比 | 不低于10% |
| 研发费用占比 | 按收入规模分档：5%/4%/3% |
| 高新产品收入占比 | 不低于60% |
| 创新能力评价 | 达标 |

#### 八大高新技术领域

1. 电子信息技术
2. 生物与新医药技术
3. 航空航天技术
4. 新材料技术
5. 高技术服务业
6. 新能源及节能技术
7. 资源与环境技术
8. 先进制造与自动化

#### 附加优惠

- **亏损弥补年限延长至10年**（一般企业为5年）
- 可同时享受研发费用加计扣除

---

### 3. 技术先进型服务企业：15%

**适用范围**：
- 中国服务外包示范城市内
- 从事技术外包、业务外包、知识外包服务

**认定条件**：符合《技术先进型服务企业认定管理办法》规定

### 4. 集成电路生产企业

| 条件 | 税率 | 政策依据 |
|------|------|----------|
| 线宽小于28纳米 | 10% | 财税[2012]27号 |
| 线宽小于65纳米 | 15% | 财税[2012]27号 |
| 线宽小于130纳米 | 15% | 财税[2012]27号 |

---

### 5. 重点软件企业和集成电路设计企业：10%

**条件**：
- 符合国家规划布局内重点软件企业条件
- 符合集成电路设计企业条件

---

### 6. 非居民企业预提所得税

**法定税率**：20%

**实际执行**：减按10%征收

**法律依据**：《企业所得税法实施条例》第九十一条

> 非居民企业取得企业所得税法第三条第三款规定的所得，减按10%的税率征收企业所得税。

## 四、注意事项

### 1. 小微企业临界点风险

年应纳税所得额超过300万元，则**全部所得**按25%税率征收，不再享受小微企业优惠。

**临界点税负分析**：

| 应纳税所得额 | 应纳税额 | 实际税负 |
|--------------|----------|----------|
| 300万元 | 100x2.5% + 200x5% = 12.5万元 | 4.17% |
| 301万元 | 301x25% = 75.25万元 | 25% |

**结论**：超过1万元，多缴税约62.75万元！

### 2. 政策时效性

小微企业优惠政策有效期至2027年12月31日，届时需关注政策是否延续。

### 3. 认定要求

- 高新技术企业需经相关部门认定，取得认定证书后方可享受优惠
- 高新技术企业资格有效期为三年，期满前需重新认定

### 4. 税收优惠备案

目前大多数优惠采取"自行判别、申报享受、相关资料留存备查"的方式，无需事前备案。

## 五、常见问题

### Q1：小微企业超过300万怎么办？

**答**：年应纳税所得额超过300万元，则**全部所得**按一般企业25%税率计算，不是仅对超限部分按25%计算。

### Q2：高新技术企业认定后有效期多久？

**答**：高新技术企业资格自颁发证书之日起有效期为三年，期满前需重新认定。

### Q3：可以同时享受多种优惠吗？

**答**：部分优惠可以叠加享受，如高新技术企业可同时享受15%税率和研发费用加计扣除。但小型微利企业优惠不能与高新技术企业15%税率同时享受，需择优选择。

### Q4：从业人数和资产总额如何计算？

**答**：从业人数和资产总额按全年季度平均值计算：
- 季度平均值 = (季初值 + 季末值) / 2
- 全年季度平均值 = 全年各季度平均值之和 / 4
