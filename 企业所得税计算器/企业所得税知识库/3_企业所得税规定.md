# 企业所得税规定

**版本号：2026.03-001**

---

## 一、征收管理

### 1. 纳税方式

根据《企业所得税法》第五十四条：

- **按年计算，分月或分季预缴**
- 年度终了后五个月内汇算清缴

### 2. 预缴申报

| 预缴方式 | 申报期限 | 法律依据 |
|----------|----------|----------|
| 按月预缴 | 次月15日内 | 企业所得税法第五十四条 |
| 按季预缴 | 季度终了后15日内 | 企业所得税法第五十四条 |

### 3. 汇算清缴

| 项目 | 内容 |
|------|------|
| 期限 | 年度终了后5个月内（次年5月31日前） |
| 内容 | 全年应纳税所得额、应纳税额、已预缴税额、应补（退）税额 |
| 申报表 | 企业所得税年度纳税申报表（A类或B类） |

## 二、税前扣除项目

### 1. 扣除原则

根据《企业所得税法》第八条：

> 企业实际发生的与取得收入有关的、合理的支出，包括成本、费用、税金、损失和其他支出，准予在计算应纳税所得额时扣除。

**扣除条件**：
- 真实性：实际发生
- 相关性：与取得收入有关
- 合理性：符合经营常规

### 2. 准予扣除项目

| 类型 | 内容 |
|------|------|
| 成本 | 销售成本、销货成本、业务支出等 |
| 费用 | 销售费用、管理费用、财务费用 |
| 税金 | 消费税、城建税、资源税、土地增值税、教育费附加等（**不含增值税、企业所得税**） |
| 损失 | 经营活动中产生的损失 |

### 3. 限额扣除项目

根据《企业所得税法实施条例》相关规定：

| 项目 | 扣除标准 | 法律条款 | 超限处理 |
|------|----------|----------|----------|
| 职工福利费 | 工资薪金总额14%以内 | 第40条 | 不得扣除 |
| 工会经费 | 工资薪金总额2%以内 | 第41条 | 不得扣除 |
| 职工教育经费 | 工资薪金总额8%以内 | 第42条 | 结转以后年度扣除 |
| 业务招待费 | 发生额60%，且不超过营业收入5‰ | 第43条 | 不得扣除 |
| 广告费和业务宣传费 | 营业收入15%以内 | 第44条 | 结转以后年度扣除 |
| 公益性捐赠 | 年度利润总额12%以内 | 第51条 | 结转以后三年扣除 |

**注意**：限额扣除项目的详细计算方法见《5_纳税调整详解.md》

### 4. 不得扣除项目

根据《企业所得税法》第十条：

| 项目 | 说明 |
|------|------|
| 股息、红利 | 向投资者支付的股息、红利等权益性投资收益款项 |
| 企业所得税税款 | 企业所得税税款本身 |
| 税收滞纳金 | 逾期缴纳税款产生的滞纳金 |
| 罚金、罚款 | 行政处罚、刑事处罚的罚金、罚款和被没收财物的损失 |
| 赞助支出 | 与生产经营活动无关的赞助支出 |
| 未经核定的准备金 | 未经核定的准备金支出（如资产减值准备） |
| 其他支出 | 与取得收入无关的其他支出、个人消费等 |

**注意**：
- 可以扣除的违约金：银行罚息、合同违约金、商业纠纷赔偿（属于经营性支出）
- 不得扣除的罚款：交通违章罚款、工商行政处罚、税收滞纳金、法院判决罚金

## 三、税收优惠政策

### 1. 免税收入

根据《企业所得税法》第二十六条：

| 项目 | 说明 |
|------|------|
| 国债利息收入 | 持有国债取得的利息 |
| 居民企业间股息红利 | 符合条件的居民企业之间的股息、红利等权益性投资收益 |
| 非居民企业股息红利 | 在中国境内设立机构、场所的非居民企业从居民企业取得的股息红利 |
| 非营利组织收入 | 符合条件的非营利组织的收入 |

### 2. 加计扣除

#### 研发费用加计扣除

**政策依据**：财政部 税务总局公告2023年第7号

| 企业类型 | 加计扣除比例 |
|----------|--------------|
| 所有企业 | **100%** |

**扣除方式**：
- 未形成无形资产的：在按规定据实扣除的基础上，再按实际发生额的100%加计扣除
- 形成无形资产的：按无形资产成本的200%摊销

**研发费用范围**：
- 人员人工费用
- 直接投入费用
- 折旧费用
- 无形资产摊销
- 新产品设计费等

#### 残疾人工资加计扣除

**扣除比例**：100%

**享受条件**（需同时满足）：
1. 依法与每位残疾人签订1年以上劳动合同
2. 残疾人在单位实际上岗工作
3. 按月足额缴纳社保
4. 通过银行支付不低于当地最低工资标准的工资
5. 具备安置残疾人上岗工作的基本设施

### 3. 加速折旧

| 政策 | 内容 |
|------|------|
| 一次性扣除 | 500万元以下设备器具可一次性税前扣除 |
| 缩短年限 | 特定固定资产可缩短折旧年限 |
| 加速方法 | 可采用双倍余额递减法或年数总和法 |

### 4. 亏损弥补

根据《企业所得税法》第十八条：

| 企业类型 | 弥补年限 |
|----------|----------|
| 一般企业 | **5年** |
| 高新技术企业 | **10年** |
| 科技型中小企业 | **10年** |

> 企业纳税年度发生的亏损，准予向以后年度结转，用以后年度的所得弥补，但结转年限最长不得超过五年。

**弥补规则**：
- 从亏损年度次年起连续计算
- 先弥补最早年度的亏损
- 弥补时以应纳税所得额为基数

## 四、特别纳税调整

### 关联交易管理

根据《企业所得税法》第四十一条至第四十八条：

| 要求 | 内容 |
|------|------|
| 独立交易原则 | 企业与关联方之间的业务往来，应符合独立交易原则 |
| 税务调整 | 不符合独立交易原则的，税务机关有权按照合理方法调整 |
| 转让定价文档 | 年度关联业务往来报告表、国别报告（符合条件的企业） |

## 五、申报资料

### 预缴申报资料

- 企业所得税月（季）度预缴纳税申报表（A类或B类）

### 汇算清缴资料

| 资料类型 | 内容 |
|----------|------|
| 申报表 | 企业所得税年度纳税申报表（A类或B类） |
| 财务报告 | 财务会计报告 |
| 其他资料 | 与纳税有关的合同、协议及凭证 |

### 优惠备案资料

大多数优惠采取"自行判别、申报享受、相关资料留存备查"的方式，需留存以下资料：
- 优惠事项备案表
- 相关证明材料
- 财务核算资料

## 六、法律责任

### 滞纳金

根据《税收征收管理法》第三十二条：

> 纳税人未按照规定期限缴纳税款的，扣缴义务人未按照规定期限解缴税款的，税务机关除责令限期缴纳外，从滞纳税款之日起，按日加收滞纳税款万分之五的滞纳金。

### 罚款

| 违法行为 | 处罚 |
|----------|------|
| 偷税 | 追缴税款、滞纳金，并处不缴或者少缴税款百分之五十以上五倍以下罚款 |
| 情节严重 | 构成犯罪的，依法追究刑事责任 |

## 七、常见问题

### Q1：增值税可以税前扣除吗？

**答**：不可以。增值税是价外税，不在企业所得税税前扣除范围内。

### Q2：企业所得税本身可以税前扣除吗？

**答**：不可以。根据《企业所得税法》第十条，企业所得税税款不得税前扣除。

### Q3：预缴的企业所得税年度内可以退吗？

**答**：一般情况下，预缴税款在年度汇算清缴时统一结算，多退少补。平时预缴时不办理退税。

### Q4：亏损企业是否需要申报企业所得税？

**答**：需要。无论盈利还是亏损，都应在规定期限内进行企业所得税申报。
