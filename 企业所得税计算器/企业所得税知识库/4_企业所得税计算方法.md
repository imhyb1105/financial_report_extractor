# 企业所得税计算方法

## 一、基本计算公式

### 应纳税所得额

**直接计算法**（企业所得税法第五条）：
```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 允许弥补的以前年度亏损
```

**间接计算法**：
```
应纳税所得额 = 会计利润总额 ± 纳税调整项目金额
```

### 应纳税额

**基本公式**（企业所得税法第三条）：
```
应纳税额 = 应纳税所得额 × 适用税率 - 减免税额 - 抵免税额
```

## 二、各类企业计算示例

### 1. 一般企业（25%税率）

**例1**：某一般企业年度数据
- 收入总额：1,000万元
- 成本费用：700万元
- 纳税调整增加：50万元

**计算**：
```
应纳税所得额 = 1,000 - 700 + 50 = 350万元
应纳税额 = 350 × 25% = 87.5万元
实际税负率 = 87.5 ÷ 1,000 = 8.75%
```

---

### 2. 小型微利企业（分档计算）

**例2**：年应纳税所得额80万元（≤100万）

**计算**：
```
应纳税额 = 80 × 25% × 20% × 50% = 2万元
实际税负率 = 2 ÷ 80 = 2.5%
```

**例3**：年应纳税所得额200万元（100万-300万区间）

**计算**：
```
≤100万部分：100 × 25% × 20% × 50% = 2.5万元
100万-300万部分：(200-100) × 25% × 20% = 5万元
应纳税额合计 = 2.5 + 5 = 7.5万元
实际税负率 = 7.5 ÷ 200 = 3.75%
```

**例4**：年应纳税所得额320万元（超过300万临界点）

**⚠️ 重要**：超过300万元，全部所得按一般企业25%税率计算，不再享受小微企业优惠

```
应纳税额 = 320 × 25% = 80万元
实际税负率 = 80 ÷ 320 = 25%
```

**对比**：如果符合小微企业条件（假设200万）
- 小微企业：7.5万元
- 一般企业：50万元
- 节税：42.5万元

---

### 3. 高新技术企业（15%税率）

**例5**：某高新技术企业年度数据
- 年应纳税所得额：500万元

**计算**：
```
应纳税额 = 500 × 15% = 75万元
实际税负率 = 75 ÷ 500 = 15%

相比一般企业节税 = 500 × (25% - 15%) = 50万元
```

---

## 三、研发费用加计扣除

### 计算公式

```
加计扣除金额 = 研发费用 × 100%
应纳税所得额 = 应纳税所得额（扣除前） - 加计扣除金额
```

### 示例

**例6**：某企业
- 会计利润：300万元
- 研发费用：100万元（已计入成本费用）

**计算**：
```
研发费用加计扣除 = 100 × 100% = 100万元
应纳税所得额 = 300 - 100 = 200万元

按一般企业25%：
应纳税额 = 200 × 25% = 50万元

按高新技术企业15%：
应纳税额 = 200 × 15% = 30万元
```

---

## 四、公益性捐赠扣除

### 计算公式

```
扣除限额 = 年度利润总额 × 12%
实际扣除额 = Min(捐赠金额, 扣除限额)
超限部分可结转以后三年扣除
```

### 示例

**例7**：某企业
- 年度利润总额：500万元
- 公益性捐赠：80万元

**计算**：
```
扣除限额 = 500 × 12% = 60万元
本年扣除 = 60万元（限额内）
结转以后年度扣除 = 80 - 60 = 20万元（可在以后三年内扣除）
```

## 五、亏损弥补

### 计算规则

- 以前年度亏损可在以后5年内弥补（高新技术企业10年）
- 弥补顺序：先弥补最早年度的亏损

### 示例

**例8**：某企业盈亏情况

| 年度 | 盈亏（万元） | 说明 |
|-----|------------|------|
| 2023年 | -100 | 亏损 |
| 2024年 | 30 | 弥补2023年亏损 |
| 2025年 | 50 | 弥补2023年亏损 |
| 2026年 | 80 | 部分弥补2023年亏损 |

**计算**：
```
2024年弥补2023年亏损：30万元，剩余亏损70万元
2025年弥补2023年亏损：50万元，剩余亏损20万元
2026年弥补2023年亏损：20万元，剩余应纳税所得额60万元
2026年应纳税额 = 60 × 25% = 15万元
```

## 六、综合计算示例

**例9**：某高新技术企业2026年度数据

| 项目 | 金额（万元） |
|-----|------------|
| 收入总额 | 2,000 |
| 成本费用 | 1,500（含研发费用200） |
| 公益性捐赠 | 30 |
| 2025年亏损 | 50 |

**计算步骤**：

```
第一步：计算会计利润
会计利润 = 2,000 - 1,500 = 500万元

第二步：计算纳税调整
- 研发费用加计扣除 = 200 × 100% = 200万元
- 公益性捐赠限额 = 500 × 12% = 60万元
  （实际30万元，未超限，可全额扣除）

第三步：计算应纳税所得额
应纳税所得额 = 500 - 200 = 300万元

第四步：弥补亏损
弥补2025年亏损后 = 300 - 50 = 250万元

第五步：计算应纳税额
应纳税额 = 250 × 15% = 37.5万元

第六步：计算节税效果
- 研发加计扣除节税 = 200 × 15% = 30万元
- 高新技术优惠节税 = 250 × (25% - 15%) = 25万元
- 合计节税 = 55万元
```

## 七、税负率参考

### 计算公式

```
实际税负率 = 应纳税额 ÷ 收入总额 × 100%
```

### 行业参考税负率

| 行业 | 参考税负率 |
|-----|----------|
| 制造业 | 2% - 4% |
| 商贸企业 | 1% - 2% |
| 服务业 | 3% - 5% |
| 房地产 | 5% - 8% |
| 高新技术企业 | 2% - 4% |

**⚠️ 提示**：税负率仅供参考，实际税负受多种因素影响，如享受的优惠政策、成本结构等。

## 八、常见问题

### Q1：小微企业超过300万怎么办？

**答**：年应纳税所得额超过300万元，则**全部所得**按一般企业25%税率计算，不再享受小微企业优惠。

### Q2：研发费用加计扣除需要备案吗？

**答**：目前研发费用加计扣除采用"自行判别、申报享受、相关资料留存备查"的方式，无需事前备案，但需留存相关资料备查。

### Q3：亏损弥补年限从哪年算起？

**答**：亏损弥补年限从亏损年度的次年开始计算，连续计算，不论期间是否有盈利或亏损。

### Q4：高新技术企业认定后有效期多久？

**答**：高新技术企业资格自颁发证书之日起有效期为三年，期满前需重新认定。
