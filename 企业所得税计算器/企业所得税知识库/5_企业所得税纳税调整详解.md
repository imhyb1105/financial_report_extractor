# 企业所得税纳税调整详解

> 本文档是企业所得税计算的核心，详细说明应纳税所得额与会计利润的差异调整

---

## 一、应纳税所得额计算总览

### 1.1 计算公式

```
应纳税所得额 = 会计利润总额
            + 纳税调整增加额
            - 纳税调整减少额
```

### 1.2 纳税调整的原因

| 调整类型 | 原因 | 示例 |
|---------|------|------|
| **限额扣除超限** | 税法规定扣除限额，超过部分不得扣除 | 业务招待费、广告费 |
| **不得扣除项目** | 税法明确禁止扣除 | 罚款、滞纳金、赞助支出 |
| **免税收入** | 会计已计入利润，但税法免税 | 国债利息、股息红利 |
| **不征税收入** | 会计已计入利润，但不属于征税范围 | 财政拨款 |
| **加计扣除** | 会计已扣除，税法允许额外扣除 | 研发费用、残疾人工资 |

## 二、限额扣除项目详解

### 2.1 业务招待费

**法律依据**：《企业所得税法实施条例》第四十三条

**扣除标准**：
```
扣除限额 = Min(实际发生额 × 60%, 销售(营业)收入 × 5‰)
超限部分 = 实际发生额 - 扣除限额（需调增应纳税所得额）
超限处理：不得结转以后年度扣除
```

**计算示例**：

| 情形 | 销售收入 | 实际发生 | 60%金额 | 5‰限额 | 扣除限额 | 调增金额 |
|------|---------|---------|---------|--------|---------|---------|
| 情形A | 1000万 | 10万 | 6万 | 5万 | **5万** | **5万** |
| 情形B | 2000万 | 8万 | 4.8万 | 10万 | **4.8万** | **3.2万** |
| 情形C | 500万 | 2万 | 1.2万 | 2.5万 | **1.2万** | **0.8万** |

### 2.2 广告费和业务宣传费

**法律依据**：《企业所得税法实施条例》第四十四条

**扣除标准**：

| 企业类型 | 扣除比例 | 超限处理 |
|---------|---------|---------|
| 一般企业 | 销售(营业)收入 × **15%** | 可结转以后年度 |
| 化妆品制造/销售 | 销售(营业)收入 × **30%** | 可结转以后年度 |
| 医药制造 | 销售(营业)收入 × **30%** | 可结转以后年度 |
| 饮料制造（不含酒类） | 销售(营业)收入 × **30%** | 可结转以后年度 |

**计算示例**：
```
某一般企业：
- 销售(营业)收入：1000万元
- 广告费实际发生：200万元
- 扣除限额 = 1000 × 15% = 150万元
- 超限调增 = 200 - 150 = 50万元（可结转）
```

### 2.3 职工福利费

**法律依据**：《企业所得税法实施条例》第四十条

**扣除标准**：
```
扣除限额 = 工资薪金总额 × 14%
超限部分 = 实际发生额 - 扣除限额（需调增应纳税所得额）
超限处理：不得结转以后年度扣除
```

**计算示例**：
```
某企业：
- 工资薪金总额：500万元
- 职工福利费实际发生：80万元
- 扣除限额 = 500 × 14% = 70万元
- 超限调增 = 80 - 70 = 10万元
```

### 2.4 工会经费

**法律依据**：《企业所得税法实施条例》第四十一条

**扣除标准**：
```
扣除限额 = 工资薪金总额 × 2%
超限调增，不得结转
```

### 2.5 职工教育经费

**法律依据**：《企业所得税法实施条例》第四十二条

**扣除标准**：
```
扣除限额 = 工资薪金总额 × 8%
超限部分可结转以后年度扣除
```

### 2.6 公益性捐赠

**法律依据**：《企业所得税法实施条例》第五十一条

**扣除标准**：
```
扣除限额 = 年度利润总额 × 12%
超限部分可结转以后三年扣除
```

**注意事项**：
- 必须通过公益性社会组织或县级以上人民政府捐赠
- 需取得合规的公益事业捐赠票据

### 2.7 限额扣除速查表

| 项目 | 扣除基数 | 扣除比例 | 超限处理 | 法律依据 |
|------|---------|---------|---------|---------|
| 业务招待费 | 销售(营业)收入 | 5‰ 与 60%孰低 | 不结转 | 实施条例第43条 |
| 广告费 | 销售(营业)收入 | 15%（特殊30%） | 可结转 | 实施条例第44条 |
| 职工福利费 | 工资薪金总额 | 14% | 不结转 | 实施条例第40条 |
| 工会经费 | 工资薪金总额 | 2% | 不结转 | 实施条例第41条 |
| 职工教育经费 | 工资薪金总额 | 8% | 可结转 | 实施条例第42条 |
| 公益性捐赠 | 年度利润总额 | 12% | 结转3年 | 实施条例第51条 |

## 三、不得扣除项目（需全额调增）

### 3.1 不得扣除项目清单

| 项目 | 说明 | 法律依据 |
|------|------|---------|
| **罚款、罚金** | 行政处罚、刑事处罚 | 税法第10条 |
| **税收滞纳金** | 逾期缴纳税款产生的滞纳金 | 税法第10条 |
| **赞助支出** | 非广告性质的赞助 | 税法第10条 |
| **与经营无关支出** | 个人消费、家庭支出 | 税法第10条 |
| **未经核定的准备金** | 资产减值准备等 | 税法第10条 |
| **企业所得税税款** | 企业自身应缴所得税 | 税法第10条 |

### 3.2 注意事项

**可以扣除的违约金/罚息**：
- 银行罚息：可扣除（属于经营性支出）
- 合同违约金：可扣除（属于经营性支出）
- 商业纠纷赔偿：可扣除（属于经营性支出）

**不得扣除的罚款/滞纳金**：
- 交通违章罚款：不得扣除
- 工商行政处罚：不得扣除
- 税收滞纳金：不得扣除
- 法院判决罚金：不得扣除

## 四、免税/不征税收入（需调减）

### 4.1 免税收入

| 项目 | 说明 | 法律依据 |
|------|------|---------|
| **国债利息收入** | 持有国债取得的利息 | 税法第26条 |
| **符合条件的股息红利** | 居民企业之间的股息红利 | 税法第26条 |
| **符合条件的非营利组织收入** | 非营利组织从事公益活动收入 | 税法第26条 |

### 4.2 不征税收入

| 项目 | 说明 | 法律依据 |
|------|------|---------|
| **财政拨款** | 各级人民政府拨付的财政资金 | 税法第7条 |
| **行政事业性收费** | 纳入财政管理的收费 | 税法第7条 |
| **政府性基金** | 纳入财政管理的基金 | 税法第7条 |

### 4.3 免税与不征税的区别

| 类型 | 会计处理 | 税务处理 | 是否可结转扣除 |
|------|---------|---------|---------------|
| 免税收入 | 计入利润 | 调减应纳税所得额 | 对应支出可扣除 |
| 不征税收入 | 计入利润 | 调减应纳税所得额 | 对应支出**不得**扣除 |

## 五、加计扣除

### 5.1 研发费用加计扣除（100%）

**法律依据**：财政部 税务总局公告2023年第7号

**扣除标准**：
```
加计扣除金额 = 研发费用实际发生额 × 100%
应纳税所得额调减 = 加计扣除金额
```

**适用范围**：
- 所有企业均可享受
- 2023年1月1日起执行

**研发费用范围**：
- 人员人工费用
- 直接投入费用
- 折旧费用
- 无形资产摊销
- 新产品设计费等

### 5.2 残疾人工资加计扣除（100%）

**法律依据**：《企业所得税法实施条例》第九十六条

**扣除标准**：
```
加计扣除金额 = 支付给残疾职工的工资 × 100%
```

**享受条件**（需同时满足）：
1. 依法与每位残疾人签订1年以上劳动合同
2. 残疾人在单位实际上岗工作
3. 按月足额缴纳社保
4. 通过银行支付不低于当地最低工资标准的工资
5. 具备安置残疾人上岗工作的基本设施

## 六、亏损弥补

### 6.1 弥补规则

| 企业类型 | 弥补年限 | 说明 |
|---------|---------|------|
| 一般企业 | **5年** | 从亏损年度次年起连续计算 |
| 高新技术企业 | **10年** | 需在资格有效期内 |
| 科技型中小企业 | **10年** | 需取得入库登记编号 |

### 6.2 弥补顺序

```
弥补规则：先弥补最早年度的亏损

示例：
2022年亏损100万
2023年亏损50万
2024年盈利120万

2024年弥补顺序：
1. 先弥补2022年亏损100万 → 剩余盈利20万
2. 再弥补2023年亏损20万 → 剩余亏损30万（2025-2028年继续弥补）
```

### 6.3 注意事项

- 亏损弥补必须在税法规定的年限内
- 弥补时以应纳税所得额为基数（非会计利润）
- 高新技术企业资格过期后，剩余亏损仍可继续弥补

```
= 应纳税所得额：980,000 元
```

## 八、常见问题解答

### Q1：限额扣除项目的计算基数是什么？

**答**：
- 业务招待费、广告费：销售(营业)收入（包括主营业务收入、其他业务收入、视同销售收入）
- 职工福利费、工会经费、教育经费：工资薪金总额
- 公益性捐赠：年度利润总额

### Q2：超限的广告费如何结转？

**答**：广告费超限部分可以无限期结转以后年度扣除，直到扣完为止。汇算清缴时填写《广告费和业务宣传费跨年度纳税调整明细表》。

### Q3：研发费用加计扣除需要备案吗？

**答**：目前采用"自行判别、申报享受、相关资料留存备查"的方式，无需事前备案。但需准备以下资料：
- 研发项目立项决议
- 研发人员名单及工时记录
- 研发费用归集明细
- 研发成果证明材料

### Q4：小微企业超过300万怎么办？

**答**：年应纳税所得额超过300万元，则**全部所得**按25%税率计算，不再享受小微企业优惠。不是仅对超限部分按25%计算。

### Q5：免税收入对应的成本费用能否扣除？

**答**：可以。免税收入（如国债利息）对应的成本费用可以扣除，但**不征税收入**（如财政拨款）对应的支出**不得扣除**。
