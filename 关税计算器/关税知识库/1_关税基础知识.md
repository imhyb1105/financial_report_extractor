## 一、关税定义

### 1.1 什么是关税

关税是海关依法对进出境货物、物品征收的一种税。它是国家财政收入的重要来源，也是调节进出口贸易的重要经济杠杆。

### 1.2 关税的特征

| 特征 | 说明 |
|------|------|
| **强制性** | 依法必须缴纳，具有法律强制力 |
| **无偿性** | 纳税人缴纳税款后无直接返还 |
| **固定性** | 税率和征收标准由法律规定 |
| **涉外性** | 涉及国际货物贸易 |

### 1.3 关税的作用

1. **财政收入**：为国家提供稳定的财政收入来源
2. **保护产业**：通过税率调节保护国内产业
3. **调节经济**：控制进出口商品数量和结构
4. **国际谈判**：作为国际贸易谈判的重要工具

## 二、纳税义务人

### 2.1 进口关税纳税人

| 纳税人类型 | 说明 |
|------------|------|
| 进口货物的收货人 | 进口货物的境内接收方 |
| 进口货物的代理人 | 受委托办理进口手续的代理人 |
| 进境物品的所有人 | 携带、邮寄物品进境的个人 |

### 2.2 出口关税纳税人

| 纳税人类型 | 说明 |
|------------|------|
| 出口货物的发货人 | 出口货物的境内发出方 |
| 出口货物的代理人 | 受委托办理出口手续的代理人 |

### 2.3 纳税人权利与义务

**权利：**
- 依法享受税收优惠
- 申请行政复议和诉讼
- 要求海关保守商业秘密
- 申请退税（符合条件时）

**义务：**
- 如实申报货物信息
- 按时缴纳税款
- 提供完整单证资料
- 接受海关监督检查

## 三、征税范围

### 3.1 进口关税征税范围

| 类别 | 具体内容 |
|------|----------|
| 货物 | 通过贸易方式进口的各类商品 |
| 物品 | 个人邮寄、携带进境的物品 |
| 进境运输工具 | 境外运输工具进境用于境内运输 |

### 3.2 出口关税征税范围

我国对大部分出口商品不征收出口关税，仅对少数资源类、高耗能产品征收出口关税，主要包括：

- 煤炭、原油
- 稀土、钨、钼等稀有金属
- 部分有色金属矿砂
- 木片、木浆等林产品
- 部分化肥产品

### 3.3 不征收关税的情形

| 情形 | 说明 |
|------|------|
| 过境货物 | 通过本国境内运往第三国的货物 |
| 转运货物 | 在境内转运但不进入国内市场的货物 |
| 保税货物 | 在保税区内暂未进入国内市场的货物 |
| 特定减免税货物 | 按规定享受减免税的货物 |

## 四、关税分类

### 4.1 按征收对象分类

```
关税
├── 进口关税
│   ├── 最惠国关税
│   ├── 协定关税
│   ├── 特惠关税
│   └── 普通关税
├── 出口关税
└── 过境关税（我国不征收）
```

### 4.2 按征收标准分类

| 类型 | 说明 | 适用商品 |
|------|------|----------|
| 从价税 | 按货物价格的百分比征收 | 大多数工业制成品 |
| 从量税 | 按货物数量/重量征收 | 原油、粮食等大宗商品 |
| 复合税 | 同时从价和从量征收 | 部分敏感商品 |
| 滑准税 | 价格越高税率越低 | 棉花等农产品 |

## 五、相关概念

### 5.1 完税价格

完税价格是海关计征关税时使用的货物价格，是计算关税的基础。

**进口货物完税价格 = 货物成交价格 + 运费 + 保险费 + 其他相关费用**

### 5.2 税则归类

税则归类是将进出口货物按照《进出口税则》的分类规则归入适当的税号，以确定适用的税率。

### 5.3 原产地规则

原产地规则是确定货物"国籍"的标准，不同原产地的货物适用不同的关税税率。

## 六、常见问题

### Q1：个人邮寄物品需要缴纳关税吗？

需要。个人邮寄进境物品，海关依法征收进口税，但有一定免税额度：
- 寄自港、澳、台地区：免税额800元人民币
- 寄自其他国家或地区：免税额1000元人民币

### Q2：跨境电商商品如何缴纳关税？

跨境电商零售进口商品按跨境电商综合税计征，税率通常为法定税率的70%，单次交易限值5000元，年度限值26000元。

### Q3：关税可以退还吗？

符合以下情形可以申请退还关税：
- 因海关误征，多纳税款的
- 海关核准免验进口的货物，完税后发现短卸，经海关认定的
- 已征出口关税的货物，因故未装运出口，申报退关的
- 按特定减免税办法批准予以减免税的货物，在海关监管年限内转让或移作他用需补税的
