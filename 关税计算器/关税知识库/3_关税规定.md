## 二、完税价格确定

### 2.1 进口货物完税价格

#### 2.1.1 一般贸易进口

**完税价格 = 成交价格 + 运费 + 保险费 + 其他相关费用**

| 项目 | 说明 |
|------|------|
| 成交价格 | 货物购买价款 |
| 运费 | 起卸前发生的运费 |
| 保险费 | 起卸前发生的保险费 |
| 其他费用 | 包装费、经纪费等 |

#### 2.1.2 应计入完税价格的费用

| 费用类型 | 是否计入 | 说明 |
|----------|----------|------|
| 货物价款 | 是 | 购买价格 |
| 运输费用 | 是 | 到卸货港的运费 |
| 保险费用 | 是 | 运输保险费 |
| 包装费用 | 是 | 货物包装费 |
| 特许权使用费 | 是 | 与进口货物相关的 |
| 协助费用 | 是 | 买方提供的协助 |
| 转售收益 | 是 | 卖方获得的转售收益 |

#### 2.1.3 不计入完税价格的费用

| 费用类型 | 是否计入 | 说明 |
|----------|----------|------|
| 厂房建设费 | 否 | 进口后发生的费用 |
| 国内运输费 | 否 | 卸货后的运输费 |
| 进口后安装费 | 否 | 进口后的费用 |
| 进口关税 | 否 | 税款本身不计入 |

### 2.2 完税价格确定方法

当成交价格无法确定时，依次使用以下方法：

```
估价方法（依次适用）
├── 1. 成交价格法（首选）
├── 2. 相同货物成交价格法
├── 3. 类似货物成交价格法
├── 4. 倒扣价格法
├── 5. 计算价格法
└── 6. 合理方法（最后选择）
```

### 2.3 出口货物完税价格

**完税价格 = 成交价格 - 出口关税**

- 出口货物完税价格不包含出口关税
- 不包含离境后的运费、保险费
- 包含出口前的国内运输费

## 三、税收优惠政策

### 3.1 法定减免税

| 类型 | 内容 |
|------|------|
| 关税税额在50元以下的一票货物 | 免征关税 |
| 无商业价值的广告品和货样 | 免征关税 |
| 外国政府、国际组织无偿赠送的物资 | 免征关税 |
| 进出境运输工具装载的途中必需燃料 | 免征关税 |
| 因故退还的进出口货物 | 免征关税（需证明） |

### 3.2 特定减免税

| 适用范围 | 优惠内容 |
|----------|----------|
| 科教用品 | 科研、教学用设备免税 |
| 残疾人专用品 | 残疾人个人专用品免税 |
| 扶贫、慈善捐赠物资 | 符合条件的捐赠物资免税 |
| 加工贸易 | 保税进口料件暂缓征税 |
| 保税区、自贸区 | 区内货物享受税收优惠 |

### 3.3 临时减免税

- 由国务院批准
- 针对特定时期、特定企业或特定用途
- 需单独申请，获批后方可享受

## 四、自贸协定优惠政策

### 4.1 主要自贸协定一览

| 协定 | 优惠内容 | 适用条件 |
|------|----------|----------|
| 中国-东盟 | 90%以上商品零关税 | 需提供FORM E原产地证 |
| 中国-韩国 | 最终零关税产品达90% | 需提供中韩原产地证 |
| 中国-澳大利亚 | 绝大多数商品零关税 | 需提供中澳原产地证 |
| RCEP | 90%商品逐步零关税 | 需提供RCEP原产地证 |

### 4.2 原产地规则要点

#### 4.2.1 原产地判定标准

| 标准 | 说明 |
|------|------|
| 完全获得 | 完全在一国生长、生产 |
| 税则归类改变 | 加工后HS编码发生变化 |
| 区域价值成分 | 原材料价值占比达到要求 |
| 制造加工工序 | 完成特定加工工序 |

#### 4.2.2 原产地证明文件

| 证书类型 | 适用协定 |
|----------|----------|
| FORM A | 普惠制（发达国家给予发展中国家） |
| FORM E | 中国-东盟自贸区 |
| FORM F | 中国-智利自贸区 |
| FORM N | 中国-新西兰自贸区 |
| RCEP原产地证 | RCEP协定 |

### 4.3 享受协定优惠的步骤

```
1. 确认货物在协定降税清单内
2. 确认货物符合原产地规则
3. 向出口国签证机构申请原产地证
4. 进口申报时提交原产地证
5. 在报关单上注明协定税率
```

## 五、缴纳规定

### 5.1 纳税期限

| 事项 | 期限 |
|------|------|
| 申报期限 | 货物申报进出口时 |
| 缴款期限 | 填发税款缴款书之日起15日内 |
| 滞纳金起算 | 逾期之日起按日加收0.05% |

### 5.2 缴纳方式

| 方式 | 说明 |
|------|------|
| 电子支付 | 通过单一窗口网上支付 |
| 银行支付 | 到指定银行柜台缴款 |
| 担保放行 | 提供担保后先放行后缴税 |

### 5.3 滞纳金计算

**滞纳金 = 欠缴税款 × 0.05% × 滞纳天数**

- 起始日：缴款期限届满次日
- 终止日：实际缴纳税款之日
- 滞纳金起征点：50元

## 六、退税规定

### 6.1 可退税情形

| 情形 | 说明 |
|------|------|
| 海关多征税款 | 因海关误征导致多缴 |
| 货物复运出境 | 已征进口税的货物复运出境 |
| 货物复运进境 | 已征出口税的货物复运进境 |
| 品质规格原因 | 进口货物因品质问题退运 |
| 技术性退运 | 因技术原因无法使用退运 |

### 6.2 退税申请期限

- 自缴纳税款之日起3年内提出
- 需提供相关证明材料
- 海关审核后办理退税

## 七、常见问题

### Q1：如何申请关税减免？

步骤：
1. 确认是否符合减免税条件
2. 准备相关证明材料
3. 向主管海关提交申请
4. 海关审核批准后享受优惠

### Q2：进口货物完税价格如何确定保险费？

- 有实际保险费：按实际费用计算
- 无法确定保险费：按(货价+运费) × 3‰计算
- 公式：保险费 = (FOB价 + 运费) / (1 - 0.003) × 0.003

### Q3：跨境电商进口税收政策？

跨境电商零售进口税收政策：
- 限值：单次5000元，年度26000元
- 税率：法定税率的70%（关税暂设为0）
- 超过限值：按一般贸易全额征税
