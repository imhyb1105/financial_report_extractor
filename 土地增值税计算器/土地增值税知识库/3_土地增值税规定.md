# 土地增值税规定

## 一、扣除项目

### 1. 取得土地使用权所支付的金额
- 地价款
- 相关税费（契税等）

### 2. 房地产开发成本
- 土地征用及拆迁补偿费
- 前期工程费
- 建筑安装工程费
- 基础设施费
- 公共配套设施费
- 开发间接费用

### 3. 房地产开发费用
- **利息支出**：按实际发生额扣除（需提供金融机构证明）
- **其他开发费用**：按（取得土地使用权支付金额 + 开发成本）× 5%以内计算

**注意**：如不能提供金融机构证明，开发费用按（取得土地使用权支付金额 + 开发成本）× 10%以内计算

### 4. 与转让房地产有关的税金
- 城市维护建设税
- 教育费附加
- 印花税（房地产开发企业扣除）

### 5. 财政部规定的其他扣除项目
房地产开发企业可加计扣除：
```
加计扣除 = (取得土地使用权支付金额 + 开发成本) × 20%
```

## 二、减免优惠

### 1. 普通标准住宅免税
纳税人建造普通标准住宅出售，增值率未超过20%的，免征土地增值税。

**普通标准住宅条件**：
- 住宅小区建筑容积率在1.0以上
- 单套建筑面积在120平方米以下
- 实际成交价格低于同级别土地上住房平均交易价格1.2倍以下

### 2. 国家征用、收回的房地产免税
因国家建设需要依法征用、收回的房地产，免征土地增值税。

### 3. 个人转让住房免税
个人因工作调动或改善居住条件而转让原自用住房，经向税务机关申报核准，凡居住满5年或以上的，免征土地增值税；居住满3年未满5年的，减半征收土地增值税。

### 4. 其他减免
- 建造普通标准住宅出售，增值额超过扣除项目金额20%的，应就其全部增值额按规定计税
- 纳税人既建普通标准住宅又搞其他房地产开发的，应分别核算增值额

## 三、征收管理

### 1. 纳税申报
- 纳税人应在转让房地产合同签订后7日内，向房地产所在地主管税务机关办理纳税申报

### 2. 纳税期限
- 纳税人应在税务机关核定的期限内缴纳土地增值税

### 3. 预征与清算
- 预征：按月或按季预征
- 清算：项目全部竣工、办理结算后进行清算，多退少补
