# 土地增值税计算方法

## 一、基本计算公式

### 1. 增值额计算
```
增值额 = 应税收入 - 扣除项目金额
```

### 2. 增值率计算
```
增值率 = 增值额 ÷ 扣除项目金额 × 100%
```

### 3. 应纳税额计算
```
应纳税额 = 增值额 × 适用税率 - 扣除项目金额 × 速算扣除系数
```

## 二、计算步骤

1. **确定应税收入**
2. **计算扣除项目金额**
3. **计算增值额**
4. **计算增值率**
5. **确定适用税率和速算扣除系数**
6. **计算应纳税额**

## 三、计算实例

### 实例1：增值率≤50%

**基本情况**：
- 应税收入：500万元
- 扣除项目金额合计：400万元

**计算过程**：
```
增值额 = 500 - 400 = 100万元
增值率 = 100 ÷ 400 × 100% = 25%
适用税率：30%（增值率≤50%）
速算扣除系数：0

应纳税额 = 100 × 30% - 400 × 0% = 30万元
```

### 实例2：增值率50%-100%

**基本情况**：
- 应税收入：600万元
- 扣除项目金额合计：400万元

**计算过程**：
```
增值额 = 600 - 400 = 200万元
增值率 = 200 ÷ 400 × 100% = 50%
适用税率：40%（增值率50%-100%）
速算扣除系数：5%

应纳税额 = 200 × 40% - 400 × 5% = 80 - 20 = 60万元
```

### 实例3：增值率100%-200%

**基本情况**：
- 应税收入：900万元
- 扣除项目金额合计：400万元

**计算过程**：
```
增值额 = 900 - 400 = 500万元
增值率 = 500 ÷ 400 × 100% = 125%
适用税率：50%（增值率100%-200%）
速算扣除系数：15%

应纳税额 = 500 × 50% - 400 × 15% = 250 - 60 = 190万元
```

### 实例4：增值率＞200%

**基本情况**：
- 应税收入：1500万元
- 扣除项目金额合计：400万元

**计算过程**：
```
增值额 = 1500 - 400 = 1100万元
增值率 = 1100 ÷ 400 × 100% = 275%
适用税率：60%（增值率＞200%）
速算扣除系数：35%

应纳税额 = 1100 × 60% - 400 × 35% = 660 - 140 = 520万元
```

## 四、房地产开发企业计算实例

**基本情况**：
- 房地产转让收入：2000万元
- 取得土地使用权支付金额：300万元
- 房地产开发成本：500万元
- 房地产开发费用：（300+500）× 10% = 80万元
- 与转让房地产有关的税金：100万元
- 加计扣除：（300+500）× 20% = 160万元

**计算过程**：
```
扣除项目金额合计 = 300 + 500 + 80 + 100 + 160 = 1140万元
增值额 = 2000 - 1140 = 860万元
增值率 = 860 ÷ 1140 × 100% = 75.44%
适用税率：40%
速算扣除系数：5%

应纳税额 = 860 × 40% - 1140 × 5% = 344 - 57 = 287万元
```

## 五、注意事项

1. **无增值或负增值**：不征收土地增值税
2. **普通住宅优惠**：增值率≤20%免税，需单独核算
3. **预征与清算**：按预征率预缴，项目完成后清算
4. **收入确认**：按合同约定的成交价格确认
