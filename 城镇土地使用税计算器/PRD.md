# 城镇土地使用税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向企业财务人员的城镇土地使用税计算器网页应用，帮助用户快速计算城镇土地使用税金额。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、会计师、土地管理人员

**核心痛点**：
- 不同城市规模税率不同
- 各省可在幅度内自行确定具体税额
- 计算需要知道土地等级

## 3. 功能需求

### 3.1 核心功能

#### 城市规模分级税率

| 级别 | 人口规模 | 每平方米年税额（元） |
|------|----------|---------------------|
| 大城市 | 50万以上 | 1.5 ～ 30 |
| 中等城市 | 20万～50万 | 1.2 ～ 24 |
| 小城市 | 20万以下 | 0.9 ～ 18 |
| 县城、建制镇、工矿区 | - | 0.6 ～ 12 |

### 3.2 数据输入/输出

**输入数据**：
- 城市规模（必选）
- 土地面积（㎡）（必填）
- 适用税额（必填）
- 计税期限（月/年）

**输出数据**：
- 年应纳税额
- 月应纳税额
- 计算明细

## 5. 计算公式

### 年应纳税额
```
年应纳税额 = 计税土地面积（㎡）× 适用税额（元/㎡）
```

### 月应纳税额
```
月应纳税额 = 年应纳税额 ÷ 12
```

## 6. 政策依据

- 《中华人民共和国城镇土地使用税暂行条例》
- 各省、自治区、直辖市城镇土地使用税实施细则
