# 增值税计算器产品需求文档 (PRD) - 快速开发版

## 1. 核心目标 (Mission)

开发一款面向财务人员的增值税计算器网页应用，提供简洁实用的界面，支持基本的增值税计算功能，帮助财务人员快速准确地进行增值税相关计算，提高工作效率。

## 2. 用户画像 (Persona)

**主要用户**：财务人员，需要快速准确计算企业增值税应纳税额，希望有简洁明了的操作界面。

## 3. 功能需求（一小时内可实现）

### 3.1 核心功能

1. **一般纳税人增值税计算**
   - 销项税额计算
   - 进项税额计算
   - 应纳税额计算
   - 支持13%、9%、6%税率

2. **小规模纳税人增值税计算**
   - 基于征收率的应纳税额计算
   - 小规模纳税人征收率：法定3%；出售不动产5%；对适用3%项目减按1%至2027-12-31
   - 免税门槛：小规模纳税人月不含税销售额不超过10万元（或季度累计不含税销售额不超过30万元），免征增值税；可按规定选择放弃免税并计税

3. **兼营/多税率合并计算工具**
   - 分别核算：按各税率分别计算销项税额后合计
   - 不能分别核算：从高适用税率

### 3.2 数据输入/输出

**输入数据**：
- 销售额（必填）
- 是否含税（必填）
- 适用税率/征收率（必填）
- 进项税额（一般纳税人计算时必填）

**输出数据**：
- 不含税销售额
- 销项税额
- 应纳税额
- 简单计算依据

## 7. 计算公式参考

### 一般纳税人计算
- 不含税销售额 = 含税销售额 ÷ (1 + 税率)
- 销项税额 = 不含税销售额 × 税率
- 应纳税额 = 销项税额 - 进项税额

### 小规模纳税人计算
- 不含税销售额 = 含税销售额 ÷ (1 + 征收率)
- 应纳税额 = 不含税销售额 × 征收率
- 月不含税销售额不超过10万元（或季度累计不含税销售额不超过30万元）免征增值税（可选择放弃免税并计税）
