# 增值税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向企业财务人员的增值税计算器网页应用，帮助用户快速计算增值税销项税额、进项税额及应纳税额。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、会计师、税务代理人员

**核心痛点**：
- 一般纳税人销项、进项计算复杂
- 不同行业适用不同税率
- 进项税额抵扣条件严格
- 小规模纳税人优惠政策变化

## 3. 功能需求

### 3.1 核心功能

#### 纳税人类型（Tab切换）
- 一般纳税人
- 小规模纳税人

#### 一般纳税人税率

| 税率类型 | 税率 | 适用范围 |
|---------|------|---------|
| 标准税率 | 13% | 销售货物、加工修理修配劳务等 |
| 低税率 | 9% | 交通运输、建筑、不动产等 |
| 基础税率 | 6% | 现代服务、生活服务等 |
| 零税率 | 0% | 出口货物、跨境应税行为 |

#### 小规模纳税人征收率

| 征收率类型 | 税率 | 适用范围 |
|-----------|------|---------|
| 减按征收率 | 1% | 2024-2027年适用3%征收率的销售 |
| 标准征收率 | 3% | 一般应税销售 |
| 不动产销售/租赁 | 5% | 销售、出租不动产 |

### 3.2 数据输入/输出

**输入数据**：
- 纳税人类型（必选）
- 销售额（必填）
- 进项税额（一般纳税人）
- 适用税率/征收率（必选）

**输出数据**：
- 销项税额
- 进项税额（一般纳税人）
- 应纳税额
- 计算明细

## 5. 计算公式

### 一般纳税人
```
销项税额 = 销售额 × 适用税率
应纳税额 = 当期销项税额 - 当期进项税额
```

### 小规模纳税人
```
应纳税额 = 销售额 × 征收率
```

### 进口货物
```
组成计税价格 = 关税完税价格 + 关税 + 消费税
应纳税额 = 组成计税价格 × 税率
```
