# 增值税基础知识

**版本：2026.03-001**
**更新日期：2026年3月16日**

---

## 一、增值税的概念

### 1.1 定义

增值税（Value-Added Tax，简称VAT）是对商品生产、流通、劳务服务中多个环节的新增价值或商品的附加值征收的一种流转税。

### 1.2 核心特征

| 特征 | 说明 |
|:-----|:-----|
| **价外税** | 税款不包含在价格之内，由消费者最终承担 |
| **流转税** | 在商品和服务的流转环节征收 |
| **抵扣机制** | 进项税额可以从销项税额中抵扣 |
| **多环节征收** | 生产、批发、零售各环节分别征收 |

### 1.3 计税原理

```
应纳增值税 = 销项税额 - 进项税额
```

- **销项税额**：纳税人发生应税销售行为时收取的增值税额
- **进项税额**：纳税人购进货物、服务、无形资产等支付或负担的增值税额

## 二、纳税人分类

### 2.1 分类标准

根据《中华人民共和国增值税法》（2026年1月1日施行），增值税纳税人分为两类：

| 纳税人类型 | 年应税销售额标准 | 其他条件 |
|:-----------|:-----------------|:---------|
| **一般纳税人** | 超过500万元 | 或会计核算健全，可申请认定 |
| **小规模纳税人** | 500万元及以下 | 会计核算不健全 |

### 2.2 一般纳税人

#### 认定条件
- 年应税销售额超过财政部、国家税务总局规定的小规模纳税人标准
- 年应税销售额未超过标准，但会计核算健全，能够提供准确税务资料的，可申请认定

#### 特点
- 使用增值税专用发票
- 实行抵扣计税方法
- 税率：13%、9%、6%、0%
- 可开具专用发票供购买方抵扣

### 2.3 小规模纳税人

#### 认定条件
- 年应税销售额在500万元及以下
- 会计核算不健全，不能准确提供税务资料

#### 特点
- 使用增值税普通发票（可代开专用发票）
- 实行简易计税方法
- 征收率：1%、3%、5%
- 不能抵扣进项税额

### 2.4 纳税人类型对比

| 对比项目 | 一般纳税人 | 小规模纳税人 |
|:---------|:-----------|:-------------|
| 年销售额标准 | >500万元 | ≤500万元 |
| 计税方法 | 抵扣法 | 简易计税 |
| 税率/征收率 | 13%/9%/6%/0% | 1%/3%/5% |
| 进项抵扣 | 可以 | 不可以 |
| 发票类型 | 专用/普通发票 | 普通发票（可代开专票） |
| 纳税申报 | 按月或按季 | 按季（部分按月） |

## 三、征税范围

### 3.1 销售货物

指有偿转让货物的所有权。

| 类别 | 说明 | 税率 |
|:-----|:-----|:-----|
| 一般货物 | 工业品、日用品等 | 13% |
| 农产品 | 粮食、蔬菜、水果等 | 9% |
| 图书报纸 | 图书、报纸、杂志 | 9% |

### 3.2 销售服务

| 服务类型 | 具体内容 | 税率 |
|:---------|:---------|:-----|
| 交通运输服务 | 陆运、水运、航空、管道运输 | 9% |
| 邮政服务 | 邮政普遍服务、特殊服务 | 9% |
| 电信服务 | 基础电信、增值电信 | 9%/6% |
| 建筑服务 | 工程、安装、修缮、装饰 | 9% |
| 金融服务 | 贷款、直接收费、保险、金融商品转让 | 6% |
| 现代服务 | 研发、信息技术、文化创意、物流辅助等 | 6% |
| 生活服务 | 文化体育、教育医疗、旅游娱乐、餐饮住宿等 | 6% |

### 3.3 销售无形资产

| 类型 | 说明 | 税率 |
|:-----|:-----|:-----|
| 技术 | 专利、非专利技术 | 6% |
| 商标 | 商标权转让 | 6% |
| 著作权 | 著作权转让 | 6% |
| 商誉 | 商誉转让 | 6% |
| 土地使用权 | 土地使用权转让 | 9% |

### 3.4 销售不动产

| 类型 | 税率 |
|:-----|:-----|
| 建筑物 | 9% |
| 构筑物 | 9% |

### 3.5 进口货物

进口货物一律征收增值税，由海关代征。

## 四、基本计税方法

### 4.1 一般计税方法（一般纳税人）

```
应纳税额 = 当期销项税额 - 当期进项税额
```

### 4.2 简易计税方法（小规模纳税人）

```
应纳税额 = 销售额 × 征收率
```

**注意：** 简易计税方法的销售额为不含税销售额

---

## 五、重要概念解释

### 5.1 销售额

- 不含税销售额 = 含税销售额 ÷ (1 + 税率或征收率)
- 销售额包括全部价款和价外费用

### 5.2 价外费用

包括价外向购买方收取的手续费、补贴、基金、集资费、返还利润、奖励费、违约金、滞纳金、延期付款利息、赔偿金、代收款项、代垫款项、包装费、包装物租金、储备费、优质费、运输装卸费以及其他各种性质的价外收费。

### 5.3 混合销售

一项销售行为如果既涉及服务又涉及货物，为混合销售。

- 从事货物生产、批发或零售的纳税人：按销售货物缴纳增值税
- 其他纳税人：按销售服务缴纳增值税

## 六、纳税义务发生时间

根据《增值税法》第二十八条：

1. 发生应税交易，纳税义务发生时间为收讫销售款项或者取得销售款项索取凭据的当日；先开具发票的，为开具发票的当日。
2. 发生视同应税交易，纳税义务发生时间为完成视同应税交易的当日。
3. 进口货物，纳税义务发生时间为货物报关进口的当日。
