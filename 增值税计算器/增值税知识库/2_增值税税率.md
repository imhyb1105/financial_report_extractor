**版本：2026.03-001**
**更新日期：2026年3月16日**

## 一、一般纳税人税率

### 1.1 税率总览

| 税率 | 适用范围 | 典型示例 |
|:-----|:---------|:---------|
| **13%** | 销售或进口货物（另有规定除外）、加工修理修配劳务 | 电子产品、服装、机械设备 |
| **9%** | 农产品、不动产、交通运输等 | 粮食、房产、运输服务 |
| **6%** | 现代服务、生活服务、金融服务、无形资产 | 咨询服务、餐饮、贷款服务 |
| **0%** | 出口货物、跨境应税行为 | 出口贸易、国际运输 |

---

### 1.2 13%税率适用范围

#### 销售或进口货物（另有规定除外）

| 类别 | 具体内容 |
|:-----|:---------|
| 工业品 | 机械设备、电子产品、化工产品等 |
| 日用品 | 服装、鞋帽、化妆品、家用电器等 |
| 建材 | 钢材、水泥、玻璃等 |
| 汽车 | 各类机动车辆 |
| 加工劳务 | 受托加工货物 |
| 修理修配 | 受托对损伤货物进行修复 |

## 二、小规模纳税人征收率

### 2.1 征收率总览

| 征收率 | 适用范围 | 备注 |
|:-------|:---------|:-----|
| **3%** | 标准征收率 | 大多数应税行为 |
| **5%** | 特定项目 | 不动产销售/租赁等 |
| **1%** | 优惠政策 | 减按1%征收（优惠至2027-12-31） |

### 2.2 3%征收率适用范围

| 适用项目 | 说明 |
|:---------|:-----|
| 销售货物 | 除不动产外的货物销售 |
| 提供加工、修理修配劳务 | 各类加工维修 |
| 销售服务 | 除适用5%征收率外的服务 |
| 销售无形资产 | 除土地使用权外的无形资产 |

### 2.3 5%征收率适用范围

| 适用项目 | 具体内容 |
|:---------|:---------|
| 销售不动产 | 销售自建不动产、取得的不动产 |
| 不动产经营租赁 | 出租不动产 |
| 劳务派遣服务 | 选择差额纳税的 |
| 人力资源外包服务 | 选择差额纳税的 |
| 安全保护服务 | 选择差额纳税的 |

### 2.4 1%征收率（优惠政策）

根据财政部、税务总局公告，小规模纳税人可享受1%征收率优惠：

| 政策内容 | 说明 |
|:---------|:-----|
| 适用范围 | 原适用3%征收率的应税销售收入 |
| 减免方式 | 减按1%征收率征收 |
| 有效期 | 执行至2027年12月31日 |

## 四、含税与不含税换算

### 4.1 换算公式

```
不含税销售额 = 含税销售额 ÷ (1 + 税率或征收率)
含税销售额 = 不含税销售额 × (1 + 税率或征收率)
```

### 4.2 换算示例

| 含税金额 | 税率 | 不含税金额 | 税额 |
|:---------|:-----|:-----------|:-----|
| 11300元 | 13% | 10000元 | 1300元 |
| 10900元 | 9% | 10000元 | 900元 |
| 10600元 | 6% | 10000元 | 600元 |
| 10300元 | 3% | 10000元 | 300元 |
