## 一、进项税额抵扣规定

### 1.1 可抵扣进项税额

#### 准予抵扣的凭证类型

| 凭证类型 | 抵扣方式 | 适用范围 |
|:---------|:---------|:---------|
| 增值税专用发票 | 发票注明的税额 | 购进货物、服务、无形资产、不动产 |
| 海关进口增值税专用缴款书 | 缴款书注明的税额 | 进口货物 |
| 农产品收购发票 | 计算抵扣 | 收购农产品 |
| 农产品销售发票 | 计算抵扣 | 购进农产品 |
| 完税凭证 | 凭证注明的税额 | 从境外购进服务、无形资产 |
| 道路通行费发票 | 计算抵扣 | 支付的道路通行费 |
| 国内旅客运输服务凭证 | 计算抵扣 | 国内旅客运输 |

### 1.2 农产品进项税额计算抵扣

#### 计算公式

```
进项税额 = 买价 × 扣除率
```

#### 扣除率标准

| 扣除率 | 适用范围 |
|:-------|:---------|
| 9% | 一般农产品购进 |
| 10% | 购进农产品用于生产13%税率货物 |

### 1.3 国内旅客运输服务进项抵扣

| 运输方式 | 抵扣基数 | 抵扣率 |
|:---------|:---------|:-------|
| 航空旅客运输 | 票价+燃油附加费 | 9% |
| 铁路旅客运输 | 票价 | 9% |
| 公路、水路旅客运输 | 票价 | 3% |

### 1.4 不得抵扣的进项税额

| 类别 | 具体内容 |
|:-----|:---------|
| 简易计税项目 | 用于简易计税方法计税项目的购进货物、服务等 |
| 免税项目 | 用于免征增值税项目的购进货物、服务等 |
| 集体福利 | 用于集体福利的购进货物、服务等 |
| 个人消费 | 用于个人消费的购进货物、服务等 |
| 非正常损失 | 非正常损失对应的进项税额 |
| 贷款服务 | 贷款服务的进项税额 |
| 餐饮服务 | 纳税人购进的餐饮服务 |
| 居民日常服务 | 纳税人购进的居民日常服务 |
| 娱乐服务 | 纳税人购进的娱乐服务 |

| 免税项目 | 说明 |
|:---------|:-----|

### 2.2 小规模纳税人优惠政策

#### 月销售额10万元以下免征

| 项目 | 内容 |
|:-----|:-----|
| 适用对象 | 小规模纳税人 |
| 免税标准 | 月销售额10万元以下（季度30万元以下） |
| 免税范围 | 销售货物、劳务、服务、无形资产 |

#### 减按1%征收

| 项目 | 内容 |
|:-----|:-----|
| 适用对象 | 小规模纳税人 |
| 适用范围 | 原适用3%征收率的应税销售收入 |
| 征收率 | 减按1%征收 |

### 2.3 重点群体创业就业优惠

| 优惠类型 | 内容 |
|:---------|:-----|
| 重点群体创业 | 登记失业人员、毕业年度内高校毕业生等创业可享税收扣减 |
| 企业招用重点群体 | 招用登记失业半年以上人员可享定额依次扣减税收优惠 |

## 三、起征点规定

### 3.1 起征点标准

| 适用对象 | 按期纳税 | 按次纳税 |
|:---------|:---------|:---------|
| 个体工商户 | 月销售额20000元 | 每次（日）销售额500元 |
| 其他个人 | 月销售额20000元 | 每次（日）销售额500元 |

**注意：** 起征点仅适用于个人（包括个体工商户），不适用于登记为一般纳税人的个体工商户。

### 3.2 起征点政策说明

- 销售额未达到起征点的，免征增值税
- 达到起征点的，全额计算缴纳增值税
- 起征点所称销售额不包括其应纳税额

## 四、纳税义务发生时间

### 4.1 销售货物

| 情形 | 纳税义务发生时间 |
|:-----|:-----------------|
| 直接收款 | 收到销售款或取得索取销售款凭据的当天 |
| 托收承付 | 发出货物并办妥托收手续的当天 |
| 赊销 | 合同约定收款日期的当天 |
| 分期收款 | 合同约定收款日期的当天 |
| 预收货款 | 货物发出的当天（特定货物为收到预收款当天） |

### 4.2 销售服务、无形资产

| 情形 | 纳税义务发生时间 |
|:-----|:-----------------|
| 直接收款 | 收到销售款或取得索取销售款凭据的当天 |
| 预收款 | 收到预收款的当天（部分服务） |
| 金融商品转让 | 所有权转移的当天 |
| 租赁服务预收款 | 收到预收款的当天 |

### 4.3 进口货物

- 报关进口的当天

## 五、发票管理规定

### 5.1 发票类型

| 发票类型 | 使用范围 | 抵扣功能 |
|:---------|:---------|:---------|
| 增值税专用发票 | 一般纳税人销售货物、服务等 | 可抵扣 |
| 增值税普通发票 | 所有纳税人 | 不可抵扣 |
| 增值税电子专用发票 | 一般纳税人 | 可抵扣 |
| 增值税电子普通发票 | 所有纳税人 | 不可抵扣 |

### 5.2 专用发票开具要求

| 要求项目 | 具体内容 |
|:---------|:---------|
| 购买方信息 | 名称、纳税人识别号、地址电话、开户行及账号 |
| 销售方信息 | 名称、纳税人识别号、地址电话、开户行及账号 |
| 货物/服务信息 | 名称、规格型号、单位、数量、单价 |
| 税额信息 | 金额、税率、税额 |

### 5.3 发票开具时限

- 不得提前或滞后开具发票
- 应在纳税义务发生后开具
- 建议在收到款项或交付货物时开具

## 七、征收管理

| 纳税人类型 | 纳税地点 |

| 银行转账 | 通过银行划缴税款 |

| POS机刷卡 | 办税服务厅刷卡缴税 |
