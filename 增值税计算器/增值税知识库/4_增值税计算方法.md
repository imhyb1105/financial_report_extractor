**版本：2026.03-001**
**更新日期：2026年3月16日**

## 一、一般纳税人计税方法

### 1.1 一般计税方法

#### 基本公式

```
应纳税额 = 当期销项税额 - 当期进项税额

其中：
销项税额 = 销售额 × 税率
销售额 = 含税销售额 ÷ (1 + 税率)
```

#### 计算步骤

| 步骤 | 内容 | 公式 |
|:-----|:-----|:-----|
| 第一步 | 确定销售额 | 不含税销售额 = 含税额 ÷ (1 + 税率) |
| 第二步 | 计算销项税额 | 销项税额 = 销售额 × 税率 |
| 第三步 | 汇总进项税额 | 可抵扣进项税额合计 |
| 第四步 | 计算应纳税额 | 应纳税额 = 销项税额 - 进项税额 |

### 1.2 计算实例

#### 实例1：销售货物（13%税率）

**业务背景：**
某贸易公司（一般纳税人）销售电子产品，含税销售额113,000元，当期进项税额8,000元。

**计算过程：**

```
不含税销售额 = 113,000 ÷ (1 + 13%) = 100,000（元）
销项税额 = 100,000 × 13% = 13,000（元）
应纳税额 = 13,000 - 8,000 = 5,000（元）
```

#### 实例2：提供服务（6%税率）

**业务背景：**
某咨询公司（一般纳税人）提供咨询服务，含税收入106,000元，当期进项税额3,000元。

**计算过程：**

```
不含税销售额 = 106,000 ÷ (1 + 6%) = 100,000（元）
销项税额 = 100,000 × 6% = 6,000（元）
应纳税额 = 6,000 - 3,000 = 3,000（元）
```

#### 实例3：混合销售业务

**业务背景：**
某公司（一般纳税人）本月发生以下业务：
- 销售货物（含税）：226,000元，税率13%
- 提供运输服务（含税）：54,500元，税率9%
- 提供咨询服务（含税）：31,800元，税率6%
- 进项税额合计：25,000元

**计算过程：**

```
货物销售：
  不含税销售额 = 226,000 ÷ 1.13 = 200,000（元）
  销项税额 = 200,000 × 13% = 26,000（元）

运输服务：
  不含税销售额 = 54,500 ÷ 1.09 = 50,000（元）
  销项税额 = 50,000 × 9% = 4,500（元）

咨询服务：
  不含税销售额 = 31,800 ÷ 1.06 = 30,000（元）
  销项税额 = 30,000 × 6% = 1,800（元）

销项税额合计 = 26,000 + 4,500 + 1,800 = 32,300（元）
应纳税额 = 32,300 - 25,000 = 7,300（元）
```

## 二、小规模纳税人计税方法

### 2.1 简易计税方法

#### 基本公式

```
应纳税额 = 销售额 × 征收率
销售额 = 含税销售额 ÷ (1 + 征收率)
```

**注意：** 小规模纳税人不得抵扣进项税额

### 2.2 计算实例

#### 实例1：标准征收率（3%）

**业务背景：**
某商店（小规模纳税人）本月含税销售额30,900元。

**计算过程：**

```
不含税销售额 = 30,900 ÷ (1 + 3%) = 30,000（元）
应纳税额 = 30,000 × 3% = 900（元）
```

#### 实例2：优惠征收率（1%）

**业务背景：**
某餐饮店（小规模纳税人）本月含税销售额50,500元，享受1%征收率优惠。

**计算过程：**

```
不含税销售额 = 50,500 ÷ (1 + 1%) = 50,000（元）
应纳税额 = 50,000 × 1% = 500（元）
```

#### 实例3：不动产销售（5%）

**业务背景：**
某个人（小规模纳税人）销售非自建不动产，含税销售额525,000元。

**计算过程：**

```
不含税销售额 = 525,000 ÷ (1 + 5%) = 500,000（元）
应纳税额 = 500,000 × 5% = 25,000（元）
```

## 三、含税与不含税换算

### 3.1 换算公式

```
不含税金额 = 含税金额 ÷ (1 + 税率或征收率)
含税金额 = 不含税金额 × (1 + 税率或征收率)
税额 = 不含税金额 × 税率或征收率
```

### 3.2 换算速查表

| 税率/征收率 | 含税金额 | 不含税金额 | 税额 |
|:-----------:|:---------|:-----------|:-----|
| 13% | 113.00 | 100.00 | 13.00 |
| 9% | 109.00 | 100.00 | 9.00 |
| 6% | 106.00 | 100.00 | 6.00 |
| 5% | 105.00 | 100.00 | 5.00 |
| 3% | 103.00 | 100.00 | 3.00 |
| 1% | 101.00 | 100.00 | 1.00 |

### 3.3 反向计算示例

**已知不含税金额100,000元，求各税率下的含税金额：**

| 税率 | 计算公式 | 含税金额 |
|:-----|:---------|:---------|
| 13% | 100,000 × 1.13 | 113,000元 |
| 9% | 100,000 × 1.09 | 109,000元 |
| 6% | 100,000 × 1.06 | 106,000元 |
| 3% | 100,000 × 1.03 | 103,000元 |
| 1% | 100,000 × 1.01 | 101,000元 |

## 四、差额计税方法

### 4.1 差额计税适用范围

| 行业 | 具体业务 | 扣除项目 |
|:-----|:---------|:---------|
| 劳务派遣 | 劳务派遣服务 | 代付工资、社保 |
| 人力资源外包 | 人力资源外包服务 | 代付工资、社保 |
| 旅游服务 | 旅游服务 | 房费、餐费、交通费、门票等 |
| 建筑服务 | 建筑服务（简易计税） | 分包款 |
| 房地产销售 | 房地产开发销售 | 土地价款 |
| 经纪代理 | 经纪代理服务 | 代收代付的政府性基金、行政事业性收费 |

### 4.2 差额计税公式

```
销售额 = 全部价款和价外费用 - 扣除项目金额
应纳税额 = 销售额 × 税率或征收率
```

### 4.3 差额计税实例

#### 实例：劳务派遣（差额计税）

**业务背景：**
某劳务派遣公司（小规模纳税人）选择差额计税：
- 全部价款（含税）：525,000元
- 代付工资社保：400,000元

**计算过程：**

```
差额销售额 = 525,000 - 400,000 = 125,000（元）
不含税销售额 = 125,000 ÷ (1 + 5%) = 119,047.62（元）
应纳税额 = 119,047.62 × 5% = 5,952.38（元）
```

#### 实例：旅游服务（差额计税）

**业务背景：**
某旅行社（一般纳税人）本月旅游收入（含税）636,000元，支付房费、餐费、交通费、门票等424,000元。

**计算过程：**

```
差额销售额（含税） = 636,000 - 424,000 = 212,000（元）
不含税销售额 = 212,000 ÷ (1 + 6%) = 200,000（元）
销项税额 = 200,000 × 6% = 12,000（元）
```

## 五、进项税额抵扣计算

### 5.1 凭证抵扣

| 凭证类型 | 抵扣金额计算 |
|:---------|:-------------|
| 增值税专用发票 | 发票注明的税额 |
| 海关进口增值税专用缴款书 | 缴款书注明的税额 |
| 完税凭证 | 凭证注明的增值税额 |

### 5.2 计算抵扣

#### 农产品计算抵扣

```
进项税额 = 买价 × 扣除率

扣除率：
- 一般农产品：9%
- 用于生产13%税率货物：10%
```

**实例：**
某食品加工厂收购农产品100,000元，用于生产13%税率的食品。

```
进项税额 = 100,000 × 10% = 10,000（元）
```

#### 国内旅客运输计算抵扣

| 运输方式 | 计算公式 |
|:---------|:---------|
| 航空运输 | （票价 + 燃油附加费）÷ (1 + 9%) × 9% |
| 铁路运输 | 票价 ÷ (1 + 9%) × 9% |
| 公路、水路运输 | 票价 ÷ (1 + 3%) × 3% |

**实例：**
某公司员工出差，取得航空运输电子客票行程单：
- 票价：1,090元
- 燃油附加费：50元

```
进项税额 = (1,090 + 50) ÷ 1.09 × 9% = 93.58（元）
```

## 六、税额计算常见场景

### 6.1 多税率业务

**业务背景：**
某综合服务公司（一般纳税人）本月发生以下业务：

| 业务类型 | 含税收入 | 税率 |
|:---------|:---------|:-----|
| 设备销售 | 226,000 | 13% |
| 运输服务 | 54,500 | 9% |
| 技术咨询 | 106,000 | 6% |
| 进项税额 | - | 20,000 |

**计算：**

```
销项税额：
  设备销售：226,000 ÷ 1.13 × 13% = 26,000（元）
  运输服务：54,500 ÷ 1.09 × 9% = 4,500（元）
  技术咨询：106,000 ÷ 1.06 × 6% = 6,000（元）
  合计：36,500（元）

应纳税额 = 36,500 - 20,000 = 16,500（元）
```

### 6.2 进项大于销项

**业务背景：**
某公司本月销项税额50,000元，进项税额80,000元。

**处理：**

```
应纳税额 = 50,000 - 80,000 = -30,000（元）
```

**说明：** 当期销项税额小于进项税额时，不足抵扣部分结转下期继续抵扣，本期应纳税额为0。

## 七、计算公式汇总

### 7.1 一般纳税人

| 计算项目 | 公式 |
|:---------|:-----|
| 不含税销售额 | 含税销售额 ÷ (1 + 税率) |
| 销项税额 | 销售额 × 税率 |
| 应纳税额 | 销项税额 - 进项税额 |

### 7.2 小规模纳税人

| 计算项目 | 公式 |
|:---------|:-----|
| 不含税销售额 | 含税销售额 ÷ (1 + 征收率) |
| 应纳税额 | 销售额 × 征收率 |

### 7.3 差额计税

| 计算项目 | 公式 |
|:---------|:-----|
| 差额销售额 | 全部价款 - 扣除项目 |
| 不含税销售额 | 差额销售额 ÷ (1 + 税率或征收率) |
| 应纳税额 | 不含税销售额 × 税率或征收率 |
