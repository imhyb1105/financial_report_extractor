## 一、进项税额转出

### 1.1 概念说明

进项税额转出是指纳税人已抵扣进项税额的购进货物、服务、无形资产或不动产，发生特定情形时，将已抵扣的进项税额从当期进项税额中扣减。

### 1.2 需要转出的情形

| 情形类别 | 具体内容 |
|:---------|:---------|
| 用于简易计税项目 | 已抵扣进项用于简易计税方法计税项目 |
| 用于免税项目 | 已抵扣进项用于免征增值税项目 |
| 用于集体福利 | 已抵扣进项用于集体福利 |
| 用于个人消费 | 已抵扣进项用于个人消费 |
| 非正常损失 | 因管理不善造成被盗、丢失、霉烂变质的损失 |
| 非正常损失（不动产） | 违法建筑被依法拆除的损失 |

### 1.3 转出计算方法

#### 按原抵扣税额转出

```
进项税额转出 = 原抵扣的进项税额
```

#### 无法确定原抵扣税额

```
进项税额转出 = 货物账面净值 × 适用税率
```

#### 不动产进项税额转出

```
进项税额转出 = 已抵扣进项税额 × 不动产净值率

不动产净值率 = (不动产净值 ÷ 不动产原值) × 100%
```

### 1.4 计算实例

#### 实例1：货物改变用途

**业务背景：**
某公司上月购进原材料100,000元（不含税），进项税额13,000元已抵扣。本月将其中20%用于集体福利。

**计算过程：**

```
进项税额转出 = 13,000 × 20% = 2,600（元）
```

#### 实例2：非正常损失

**业务背景：**
某公司因管理不善导致库存商品丢失，账面成本50,000元，其中原材料成本30,000元（税率13%），人工成本20,000元。

**计算过程：**

```
进项税额转出 = 30,000 × 13% = 3,900（元）
```

#### 实例3：不动产进项转出

**业务背景：**
某公司购进办公楼原值1,000万元，已抵扣进项税额90万元。使用2年后转为集体宿舍，此时不动产净值800万元。

**计算过程：**

```
不动产净值率 = (800 ÷ 1,000) × 100% = 80%
进项税额转出 = 90 × 80% = 72（万元）
```

## 二、视同销售

### 2.1 概念说明

视同销售是指纳税人发生特定行为，虽然未取得销售收入，但税法规定应当视同销售货物或服务，缴纳增值税。

### 2.2 视同销售货物情形

| 情形 | 具体内容 |
|:-----|:---------|
| 代销货物 | 将货物交付他人代销 |
| 代销货物 | 销售代销货物 |
| 跨机构移送 | 设有两个以上机构并实行统一核算的纳税人，将货物从一个机构移送到其他机构用于销售 |
| 自产/委托加工用于非应税 | 将自产或者委托加工的货物用于非增值税应税项目 |
| 自产/委托加工用于集体福利/个人消费 | 将自产或者委托加工的货物用于集体福利或者个人消费 |
| 自产/委托加工/购进用于投资 | 将自产、委托加工或购进的货物作为投资 |
| 自产/委托加工/购进用于分配 | 将自产、委托加工或购进的货物分配给股东或投资者 |
| 自产/委托加工/购进用于无偿赠送 | 将自产、委托加工或购进的货物无偿赠送其他单位或个人 |

### 2.3 视同销售服务、无形资产、不动产情形

| 情形 | 具体内容 |
|:-----|:---------|
| 无偿提供服务 | 单位或个体工商户向其他单位或个人无偿提供服务 |
| 无偿转让无形资产 | 单位或个人向其他单位或个人无偿转让无形资产 |
| 无偿转让不动产 | 单位或个人向其他单位或个人无偿转让不动产 |

**注意：** 用于公益事业或以社会公众为对象的除外。

### 2.4 视同销售销售额确定

| 确定顺序 | 方法 |
|:---------|:-----|
| 1 | 按纳税人最近时期同类货物/服务平均价格 |
| 2 | 按其他纳税人最近时期同类货物/服务平均价格 |
| 3 | 按组成计税价格 |

### 2.5 组成计税价格

```
组成计税价格 = 成本 × (1 + 成本利润率)

或

组成计税价格 = 成本 × (1 + 成本利润率) ÷ (1 - 消费税税率)
  （属于消费税征税范围的货物）

应纳税额 = 组成计税价格 × 税率
```

**成本利润率：** 一般为10%，由国家税务总局确定。

### 2.6 计算实例

#### 实例1：自产货物用于职工福利

**业务背景：**
某食品公司（一般纳税人）将自产食品发放给职工作为福利，生产成本50,000元，同类产品不含税售价80,000元。

**计算过程：**

```
视同销售额 = 80,000（元）（按同类产品售价）
销项税额 = 80,000 × 13% = 10,400（元）
```

#### 实例2：无偿赠送

**业务背景：**
某公司将自产产品无偿赠送给客户，生产成本30,000元，无同类产品售价，成本利润率10%。

**计算过程：**

```
组成计税价格 = 30,000 × (1 + 10%) = 33,000（元）
销项税额 = 33,000 × 13% = 4,290（元）
```

## 三、混合销售

### 3.1 概念说明

混合销售是指一项销售行为既涉及服务又涉及货物的销售行为。

**特征：**
- 同一销售行为
- 同一客户
- 既涉及服务又涉及货物

### 3.2 税务处理

| 纳税人类型 | 税务处理 |
|:-----------|:---------|
| 从事货物生产、批发或零售的纳税人 | 按销售货物缴纳增值税 |
| 其他纳税人 | 按销售服务缴纳增值税 |

### 3.3 常见混合销售情形

| 情形 | 纳税人类型 | 适用税目 |
|:-----|:-----------|:---------|
| 销售空调并提供安装服务 | 家电商场 | 销售货物（13%） |
| 餐饮企业销售外卖食品 | 餐饮企业 | 餐饮服务（6%） |
| 酒店提供住宿并销售商品 | 酒店 | 住宿服务（6%） |

### 3.4 计算实例

#### 实例：销售空调并安装

**业务背景：**
某家电商场（一般纳税人）销售空调并提供安装服务，合计收取含税金额11,300元。

**税务处理：**
该商场属于从事货物零售的纳税人，按销售货物缴纳增值税。

```
不含税销售额 = 11,300 ÷ 1.13 = 10,000（元）
销项税额 = 10,000 × 13% = 1,300（元）
```

## 四、兼营业务

### 4.1 概念说明

兼营是指纳税人在经营过程中，同时经营不同税率或征收率的货物、服务、无形资产或不动产的销售业务。

**与混合销售的区别：**
- 混合销售：同一项销售行为涉及两类
- 兼营：多项独立的销售行为

### 4.2 税务处理

| 情形 | 处理原则 |
|:-----|:---------|
| 分别核算 | 分别适用不同税率或征收率 |
| 未分别核算 | 从高适用税率 |

### 4.3 计算实例

#### 实例：兼营多税率业务

**业务背景：**
某公司（一般纳税人）本月发生以下业务：
- 销售货物（13%）：含税收入113,000元
- 提供运输服务（9%）：含税收入54,500元
- 提供咨询服务（6%）：含税收入31,800元

**计算过程：**

```
销售货物：
  不含税销售额 = 113,000 ÷ 1.13 = 100,000（元）
  销项税额 = 100,000 × 13% = 13,000（元）

运输服务：
  不含税销售额 = 54,500 ÷ 1.09 = 50,000（元）
  销项税额 = 50,000 × 9% = 4,500（元）

咨询服务：
  不含税销售额 = 31,800 ÷ 1.06 = 30,000（元）
  销项税额 = 30,000 × 6% = 1,800（元）

合计销项税额 = 13,000 + 4,500 + 1,800 = 19,300（元）
```

## 五、包装物押金

### 5.1 税务处理规定

| 情形 | 处理方式 |
|:-----|:---------|
| 单独记账核算 | 不并入销售额征税 |
| 逾期（超过1年） | 并入销售额按所包装货物税率征税 |
| 酒类包装物（非啤酒、黄酒） | 收取时并入销售额征税 |

### 5.2 计算公式

```
逾期押金不含税销售额 = 逾期押金 ÷ (1 + 所包装货物税率)
应纳税额 = 逾期押金不含税销售额 × 税率
```

### 5.3 计算实例

**业务背景：**
某公司收取包装物押金11,300元，已逾期超过1年，所包装货物税率为13%。

**计算过程：**

```
不含税销售额 = 11,300 ÷ 1.13 = 10,000（元）
销项税额 = 10,000 × 13% = 1,300（元）
```

## 六、折扣销售

### 6.1 税务处理规定

| 情形 | 处理方式 |
|:-----|:---------|
| 折扣额与销售额在同一张发票"金额"栏注明 | 可按折扣后的销售额征收增值税 |
| 折扣额另开发票 | 折扣额不得从销售额中减除 |
| 实物折扣 | 按视同销售处理 |

### 6.2 计算实例

#### 实例1：正常折扣

**业务背景：**
某公司销售货物，不含税销售额100,000元，给予5%折扣，折扣额与销售额在同一张发票"金额"栏注明。

**计算过程：**

```
计税销售额 = 100,000 × (1 - 5%) = 95,000（元）
销项税额 = 95,000 × 13% = 12,350（元）
```

#### 实例2：实物折扣

**业务背景：**
某公司销售货物100件，不含税单价100元，赠送同种货物10件。

**计算过程：**

```
销售货物销项税额 = 100 × 100 × 13% = 1,300（元）
赠送货物视同销售销项税额 = 10 × 100 × 13% = 130（元）
合计销项税额 = 1,300 + 130 = 1,430（元）
```

## 七、以旧换新

### 7.1 税务处理规定

| 情形 | 处理方式 |
|:-----|:---------|
| 一般货物 | 按新货物的同期销售价格确定销售额 |
| 金银首饰 | 按实际收取的不含增值税的全部价款征收增值税 |

### 7.2 计算实例

#### 实例1：一般货物以旧换新

**业务背景：**
某商场以旧换新销售家电，新家电含税售价2,260元，旧家电折价200元，实际收取2,060元。

**计算过程：**

```
计税销售额 = 2,260 ÷ 1.13 = 2,000（元）
销项税额 = 2,000 × 13% = 260（元）
```

#### 实例2：金银首饰以旧换新

**业务背景：**
某金店以旧换新销售金项链，新项链含税售价4,520元，旧首饰折价1,000元，实际收取3,520元。

**计算过程：**

```
计税销售额 = 3,520 ÷ 1.13 = 3,115.04（元）
销项税额 = 3,115.04 × 13% = 404.96（元）
```

## 八、还本销售

### 8.1 税务处理规定

还本销售是指纳税人在销售货物后，到一定期限由销售方一次或分次退还给购货方全部或部分价款。

**处理原则：** 销售额为货物销售价格，不得从销售额中减除还本支出。

### 8.2 计算实例

**业务背景：**
某公司采用还本方式销售货物，不含税售价100,000元，约定3年后退还50,000元。

**计算过程：**

```
计税销售额 = 100,000（元）（不得扣除还本支出）
销项税额 = 100,000 × 13% = 13,000（元）
```

## 九、纳税调整汇总表

| 调整类型 | 触发条件 | 处理方式 |
|:---------|:---------|:---------|
| 进项税额转出 | 已抵扣进项改变用途或非正常损失 | 从当期进项中扣减 |
| 视同销售 | 货物/服务用于非销售目的 | 按规定确定销售额计税 |
| 混合销售 | 同一行为涉及货物和服务 | 按主营业务适用税目 |
| 兼营业务 | 同时经营不同税率业务 | 分别核算，未分别从高 |
| 包装物押金 | 逾期未收回 | 并入销售额计税 |
| 折扣销售 | 同一发票注明折扣 | 按折扣后销售额计税 |
| 以旧换新 | 旧货抵价换新 | 一般按新货全价计税 |
| 还本销售 | 销售后返还价款 | 不得扣除还本支出 |
