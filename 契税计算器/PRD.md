# 契税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向购房者的契税计算器网页应用，帮助用户快速计算购房契税金额，支持首套房、二套房、三套及以上住房的不同税率计算。

## 2. 用户画像 (Persona)

**主要用户**：购房者、房产中介、财务人员

**核心痛点**：
- 契税税率根据房屋套数和面积不同而变化
- 2024年新政策调整了面积标准（90㎡→140㎡）
- 二套房税率大幅下调

## 3. 功能需求

### 3.1 核心功能

#### 房屋套数选择
- 首套房
- 二套房
- 三套及以上

#### 面积输入
- 140㎡及以下
- 140㎡以上

### 3.2 税率表

| 房屋性质 | 建筑面积 | 契税税率 |
|---------|---------|---------|
| 首套房 | ≤140㎡ | 1% |
| 首套房 | ＞140㎡ | 1.5% |
| 二套房 | ≤140㎡ | 1% |
| 二套房 | ＞140㎡ | 2% |
| 三套及以上 | 不限面积 | 3% |

### 3.3 数据输入/输出

**输入数据**：
- 房屋成交价格（必填）
- 房屋套数（必选）
- 建筑面积（必填）
- 是否含增值税（可选）

**输出数据**：
- 契税金额
- 适用税率
- 计算明细

## 5. 计算公式

### 基本计算公式
```
契税 = 不含增值税的房屋成交价格 × 适用税率
```

## 6. 政策依据

- 《中华人民共和国契税法》
- 财政部、税务总局关于促进房地产市场平稳健康发展有关税收政策的公告（2024年11月）
