# 契税基础知识

**版本：2026.03-001**
**更新日期：2026年3月**

---

## 一、契税定义

### 1.1 什么是契税

契税是指在中华人民共和国境内转移土地、房屋权属，承受的单位和个人为契税的纳税人，应当依照《中华人民共和国契税法》规定缴纳的税种。

**通俗理解**：契税是在买房、买地时，由购买方（承受方）缴纳的一种财产税。

### 1.2 契税的特点

| 特点 | 说明 |
|------|------|
| 财产税性质 | 针对土地、房屋等不动产转移征收 |
| 一次性征收 | 每次权属转移时缴纳一次 |
| 由承受方缴纳 | 与大多数税种不同，由买方而非卖方缴纳 |
| 属地管理 | 由不动产所在地税务机关征收 |

---

## 二、纳税人

### 2.1 纳税人范围

在中华人民共和国境内转移土地、房屋权属，承受的单位和个人为契税的纳税人。

**承受**是指以受让、购买、受赠、交换等方式取得土地、房屋权属的行为。

### 2.2 纳税人类型

| 纳税人类型 | 具体范围 |
|------------|----------|
| 单位 | 企业单位、事业单位、国家机关、军事单位、社会团体及其他组织 |
| 个人 | 个体经营者及其他个人（包括中国公民、外籍个人） |

### 2.3 纳税义务发生时间

- 纳税人签订土地、房屋权属转移合同的当天
- 纳税人取得其他具有土地、房屋权属转移合同性质凭证的当天

---

## 三、征税范围

### 3.1 征税对象

契税的征税对象为发生土地使用权和房屋所有权权属转移的土地和房屋。

### 3.2 具体征税范围

| 权属类型 | 转移方式 | 说明 |
|----------|----------|------|
| **土地使用权** | 出让 | 国家将土地使用权在一定年限内让与土地使用者 |
| | 转让 | 土地使用者将土地使用权再转移的行为 |
| | 赠与 | 无偿转让土地使用权 |
| | 交换 | 土地使用者之间相互交换土地使用权 |
| **房屋所有权** | 买卖 | 以货币为媒介，出卖人向买受人转让房屋所有权 |
| | 赠与 | 无偿将房屋转让给受赠人 |
| | 交换 | 房屋所有者之间相互交换房屋 |
| | 作价投资（入股） | 以房屋作价投资、入股 |
| | 抵偿债务 | 以房屋抵偿债务 |
| | 获奖承受 | 以获奖方式承受房屋 |
| | 预售方式 | 以预购方式或者预付集资建房款方式承受 |

### 3.3 不征收契税的情形

以下情形不征收契税：

| 情形 | 说明 |
|------|------|
| 经营性事业单位改制 | 经营性事业单位在改制过程中，通过出让方式取得土地使用权的 |
| 夫妻财产分割 | 夫妻之间的房屋权属转移 |
| 法定继承 | 法定继承人通过继承承受土地、房屋权属 |
| 共有财产分割 | 共有财产分割导致的权属转移（不发生权属转移的） |
| 同一投资主体内部 | 同一投资主体内部所属企业之间土地、房屋权属的划转 |

## 四、计税依据

### 4.1 一般规定

| 转移方式 | 计税依据 |
|----------|----------|
| 土地使用权出让、出售 | 成交价格 |
| 房屋买卖 | 成交价格 |
| 土地使用权赠与、房屋赠与 | 市场价格 |
| 土地使用权交换、房屋交换 | 所交换的土地使用权、房屋价格的差额 |
| 作价投资（入股） | 成交价格或市场价格 |
| 抵偿债务 | 成交价格或市场价格 |

### 4.2 成交价格的确定

**成交价格**是指土地、房屋权属转移合同确定的价格，包括：

- 应交付的货币
- 实物
- 其他经济利益

### 4.3 市场价格的确定

**市场价格**是指税务机关参照土地使用权出售、房屋买卖的市场价格依法核定的价格。

### 4.4 特殊情形

| 情形 | 计税依据确定方法 |
|------|------------------|
| 成交价格明显低于市场价格且无正当理由 | 由税务机关参照市场价格核定 |
| 交换价格不相等 | 由多交付货币、实物、其他经济利益的一方缴纳税款 |

## 五、纳税地点与期限

### 5.1 纳税地点

契税由土地、房屋所在地的税务机关征收。

### 5.2 纳税期限

| 事项 | 期限要求 |
|------|----------|
| 纳税义务发生 | 签订权属转移合同或取得具有合同性质凭证的当天 |
| 申报纳税期限 | 纳税义务发生之日起90日内 |
| 完税手续 | 依法办理土地、房屋权属登记手续前 |

### 5.3 逾期缴纳后果

- 依法加收滞纳金
- 无法办理不动产登记手续

## 六、契税与其他税费的区别

| 税种 | 纳税人 | 征税对象 | 计税依据 |
|------|--------|----------|----------|
| **契税** | 买方（承受方） | 土地、房屋权属转移 | 成交价格或市场价格 |
| 增值税 | 卖方（转让方） | 销售不动产的增值额 | 销售额 |
| 个人所得税 | 卖方（转让方） | 财产转让所得 | 转让收入减除原值和费用 |
| 印花税 | 买卖双方 | 产权转移书据 | 合同金额 |
