## 二、契税减免优惠

### 2.1 法定免税项目

根据《契税法》第六条规定，以下情形免征契税：

| 免税情形 | 具体内容 | 备注 |
|----------|----------|------|
| 国家机关、事业单位、社会团体、军事单位 | 承受土地、房屋权属用于办公、教学、医疗、科研、军事设施 | 限于本单位使用 |
| 非营利性的学校、医疗机构、社会福利机构 | 承受土地、房屋权属用于办公、教学、医疗、科研、养老、救助 | 需符合条件 |
| 承受荒山、荒地、荒滩土地使用权 | 用于农、林、牧、渔业生产 | |
| 婚姻关系存续期间夫妻之间变更土地、房屋权属 | 夫妻间房产更名、加名、减名 | |
| 法定继承人通过继承承受土地、房屋权属 | 包括配偶、子女、父母、兄弟姐妹、祖父母、外祖父母 | 遗嘱继承、遗赠不免税 |
| 依照法律规定应当予以免税的外国驻华使馆、领事馆、联合国驻华机构及其外交代表、领事官员等承受土地、房屋权属 | |

### 2.2 减征优惠

| 减征情形 | 优惠内容 | 政策依据 |
|----------|----------|----------|
| 个人购买首套住房 | 90㎡及以下减按1%，90㎡以上减按1.5% | 财税〔2016〕23号 |
| 个人购买第二套改善性住房 | 90㎡及以下减按1%，90㎡以上减按2% | 财税〔2016〕23号 |

### 2.3 地方性减免

| 地区 | 优惠政策 | 适用条件 |
|------|----------|----------|
| 北京市 | 人才购房契税补贴 | 高层次人才 |
| 上海市 | 特定区域契税减免 | 临港新片区等 |
| 深圳市 | 人才住房契税优惠 | 认定的高层次人才 |
| 其他地区 | 以当地政策为准 | |

## 三、特殊情形处理

### 3.1 退房退税

| 情形 | 处理方式 |
|------|----------|
| 商品房买卖合同解除 | 可申请退还已缴纳的契税 |
| 房屋交付前退房 | 凭解除合同证明申请退税 |
| 房屋交付后退房 | 视具体情况处理 |

**退税所需材料**：
- 退税申请表
- 原契税完税证明
- 商品房买卖合同解除协议
- 开发商出具的退房证明
- 身份证明文件

### 3.2 房屋交换

| 情形 | 契税处理 |
|------|----------|
| 等价交换 | 免征契税 |
| 差价交换 | 由支付差价方按差价金额缴纳契税 |
| 交换双方均需缴税 | 按各自取得房产价值分别计算 |

### 3.3 拆迁安置

| 情形 | 契税处理 |
|------|----------|
| 货币补偿后购房 | 成交价格超过补偿款部分缴纳契税 |
| 产权调换 | 超过原面积部分缴纳契税 |
| 等价置换 | 免征契税 |

### 3.4 继承与赠与

| 情形 | 契税处理 | 说明 |
|------|----------|------|
| 法定继承 | 免征 | 需办理继承公证 |
| 遗嘱继承 | 征收3%-5% | 不属于法定免税范围 |
| 遗赠 | 征收3%-5% | 受遗赠人非法定继承人 |
| 房屋赠与 | 征收3%-5% | 按市场价格计税 |

### 3.5 夫妻房产处理

| 情形 | 契税处理 |
|------|----------|
| 婚后房产加名 | 免征契税 |
| 婚后房产去名 | 免征契税 |
| 婚后房产更名 | 免征契税 |
| 离婚房产分割 | 免征契税 |
| 离婚后房产过户 | 免征契税 |

**商品房**：
| 材料 | 说明 |
|------|------|
| 商品房买卖合同 | 原件及复印件 |
| 购房发票 | 原件及复印件 |

| 家庭住房情况查询结果 | 不动产登记查询证明 |
| 婚姻状况证明 | 结婚证或单身证明 |

**二手房**：
| 材料 | 说明 |
|------|------|
| 房屋买卖合同 | 原件及复印件 |
| 原房产证 | 原件及复印件 |
| 身份证明 | 买卖双方身份证 |
| 家庭住房情况查询结果 | 买方提供 |
| 评估报告 | 如需要 |

### 4.2 办理流程

```
1. 签订房屋买卖合同
        ↓
2. 查询家庭住房情况（如需享受优惠税率）
        ↓
3. 准备申报材料
        ↓
4. 前往税务机关或网上申报
        ↓
5. 税务机关审核
        ↓
6. 缴纳契税
        ↓
7. 取得完税证明
        ↓
8. 办理不动产登记
```

### 4.3 网上办理

| 办理渠道 | 说明 |
|----------|------|
| 电子税务局 | 各省电子税务局网上申报 |

## 六、违法责任

### 6.1 逾期申报

| 违法行为 | 处罚措施 |
|----------|----------|
| 未按规定期限申报 | 责令限期改正，可处2000元以下罚款 |
| 情节严重 | 可处2000元以上10000元以下罚款 |

### 6.2 滞纳金

- 滞纳金自滞纳税款之日起，按日加收滞纳税款**万分之五**的滞纳金
- 计算公式：滞纳金 = 滞纳税额 × 0.05% × 滞纳天数

### 6.3 偷逃税款

| 违法行为 | 处罚措施 |
|----------|----------|
| 偷税 | 追缴税款、加收滞纳金，并处不缴或少缴税款50%以上5倍以下罚款 |
| 构成犯罪 | 依法追究刑事责任 |

## 七、常见问题解答

**Q1：购买期房契税什么时候缴纳？**

A：可在房屋交付前或办理产权证前缴纳，但最晚需在办理不动产登记前完成缴纳。

**Q2：契税可以抵扣个人所得税吗？**

A：契税不能抵扣个人所得税，两者是独立的税种。

**Q3：同一套房屋多次交易，每次都要交契税吗？**

A：是的，每次权属转移，承受方都需缴纳契税。

**Q4：购买车位需要缴纳契税吗？**

A：独立车位属于非住宅，需缴纳契税，税率一般为3%-5%。

**Q5：契税完税证明丢失怎么办？**

A：可携带身份证明到原征税机关申请补开或开具完税证明。
