**版本：2026.03-001**

## 一、契税计算基本要素

### 1.1 计算公式

```
应纳契税 = 计税依据 × 适用税率
```

### 1.2 基本要素说明

| 要素 | 说明 |
|------|------|
| **计税依据** | 房屋成交价格或税务机关核定价格 |
| **适用税率** | 根据购房套数和面积确定 |

---

## 二、契税计算公式详解

### 2.1 个人购买住房计算公式

```
应纳契税 = 房屋成交价格 × 适用税率

其中：
- 首套房≤90㎡：税率 = 1%
- 首套房＞90㎡：税率 = 1.5%
- 二套房≤90㎡：税率 = 1%
- 二套房＞90㎡：税率 = 2%
- 三套及以上：税率 = 3%（或当地法定税率）
```

### 2.2 其他情形计算公式

| 情形 | 计算公式 |
|------|----------|
| 房屋赠与 | 应纳税额 = 市场评估价格 × 3%（或当地税率） |
| 房屋交换（有差价） | 应纳税额 = 差价金额 × 3%（或当地税率） |
| 购买非住宅 | 应纳税额 = 成交价格 × 3%-5%（当地税率） |

## 五、计算步骤总结

### 5.1 标准计算步骤

```
步骤1：确定房屋成交价格（计税依据）
         ↓
步骤2：查询家庭成员名下房产情况
         ↓
步骤3：确定购房套数（首套/二套/三套及以上）
         ↓
步骤4：确认房屋面积
         ↓
步骤5：根据套数和面积确定适用税率
         ↓
步骤6：计算应纳契税 = 成交价格 × 适用税率
```

### 5.2 快速计算参考

| 首付款（假设） | 贷款金额 | 参考房屋总价 | 首套房契税（≤90㎡） |
|----------------|----------|--------------|---------------------|
| 30万 | 70万 | 100万 | 1万 |
| 45万 | 105万 | 150万 | 1.5万 |
| 60万 | 140万 | 200万 | 2万 |
| 90万 | 210万 | 300万 | 3万 |

## 六、契税计算注意事项

### 6.1 计税依据注意事项

| 注意事项 | 说明 |
|----------|------|
| 取高值 | 成交价格与核定价格相比，取较高者 |
| 全价计算 | 按房屋总价计算，不能扣除贷款 |
| 包含附属设施 | 成交价格包含储藏室等附属设施价款 |
| 精装修房 | 装修款计入成交价格 |

### 6.2 税率适用注意事项

| 注意事项 | 说明 |
|----------|------|
| 套数认定时间 | 以申报时家庭名下房产为准 |
| 面积认定 | 以不动产权证或合同约定建筑面积为准 |
| 优惠备案 | 需主动申请并提供住房情况证明 |
| 异地购房 | 按房屋所在地政策执行 |

### 6.3 计算中的常见错误

| 错误做法 | 正确做法 |
|----------|----------|
| 按贷款金额计算 | 按房屋总价计算 |
| 忽略面积临界点 | 注意90㎡的分界线 |
| 未核实家庭房产 | 先查询名下房产情况 |
| 装修款不计入 | 精装房装修款计入总价 |

## 七、契税与其他购房费用计算对比

### 7.1 购房相关税费一览（买方）

| 税费项目 | 计算方式 | 费率/税率 |
|----------|----------|-----------|
| **契税** | 成交价 × 税率 | 1%-3%（首套/二套优惠） |
| 印花税 | 成交价 × 0.05% | 0.05% |
| 权证印花税 | 固定 | 5元/本 |
| 登记费 | 固定 | 住宅80元，非住宅550元 |
| 交易手续费 | 面积 × 单价 | 各地标准不同 |
| 评估费 | 评估价 × 费率 | 约0.1%-0.5%（如需评估） |
| 抵押登记费 | 固定 | 约80元（贷款购房） |

### 7.2 购房费用计算实例
