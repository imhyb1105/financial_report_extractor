# 房产税税率

**版本：2026.03-001**
**更新日期：2026年3月**

---

## 一、税率总览

房产税采用比例税率，根据计税方式的不同，设置两档税率：

| 计税方式 | 税率 | 适用范围 |
|----------|------|----------|
| **从价计征** | 1.2%（年税率） | 经营自用的房产 |
| **从租计征** | 12% | 出租的房产 |

---

## 二、从价计征税率（1.2%）

### 2.1 基本规定

- **税率**：年税率1.2%
- **计税依据**：房产原值减除一定比例后的余值
- **适用对象**：经营自用的房产

### 2.2 减除比例

房产余值计算中的减除比例由各省、自治区、直辖市人民政府规定。

**常见减除比例对照表：**

| 减除比例 | 房产余值占比 | 适用地区示例 |
|----------|--------------|--------------|
| 30% | 70% | 北京、上海、广东等多数地区 |
| 25% | 75% | 部分省份 |
| 20% | 80% | 部分省份 |
| 10% | 90% | 个别省份 |

### 2.3 从价计征计算要素

```
年应纳税额 = 房产原值 x (1 - 减除比例) x 1.2%
                    ↓
              房产余值
```

### 2.4 从价计征示例

假设某企业拥有办公用房，房产原值1000万元，当地减除比例为30%。

| 项目 | 计算 |
|------|------|
| 房产原值 | 10,000,000 元 |
| 减除比例 | 30% |
| 房产余值 | 10,000,000 x (1-30%) = 7,000,000 元 |
| 年税率 | 1.2% |
| 年应纳税额 | 7,000,000 x 1.2% = 84,000 元 |
| 月应纳税额 | 84,000 / 12 = 7,000 元 |

---

## 三、从租计征税率（12%）

### 3.1 基本规定

- **税率**：12%
- **计税依据**：房产租金收入
- **适用对象**：出租的房产

### 3.2 租金收入范围

租金收入包括但不限于：

| 收入类型 | 说明 |
|----------|------|
| 货币收入 | 以现金形式收取的租金 |
| 实物收入 | 以实物形式收取的租金 |
| 其他经济利益 | 以其他形式取得的经济利益 |

### 3.3 从租计征计算公式

```
应纳税额 = 租金收入 x 12%
```

### 3.4 从租计征示例

假设某企业将闲置房产出租，年租金收入60万元。

| 项目 | 计算 |
|------|------|
| 年租金收入 | 600,000 元 |
| 税率 | 12% |
| 年应纳税额 | 600,000 x 12% = 72,000 元 |

## 四、特殊税率规定

### 4.1 个人出租住房优惠税率

个人出租住房用于居住的，房产税减按4%的税率征收。

| 出租类型 | 税率 |
|----------|------|
| 个人出租住房（居住） | 4% |
| 个人出租非住房 | 12% |

### 4.2 企事业单位出租住房

企事业单位、社会团体以及其他组织按市场价格向个人出租用于居住的住房，减按4%的税率征收房产税。

### 4.3 公租房税收优惠

对公租房免征房产税（政策执行期限内）。

---

## 五、税率适用规则

### 5.1 同一房产多种用途的处理

同一房产部分自用、部分出租的，应分别计算：

```
总应纳税额 = 自用部分房产税 + 出租部分房产税
```

**计算方法：**

| 部分 | 计税方式 | 计税依据 |
|------|----------|----------|
| 自用部分 | 从价计征 | 自用部分房产原值 x (1-减除比例) x 1.2% |
| 出租部分 | 从租计征 | 出租部分租金收入 x 12% |

### 5.2 税率选择原则

| 使用状态 | 税率选择 |
|----------|----------|
| 完全自用 | 1.2%（从价计征） |
| 完全出租 | 12%（从租计征） |
| 部分自用、部分出租 | 分别适用相应税率 |

## 六、税率速查表

### 6.1 完整税率速查表

| 序号 | 情形 | 计税方式 | 税率 | 计税依据 |
|------|------|----------|------|----------|
| 1 | 企业经营自用房产 | 从价计征 | 1.2% | 房产余值 |
| 2 | 企业出租房产 | 从租计征 | 12% | 租金收入 |
| 3 | 个人出租住房（居住） | 从租计征 | 4% | 租金收入 |
| 4 | 个人出租非住房 | 从租计征 | 12% | 租金收入 |
| 5 | 企事业单位出租住房（居住） | 从租计征 | 4% | 租金收入 |
| 6 | 无租使用免税单位房产 | 从价计征 | 1.2% | 房产余值 |

### 6.2 月税率换算

从价计征年税率为1.2%，如需按月计算：

| 计算周期 | 税率 |
|----------|------|
| 年税率 | 1.2% |
| 季税率 | 0.3% |
| 月税率 | 0.1% |

## 七、注意事项

### 7.1 税率适用注意事项

1. **准确判断计税方式**：自用按从价计征，出租按从租计征
2. **核实减除比例**：不同地区减除比例可能不同
3. **租金收入完整性**：租金收入应包括全部收入
4. **优惠税率适用条件**：注意优惠税率的适用条件

### 7.2 常见误区

| 误区 | 正确理解 |
|------|----------|
| 所有房产税都是1.2% | 出租房产适用12%税率 |
| 减除比例全国统一 | 减除比例由各省规定 |
| 个人出租房产都是4% | 只有出租住房用于居住才适用4% |
