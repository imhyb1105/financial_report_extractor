# 房产税规定

**版本：2026.03-001**
**更新日期：2026年3月**

---

## 一、法律法规依据

### 1.1 主要法规

| 法规名称 | 发布机关 | 生效时间 |
|----------|----------|----------|
| 《中华人民共和国房产税暂行条例》 | 国务院 | 1986年10月1日 |
| 各省、自治区、直辖市房产税实施细则 | 地方政府 | 各地不同 |

### 1.2 法规层级

```
房产税法规体系
├── 国务院行政法规
│   └── 《中华人民共和国房产税暂行条例》
├── 财政部、国家税务总局规范性文件
│   ├── 减免优惠政策
│   ├── 具体问题批复
│   └── 征管规定
└── 地方政府实施细则
    ├── 各省实施细则
    ├── 减除比例规定
    └── 地方减免优惠
```

## 二、法定免税规定

### 2.1 法定免税项目

根据《房产税暂行条例》规定，下列房产免纳房产税：

| 序号 | 免税房产类型 | 具体范围 |
|------|--------------|----------|
| 1 | 国家机关、人民团体、军队自用的房产 | 办公用房、公务用房 |
| 2 | 由国家财政部门拨付事业经费的单位自用的房产 | 学校、医院、科研机构等 |
| 3 | 宗教寺庙、公园、名胜古迹自用的房产 | 宗教活动场所、游览场所 |
| 4 | 个人所有非营业用的房产 | 居民住房 |
| 5 | 经财政部批准免税的其他房产 | 特殊情况 |

### 2.2 法定免税详细说明

#### (1) 国家机关、人民团体、军队自用的房产

- **免税范围**：仅限于自用房产
- **非免税**：出租房产、营业用房不免税

#### (2) 由国家财政部门拨付事业经费的单位自用的房产

- **包括**：学校、医疗卫生单位、托儿所、幼儿园、敬老院、文化、体育、艺术类单位
- **免税范围**：仅限于自用房产
- **非免税**：出租、营业部分不免税

#### (3) 宗教寺庙、公园、名胜古迹自用的房产

- **宗教寺庙**：举行宗教仪式等的房屋和宗教人员使用的生活用房屋
- **公园、名胜古迹**：供公共参观游览的房屋及其管理单位的办公用房屋
- **非免税**：附设的营业单位如影剧院、饮食部、茶社、照相馆等使用房产

#### (4) 个人所有非营业用的房产

- **免税范围**：个人居住用房
- **非免税**：个人营业用房、出租用房

## 三、政策性减免优惠

### 3.1 小微企业优惠

| 政策内容 | 具体规定 |
|----------|----------|
| 小微企业房产税优惠 | 对小微企业减免房产税（具体按当地政策执行） |
| 增值税小规模纳税人 | 可在50%幅度内减征房产税 |

### 3.2 特殊行业优惠

| 行业/类型 | 优惠政策 |
|-----------|----------|
| 金融企业 | 涉农和中小企业贷款损失准备金税前扣除相关房产优惠 |
| 供热企业 | 向居民供热取得的采暖费收入免征房产税 |
| 公租房 | 公租房经营管理单位免征房产税 |
| 商品储备 | 商品储备管理公司及其直属库免征房产税 |
| 高校学生公寓 | 高校学生公寓免征房产税 |
| 军队空余房产 | 军队空余房产租赁收入暂免征收房产税 |

### 3.3 困难减免

纳税人纳税确有困难的，可由省、自治区、直辖市人民政府确定，定期减征或者免征房产税。

**常见困难减免情形：**

- 因自然灾害导致房产损毁
- 企业停产、停业
- 房产大修导致停用
- 经营困难企业

## 四、特殊情形规定

### 4.1 地下建筑房产税

| 类型 | 计税方法 |
|------|----------|
| 工业用途地下建筑 | 以房屋原价的50%-60%作为应税房产原值 |
| 商业和其他用途地下建筑 | 以房屋原价的70%-80%作为应税房产原值 |

### 4.2 无租使用房产

纳税单位无租使用免税单位房产，应由使用人代缴房产税。

**计算方法：**
```
应纳税额 = 房产原值 x (1 - 减除比例) x 1.2%
```

### 4.3 融资租赁房产

融资租赁的房产，由承租人自融资租赁合同约定开始日的次月起依照房产余值缴纳房产税。

### 4.4 投资联营房产

| 情形 | 税务处理 |
|------|----------|
| 参与投资利润分配，共担风险 | 按房产余值计征房产税 |
| 收取固定收入，不承担联营风险 | 按租金收入计征房产税 |

### 4.5 产权出典房产

产权出典的房产，由承典人依照房产余值缴纳房产税。

## 五、纳税期限和地点

### 5.1 纳税期限

| 征收方式 | 纳税期限 |
|----------|----------|
| 按年征收、分期缴纳 | 各地规定不同，一般按季或半年缴纳 |
| 按次缴纳 | 新建、购置房产等特定情形 |

### 5.2 纳税地点

房产税由房产所在地的税务机关征收。

**房产不在同一地点的处理：**
- 应分别向房产所在地的税务机关纳税
- 可由各地税务机关协商确定汇总缴纳

## 六、部分地区减免政策

### 6.1 疫情期间优惠（参考）

疫情期间部分地区出台房产税优惠：
- 房屋租赁减免的可享税收优惠
- 困难行业企业减免

### 6.2 增值税小规模纳税人优惠

自2019年1月1日至2027年12月31日，对增值税小规模纳税人可在50%幅度内减征房产税。

### 6.3 六税两费减免

符合条件的增值税小规模纳税人、小型微利企业、个体工商户可享受"六税两费"减免政策，包括房产税。

## 七、征管规定

### 7.1 申报方式

| 申报方式 | 说明 |
|----------|------|
| 电子税务局 | 网上申报 |
| 办税服务厅 | 现场申报 |
| 委托代征 | 特殊情况 |

### 7.2 申报资料

一般需要提供：
- 房产税纳税申报表
- 房产原值、面积等明细资料
- 租金收入证明（出租房产）
- 减免税相关证明（适用减免优惠）

### 7.3 违章处理

| 违章行为 | 处罚规定 |
|----------|----------|
| 逾期未申报 | 责令限期改正，可处罚款 |
| 逾期未缴纳税款 | 按日加收滞纳金（万分之五） |
| 偷逃税款 | 补税、加收滞纳金、罚款 |

## 九、注意事项

1. **及时关注政策变化**：税收优惠政策可能有执行期限
2. **准确理解适用条件**：确保符合优惠条件再适用
3. **保留相关资料**：减免优惠需要相关证明材料
4. **咨询当地税务机关**：具体执行以当地规定为准
