# 消费税计算方法

## 一、从价计征

### 公式
```
应纳税额 = 销售额 × 税率
```

### 实例
某企业销售高档化妆品，不含增值税销售额100万元，税率15%。
```
应纳税额 = 100 × 15% = 15万元
```

## 二、从量计征

### 公式
```
应纳税额 = 销售数量 × 单位税额
```

### 实例
某炼油厂销售汽油10万升，税额1.52元/升。
```
应纳税额 = 100,000 × 1.52 = 152,000元
```

## 三、复合计征

### 公式
```
应纳税额 = 销售额 × 税率 + 销售数量 × 单位税额
```

### 实例1：卷烟
某卷烟厂销售甲类卷烟，销售额200万元（不含税），销售数量500万支。
```
从价税额 = 2,000,000 × 56% = 1,120,000元
从量税额 = 5,000,000 × 0.003 = 15,000元
应纳税额 = 1,120,000 + 15,000 = 1,135,000元
```

### 实例2：白酒
某酒厂销售白酒，销售额100万元（不含税），销售数量1万斤。
```
从价税额 = 1,000,000 × 20% = 200,000元
从量税额 = 10,000 × 0.5 = 5,000元
应纳税额 = 200,000 + 5,000 = 205,000元
```

## 四、小汽车消费税计算

### 实例
某汽车厂销售排量2.0L的小汽车，不含税销售额50万元。
```
适用税率：5%（1.5L-2.0L）
应纳税额 = 500,000 × 5% = 25,000元
```

## 五、进口应税消费品

### 公式
```
组成计税价格 = (关税完税价格 + 关税) ÷ (1 - 消费税税率)
应纳税额 = 组成计税价格 × 消费税税率
```

### 实例
进口高档化妆品，关税完税价格100万元，关税税率10%，消费税税率15%。
```
关税 = 100 × 10% = 10万元
组成计税价格 = (100 + 10) ÷ (1 - 15%) = 129.41万元
应纳税额 = 129.41 × 15% = 19.41万元
```
