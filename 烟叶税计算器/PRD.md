# 烟叶税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向烟草收购企业和烟叶收购站的烟叶税计算器网页应用，帮助用户快速计算收购烟叶应缴纳的烟叶税。

## 2. 用户画像 (Persona)

**主要用户**：烟草公司财务人员、烟叶收购站工作人员

**核心痛点**：
- 烟叶税按收购金额计算
- 需要区分收购价款和价外补贴
- 计算涉及两个组成部分

## 3. 功能需求

### 3.1 核心功能

#### 税率

| 税目 | 税率 |
|-----|-----|
| 烟叶（收购金额） | 20% |

#### 收购金额组成

```
收购金额 = 收购价款 + 价外补贴
价外补贴 = 收购价款 × 10%（统一按10%计算）
```

### 3.2 数据输入/输出

**输入数据**：
- 收购价款（必填）
- 价外补贴比例（默认10%）

**输出数据**：
- 收购金额
- 应纳烟叶税
- 计算明细

## 5. 计算公式

### 基本公式
```
收购金额 = 收购价款 × (1 + 价外补贴比例)
应纳税额 = 收购金额 × 20%
```

### 示例
```
收购价款：10000元
价外补贴比例：10%
收购金额 = 10000 × (1 + 10%) = 11000元
应纳税额 = 11000 × 20% = 2200元
```
