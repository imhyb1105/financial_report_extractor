# 烟叶税税率

## 一、税率标准

### 法定税率

根据《中华人民共和国烟叶税法》规定，烟叶税实行**比例税率**，税率为**20%**。

### 税率特点

1. **比例税率**：不区分烟叶类型，统一适用20%税率
2. **从价计征**：以收购金额为计税依据
3. **全国统一**：全国各地适用相同税率

## 二、收购金额的确定

### 收购金额构成

收购金额包括两部分：

```
收购金额 = 收购价款 + 价外补贴
```

### 1. 收购价款

收购价款是指纳税人支付给烟叶销售者的烟叶收购价格。

**确定方式**：
- 按照实际支付的价款确定
- 以发票或支付凭证为准

### 2. 价外补贴

价外补贴是指纳税人支付给烟叶销售者的除收购价款以外的各种补贴。

**计算标准**：
- 统一按收购价款的**10%**计算
- 实际补贴超过10%的，按实际补贴金额计算

**补贴类型**：
- 生产投入补贴
- 烟叶生产专用物资补贴
- 其他各类补贴

## 三、价外补贴详解

### 补贴项目

价外补贴通常包括以下项目：

| 补贴类型 | 说明 |
|---------|-----|
| 生产投入补贴 | 肥料、农药等生产资料补贴 |
| 技术服务补贴 | 技术指导、培训等补贴 |
| 基础设施补贴 | 烟水配套、烤房建设等补贴 |
| 优质烟补贴 | 对优质烟叶的额外补贴 |
| 保险补贴 | 烟叶种植保险补贴 |

### 补贴计算

**标准计算**（补贴不超过10%）：
```
价外补贴 = 收购价款 × 10%
```

**实际计算**（补贴超过10%）：
```
价外补贴 = 实际支付的补贴金额
```

### 示例

**示例1：补贴比例为10%**
```
收购价款：100000元
价外补贴：100000 × 10% = 10000元
收购金额：100000 + 10000 = 110000元
```

**示例2：实际补贴超过10%**
```
收购价款：100000元
实际补贴：15000元（15%）
收购金额：100000 + 15000 = 115000元
```

## 四、收购金额与税额关系

### 计算公式

```
收购金额 = 收购价款 × (1 + 补贴比例)
应纳税额 = 收购金额 × 20%
```

### 综合税率

如果补贴按10%计算，则：

```
实际税负 = 收购价款 × 1.1 × 20% = 收购价款 × 22%
```

即相当于收购价款的**22%**。

### 不同补贴比例下的税负

| 补贴比例 | 收购金额 | 应纳税额 | 相当于收购价款比例 |
|---------|---------|---------|------------------|
| 10% | 110% | 22% | 22% |
| 15% | 115% | 23% | 23% |
| 20% | 120% | 24% | 24% |

## 五、与增值税的关系

### 增值税进项抵扣

烟叶收购涉及增值税进项税额抵扣：

```
进项税额 = 收购金额 × 9%
```

### 实际税负

综合考虑烟叶税和增值税：

```
烟叶税：收购金额 × 20%
增值税进项：收购金额 × 9%（可抵扣）
```
