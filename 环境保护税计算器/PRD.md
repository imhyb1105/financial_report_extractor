# 环境保护税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向排污企业的环境保护税计算器网页应用，帮助用户快速计算各类污染物排放应缴纳的环境保护税。

## 2. 用户画像 (Persona)

**主要用户**：排污企业财务人员、环保专员、税务代理人员

**核心痛点**：
- 环保税涉及大气、水、固体废物、噪声四大类
- 污染当量计算复杂
- 不同污染物计税方式不同

## 3. 功能需求

### 3.1 核心功能

#### 污染物分类（Tab切换）
- 大气污染物
- 水污染物
- 固体废物
- 噪声

### 3.2 主要税率表

| 污染物类型 | 计税单位 | 税额 |
|-----------|---------|------|
| 大气污染物 | 污染当量 | 1.2-12元/当量 |
| 水污染物 | 污染当量 | 1.4-14元/当量 |
| 危险废物 | 吨 | 1000元/吨 |
| 冶炼渣 | 吨 | 25元/吨 |
| 噪声超标1-3分贝 | 月 | 350元/月 |
| 噪声超标16分贝以上 | 月 | 11200元/月 |

### 3.3 数据输入/输出

**输入数据**：
- 污染物类型（必选）
- 排放量/超标分贝数（必填）
- 污染当量值（如适用）
- 适用税额（必填）

**输出数据**：
- 应纳税额
- 污染当量数
- 计算明细

## 5. 计算公式

### 大气/水污染物
```
应纳税额 = 污染当量数 × 适用税额
污染当量数 = 污染物排放量 ÷ 污染当量值
```

### 固体废物
```
应纳税额 = 固体废物排放量 × 适用税额
```

### 噪声
```
应纳税额 = 适用税额（按超标分贝数确定）
```

## 6. 政策依据

- 《中华人民共和国环境保护税法》
- 各省、自治区、直辖市环境保护税适用税额决定
