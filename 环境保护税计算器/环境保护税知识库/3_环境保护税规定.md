**版本：** 2026.03-001
**更新日期：** 2026年3月16日

## 一、政策规定

### 1.1 法律法规体系

| 层级 | 名称 | 发布机关 | 施行日期 |
|------|------|----------|----------|
| 法律 | 《中华人民共和国环境保护税法》 | 全国人大常委会 | 2018年1月1日 |
| 行政法规 | 《中华人民共和国环境保护税法实施条例》 | 国务院 | 2018年1月1日 |
| 部门规章 | 各类配套管理办法 | 财政部、税务总局、生态环境部 | 按发布时间 |

### 1.2 核心政策要点

| 政策要点 | 具体规定 |
|----------|----------|
| 征税原则 | 谁污染谁纳税 |
| 计税依据 | 污染物排放量 |
| 征收机关 | 税务机关 |
| 协管部门 | 生态环境主管部门 |
| 申报方式 | 按季申报为主 |

### 1.3 征收管理流程

```
排放污染物 → 监测/核算排放量 → 计算污染当量 → 计算应纳税额 → 申报缴纳
```

## 二、污染当量计算

### 2.1 污染当量的概念

**污染当量** 是根据污染物或者污染排放活动对环境的有害程度以及处理的技术经济性，衡量不同污染物对环境污染的综合性指标。

### 2.2 污染当量数计算公式

**基本公式：**

```
污染当量数 = 污染物排放量 ÷ 污染当量值
```

**参数说明：**

| 参数 | 单位 | 说明 |
|------|------|------|
| 污染物排放量 | 千克（kg） | 污染物的实际排放质量 |
| 污染当量值 | 千克（kg） | 每一污染当量的污染物质量 |
| 污染当量数 | 当量 | 计算出的当量数值 |

### 2.3 排放量计算方法

#### 方法一：自动监测

| 项目 | 规定 |
|------|------|
| 适用对象 | 重点排污单位 |
| 要求 | 安装使用符合国家规定的监测设备 |
| 数据来源 | 监测设备实时监测数据 |
| 月排放量 | 月排放量 = 月监测数据累加 |

#### 方法二：监测机构监测

| 项目 | 规定 |
|------|------|
| 适用对象 | 不具备自动监测条件的纳税人 |
| 要求 | 委托有资质的监测机构监测 |
| 监测频次 | 按规定周期进行监测 |
| 月排放量 = 月监测数据 × 生产时间 |

#### 方法三：排污系数法

| 项目 | 规定 |
|------|------|
| 适用对象 | 不能通过监测方法计算的纳税人 |
| 计算公式 | 排放量 = 产污量 × 排污系数 |
| 数据来源 | 国家或地方发布的排污系数 |

#### 方法四：物料衡算

| 项目 | 规定 |
|------|------|
| 适用对象 | 因客观原因无法监测的纳税人 |
| 计算原理 | 物质守恒定律 |
| 计算公式 | 排放量 = 投入物料量 - 回收利用量 - 产品含物料量 |

### 2.5 污染当量计算实例

**【例1】二氧化硫排放量计算**

某企业本月排放二氧化硫 950 千克。

```
污染当量数 = 950 ÷ 0.95 = 1000 当量
```

**【例2】化学需氧量排放量计算**

某企业本月排放COD 500 千克。

```
污染当量数 = 500 ÷ 1 = 500 当量
```

**【例3】多种污染物计算**

某企业本月排放：
- 二氧化硫：1900 千克
- 氮氧化物：950 千克
- 一般性粉尘：800 千克

```
二氧化硫当量数 = 1900 ÷ 0.95 = 2000 当量
氮氧化物当量数 = 950 ÷ 0.95 = 1000 当量
一般性粉尘当量数 = 800 ÷ 4 = 200 当量
```

## 三、应税污染物排序规则

### 3.1 大气污染物排序

每一排放口或者没有排放口的应税大气污染物，按污染当量数从大到小排序：

| 排序规则 | 说明 |
|----------|------|
| 排序依据 | 按污染当量数从大到小 |
| 征税数量 | 对前三项征税 |
| 计算顺序 | 先排序，再取前三项 |

### 3.2 水污染物排序

#### 第一类水污染物

| 排序规则 | 说明 |
|----------|------|
| 排序依据 | 按污染当量数从大到小 |
| 征税数量 | 对前五项征税 |

#### 第二类水污染物

| 排序规则 | 说明 |
|----------|------|
| 排序依据 | 按污染当量数从大到小 |
| 征税数量 | 对前三项征税 |

## 四、税收优惠规定

### 4.1 减免税条件

| 减免档次 | 适用条件 | 优惠幅度 |
|----------|----------|----------|
| 一档 | 排放浓度低于排放标准30% | 减按75%征收 |
| 二档 | 排放浓度低于排放标准50% | 减按50%征收 |

### 4.2 减免适用范围

| 污染物类型 | 是否适用减免 | 说明 |
|------------|--------------|------|
| 大气污染物 | 适用 | 需满足排放浓度标准 |
| 水污染物 | 适用 | 需满足排放浓度标准 |
| 固体废物 | 不适用 | - |
| 工业噪声 | 不适用 | - |

### 4.3 减免认定条件

1. 符合国家或地方规定的排放标准
2. 排放浓度低于标准30%或50%
3. 有合法有效的监测数据
4. 申报时提交相关证明材料

### 4.4 免税规定

以下情形免征环境保护税：

| 免税情形 | 具体条件 |
|----------|----------|
| 农业生产 | 规模化养殖以外的农业生产排放应税污染物 |
| 机动车船舶 | 机动车、铁路机车、非道路移动机械、船舶等流动污染源排放应税污染物 |
| 污水处理 | 依法设立的城乡污水集中处理场所，不超过国家和地方规定的排放标准向环境排放应税污染物 |
| 垃圾处理 | 依法设立的生活垃圾集中处理场所，不超过国家和地方规定的排放标准向环境排放应税污染物 |

## 五、申报缴纳规定

### 5.1 纳税期限

| 类型 | 计算周期 | 申报期限 |
|------|----------|----------|
| 按期申报 | 按月计算 | 季度终了之日起15日内 |
| 按次申报 | 按次计算 | 纳税义务发生之日起15日内 |

### 5.2 纳税地点

| 情形 | 纳税地点 |
|------|----------|
| 一般情形 | 应税污染物排放地 |
| 跨区域排放 | 分别向各排放地税务机关申报 |

### 5.3 申报资料

| 资料名称 | 说明 |
|----------|------|
| 环境保护税纳税申报表 | 主表 |
| 大气污染物纳税申报表 | 附表（适用时） |
| 水污染物纳税申报表 | 附表（适用时） |
| 固体废物纳税申报表 | 附表（适用时） |
| 噪声纳税申报表 | 附表（适用时） |

## 六、法律责任

### 6.1 逾期申报

| 违法行为 | 处罚措施 |
|----------|----------|
| 未按期申报 | 责令限期改正，可处2000元以下罚款 |
| 情节严重 | 处2000元以上10000元以下罚款 |

### 6.2 逃税行为

| 违法行为 | 处罚措施 |
|----------|----------|
| 逃避缴纳税款 | 补缴税款、滞纳金，并处税款0.5倍至5倍罚款 |
| 构成犯罪 | 依法追究刑事责任 |

### 6.3 虚假申报

| 违法行为 | 处罚措施 |
|----------|----------|
| 编造虚假计税依据 | 责令限期改正，处50000元以下罚款 |
| 不缴或少缴税款 | 补缴税款、滞纳金，并处税款0.5倍至5倍罚款 |
