# 环境保护税计算方法

**版本：** 2026.03-001
**更新日期：** 2026年3月16日

---

## 一、计算公式汇总

### 1.1 大气污染物应纳税额

```
应纳税额 = 污染当量数 × 适用税额
污染当量数 = 污染物排放量（kg）÷ 污染当量值（kg）
```

**说明：**
- 每一排放口按污染当量数从大到小排序，对前三项征税
- 没有排放口的，按前三项征税

### 1.2 水污染物应纳税额

```
应纳税额 = 污染当量数 × 适用税额
污染当量数 = 污染物排放量（kg）÷ 污染当量值（kg）
```

**说明：**
- 第一类水污染物：按前五项征税
- 第二类水污染物：按前三项征税

### 1.3 固体废物应纳税额

```
应纳税额 = 固体废物排放量（吨）× 适用税额（元/吨）
```

**说明：**
- 排放量 = 产生量 - 综合利用量 - 贮存量 - 处置量
- 综合利用、贮存、处置符合国家和地方环境保护标准的，不征税

### 1.4 工业噪声应纳税额

```
应纳税额 = 适用税额（元/月）
```

**说明：**
- 根据超标分贝数确定适用税额
- 昼夜分别计算，合并申报
- 超过16分贝的，按11200元/月征收

## 二、大气污染物计算实例

### 2.1 单一排放口计算

**【例1】某工厂排放情况**

某工厂有一个废气排放口，2026年1月排放数据如下：

| 污染物 | 排放量（kg） | 污染当量值（kg） |
|--------|--------------|------------------|
| 二氧化硫 | 1900 | 0.95 |
| 氮氧化物 | 2850 | 0.95 |
| 一氧化碳 | 8350 | 16.7 |
| 烟尘 | 1090 | 2.18 |

**适用税额：** 6元/污染当量

**计算步骤：**

**第一步：计算各污染物当量数**

```
二氧化硫当量数 = 1900 ÷ 0.95 = 2000 当量
氮氧化物当量数 = 2850 ÷ 0.95 = 3000 当量
一氧化碳当量数 = 8350 ÷ 16.7 = 500 当量
烟尘当量数 = 1090 ÷ 2.18 = 500 当量
```

**第二步：按当量数排序（取前三项）**

| 排名 | 污染物 | 当量数 |
|------|--------|--------|
| 1 | 氮氧化物 | 3000 |
| 2 | 二氧化硫 | 2000 |
| 3 | 一氧化碳 | 500 |
| 4 | 烟尘 | 500 |

**第三步：计算应纳税额**

```
氮氧化物应纳税额 = 3000 × 6 = 18000 元
二氧化硫应纳税额 = 2000 × 6 = 12000 元
一氧化碳应纳税额 = 500 × 6 = 3000 元

本月应纳税额合计 = 18000 + 12000 + 3000 = 33000 元
```

### 2.2 多排放口计算

**【例2】某企业多个排放口**

某企业有2个废气排放口，2026年1月排放数据：

**排放口A：**

| 污染物 | 排放量（kg） | 当量值（kg） | 当量数 |
|--------|--------------|--------------|--------|
| 二氧化硫 | 950 | 0.95 | 1000 |
| 氮氧化物 | 475 | 0.95 | 500 |
| 烟尘 | 218 | 2.18 | 100 |

**排放口B：**

| 污染物 | 排放量（kg） | 当量值（kg） | 当量数 |
|--------|--------------|--------------|--------|
| 二氧化硫 | 1425 | 0.95 | 1500 |
| 氮氧化物 | 1900 | 0.95 | 2000 |
| 一般性粉尘 | 800 | 4 | 200 |

**适用税额：** 6元/污染当量

**计算过程：**

**排放口A（取前三项）：**
```
二氧化硫：1000 × 6 = 6000 元
氮氧化物：500 × 6 = 3000 元
烟尘：100 × 6 = 600 元
排放口A小计：9600 元
```

**排放口B（取前三项）：**
```
氮氧化物：2000 × 6 = 12000 元
二氧化硫：1500 × 6 = 9000 元
一般性粉尘：200 × 6 = 1200 元
排放口B小计：22200 元
```

**合计应纳税额：**
```
应纳税额 = 9600 + 22200 = 31800 元
```

### 2.3 减免税计算

**【例3】符合减排优惠条件**

某企业大气污染物排放浓度低于国家排放标准50%。

| 项目 | 数值 |
|------|------|
| 氮氧化物当量数 | 2000 当量 |
| 适用税额 | 6 元/当量 |
| 减免比例 | 50% |

**计算过程：**
```
正常应纳税额 = 2000 × 6 = 12000 元
减免后应纳税额 = 12000 × 50% = 6000 元
```

## 三、水污染物计算实例

### 3.1 含第一类、第二类水污染物

**【例4】某化工厂污水排放**

某化工厂2026年1月污水排放数据：

**第一类水污染物：**

| 污染物 | 排放量（kg） | 当量值（kg） | 当量数 |
|--------|--------------|--------------|--------|
| 总汞 | 0.005 | 0.0005 | 10 |
| 总镉 | 0.025 | 0.005 | 5 |
| 总铬 | 0.8 | 0.04 | 20 |
| 六价铬 | 0.2 | 0.02 | 10 |
| 总砷 | 0.1 | 0.02 | 5 |

**第二类水污染物：**

| 污染物 | 排放量（kg） | 当量值（kg） | 当量数 |
|--------|--------------|--------------|--------|
| COD | 1500 | 1 | 1500 |
| 氨氮 | 480 | 0.8 | 600 |
| 总磷 | 75 | 0.25 | 300 |
| 悬浮物 | 2000 | 4 | 500 |

**适用税额：** 7元/污染当量

**计算步骤：**

**第一步：第一类水污染物（取前五项，按当量数排序）**

排序：总铬(20) > 总汞(10) = 六价铬(10) > 总镉(5) = 总砷(5)

```
总铬应纳税额 = 20 × 7 = 140 元
总汞应纳税额 = 10 × 7 = 70 元
六价铬应纳税额 = 10 × 7 = 70 元
总镉应纳税额 = 5 × 7 = 35 元
总砷应纳税额 = 5 × 7 = 35 元

第一类小计 = 350 元
```

**第二步：第二类水污染物（取前三项，按当量数排序）**

排序：COD(1500) > 氨氮(600) > 悬浮物(500) > 总磷(300)

```
COD应纳税额 = 1500 × 7 = 10500 元
氨氮应纳税额 = 600 × 7 = 4200 元
悬浮物应纳税额 = 500 × 7 = 3500 元

第二类小计 = 18200 元
```

**合计应纳税额：**
```
应纳税额 = 350 + 18200 = 18550 元
```

### 3.2 pH值和色度计算

**【例5】pH值异常排放**

某企业排放废水500吨，pH值为3（标准范围6-9）。

**pH值污染当量值：** 1吨·pH值

**计算方法：**
```
超标pH值 = |3 - 6| = 3（取偏离标准下限值）
污染当量数 = 500 × 3 ÷ 1 = 1500 当量
应纳税额 = 1500 × 7 = 10500 元
```

**【例6】色度超标排放**

某企业排放废水1000吨，色度倍数为200倍（标准为50倍）。

**色度污染当量值：** 5吨水·倍

**计算方法：**
```
超标倍数 = 200 - 50 = 150 倍
污染当量数 = 1000 × 150 ÷ 5 = 30000 当量
应纳税额 = 30000 × 7 = 210000 元
```

## 五、工业噪声计算实例

### 5.1 昼间噪声超标

**【例10】昼间超标3分贝**

某工厂位于2类声环境功能区（昼间限值60dB），实际昼间厂界噪声为63dB。

**超标分贝数：** 63 - 60 = 3dB

**适用税额：** 350元/月

**计算结果：**
```
月应纳税额 = 350 元
季度应纳税额 = 350 × 3 = 1050 元
```

### 5.2 昼夜均超标

**【例11】昼夜分别超标**

某工厂位于3类声环境功能区（昼间限值65dB，夜间限值55dB）。

| 时段 | 实际值（dB） | 标准值（dB） | 超标（dB） |
|------|--------------|--------------|------------|
| 昼间 | 70 | 65 | 5 |
| 夜间 | 62 | 55 | 7 |

**适用税额：**
- 昼间超标5dB（4-6档）：700元/月
- 夜间超标7dB（7-9档）：1400元/月

**计算结果：**
```
月应纳税额 = 700 + 1400 = 2100 元
季度应纳税额 = 2100 × 3 = 6300 元
```

### 5.3 严重超标

**【例12】超标18分贝**

某工厂位于2类声环境功能区，昼间实际噪声为78dB。

**超标分贝数：** 78 - 60 = 18dB

**适用税额：** 11200元/月（16-18分贝档）

**计算结果：**
```
月应纳税额 = 11200 元
季度应纳税额 = 11200 × 3 = 33600 元
```

## 六、综合计算实例

### 6.1 企业全税种计算

**【例13】某制造企业综合计算**

某制造企业2026年第一季度排放数据如下：

**一、大气污染物（1个排放口）**

| 污染物 | 月排放量（kg） | 当量值（kg） | 月当量数 |
|--------|----------------|--------------|----------|
| 二氧化硫 | 2850 | 0.95 | 3000 |
| 氮氧化物 | 1900 | 0.95 | 2000 |
| 烟尘 | 436 | 2.18 | 200 |

**二、水污染物（1个排放口）**

| 污染物 | 月排放量（kg） | 当量值（kg） | 月当量数 |
|--------|----------------|--------------|----------|
| COD | 3000 | 1 | 3000 |
| 氨氮 | 1600 | 0.8 | 2000 |
| 总磷 | 100 | 0.25 | 400 |

**三、固体废物**

| 类型 | 月排放量（吨） | 税额（元/吨） |
|------|----------------|---------------|
| 粉煤灰 | 100 | 25 |

**四、工业噪声**

位于3类区，昼间超标4dB。

**适用税额：**
- 大气污染物：6元/当量
- 水污染物：7元/当量

**计算过程：**

**1. 大气污染物（取前三项）**
```
二氧化硫：3000 × 6 = 18000 元/月
氮氧化物：2000 × 6 = 12000 元/月
烟尘：200 × 6 = 1200 元/月
大气月税额 = 31200 元
```

**2. 水污染物（取前三项）**
```
COD：3000 × 7 = 21000 元/月
氨氮：2000 × 7 = 14000 元/月
总磷：400 × 7 = 2800 元/月
水月税额 = 37800 元
```

**3. 固体废物**
```
粉煤灰月税额 = 100 × 25 = 2500 元
```

**4. 工业噪声**
```
噪声月税额 = 700 元（超标4-6分贝档）
```

**5. 季度汇总**
```
大气季度税额 = 31200 × 3 = 93600 元
水季度税额 = 37800 × 3 = 113400 元
固废季度税额 = 2500 × 3 = 7500 元
噪声季度税额 = 700 × 3 = 2100 元

季度合计 = 93600 + 113400 + 7500 + 2100 = 216600 元
```

## 七、计算注意事项

### 7.1 计算精度

| 项目 | 精度要求 |
|------|----------|
| 污染物排放量 | 保留2位小数 |
| 污染当量数 | 保留2位小数 |
| 应纳税额 | 保留2位小数 |

### 7.2 常见错误

| 错误类型 | 正确做法 |
|----------|----------|
| 未按排放口分别计算 | 每个排放口分别计算后汇总 |
| 未正确排序 | 按当量数从大到小排序 |
| 超额选取征税项目 | 大气取前3，一类水取前5，二类水取前3 |
| 昼夜噪声合并计算 | 昼夜分别计算后相加 |
| 综合利用量计入排放 | 符合标准的综合利用量不征税 |

### 7.3 申报核对清单

- [ ] 核实排放口数量
- [ ] 确认各污染物排放量
- [ ] 检查污染当量值是否正确
- [ ] 验证排序是否准确
- [ ] 确认适用税额标准
- [ ] 检查是否适用税收优惠
- [ ] 核对申报期限
