## 一、专项附加扣除详解

### 1.1 子女教育专项附加扣除

#### 1.1.1 扣除条件

| 条件项 | 具体要求 |
|--------|----------|
| 子女年龄 | 年满3周岁至小学入学前（学前教育）；接受学历教育期间 |
| 教育阶段 | 学前教育、义务教育、高中教育、高等教育（含专科、本科、硕士研究生、博士研究生） |
| 境内外 | 境内外接受教育均可 |

#### 1.1.2 扣除标准

| 项目 | 标准 |
|------|------|
| 扣除金额 | 2,000元/月/孩 |
| 扣除方式 | 父母各50%或一方100% |
| 起止时间 | 学前教育：子女年满3周岁的当月至小学入学前一月；学历教育：子女接受学历教育的当月至教育结束当月 |

#### 1.1.3 申报材料

- 子女身份证号
- 教育阶段信息
- 境外学校录取通知书、留学签证等（境外教育）

#### 1.1.4 常见问题

| 问题 | 解答 |
|------|------|
| 子女休学期间能否扣除？ | 可以，学籍保留期间可继续扣除 |
| 两个子女如何扣除？ | 每个子女均可扣除2,000元/月 |
| 离婚家庭如何扣除？ | 由子女的法定监护人扣除，可协商确定 |

### 1.2 继续教育专项附加扣除

#### 1.2.1 扣除类型

| 类型 | 范围 | 标准 | 期限 |
|------|------|------|------|
| 学历继续教育 | 专科、本科、硕士、博士在读期间 | 400元/月 | 最长48个月 |
| 职业资格继续教育 | 取得符合《国家职业资格目录》的证书 | 3,600元/年 | 取得证书当年 |

#### 1.2.2 职业资格范围

职业资格继续教育包括：
- 技能人员职业资格（如电工、焊工等）
- 专业技术人员职业资格（如教师资格、法律职业资格、注册会计师等）

#### 1.2.3 申报材料

| 类型 | 所需材料 |
|------|----------|
| 学历继续教育 | 不需要留存资料，但需备查学籍信息 |
| 职业资格继续教育 | 证书编号、发证日期等信息 |

### 1.3 大病医疗专项附加扣除

#### 1.3.1 扣除条件

| 条件项 | 具体要求 |
|--------|----------|
| 医疗费用范围 | 医保目录范围内自付部分 |
| 起扣线 | 年度累计超过15,000元 |
| 扣除上限 | 最高80,000元/年 |
| 适用对象 | 本人、配偶、未成年子女 |

#### 1.3.2 计算公式

```
可扣除金额 = MIN(医保目录内自付金额 - 15,000, 80,000)
```

#### 1.3.3 计算示例

**例1**：医保目录内自付20,000元
```
可扣除金额 = 20,000 - 15,000 = 5,000元
```

**例2**：医保目录内自付100,000元
```
可扣除金额 = MIN(100,000 - 15,000, 80,000) = 80,000元
```

#### 1.3.4 注意事项

| 事项 | 说明 |
|------|------|
| 纳税年度 | 按实际结算年度计算，非医疗发生年度 |
| 医保要求 | 必须是医保目录范围内的费用 |
| 汇算清缴 | 只能在年度汇算清缴时扣除，平时预扣预缴不能享受 |

### 1.4 住房贷款利息专项附加扣除

#### 1.4.1 扣除条件

| 条件项 | 具体要求 |
|--------|----------|
| 房屋性质 | 首套住房贷款 |
| 贷款状态 | 处于还贷期间 |
| 扣除主体 | 本人或配偶 |
| 扣除期限 | 最长240个月 |

#### 1.4.2 首套房认定

首套住房贷款是指购买住房享受首套住房贷款利率的住房贷款。

#### 1.4.3 扣除标准

| 项目 | 标准 |
|------|------|
| 扣除金额 | 1,000元/月 |
| 扣除方式 | 夫妻双方约定由一方扣除 |
| 扣除期限 | 贷款合同约定还款期的当月至贷款全部归还或贷款合同终止当月 |

#### 1.4.4 注意事项

| 事项 | 说明 |
|------|------|
| 与住房租金的关系 | 不得同时享受住房租金扣除 |
| 婚前贷款 | 夫妻双方婚前分别购买住房发生的首套住房贷款，婚后可以选择其中一套购买的住房，由购买方按扣除标准的100%扣除，也可以由夫妻双方对各自购买的住房分别按扣除标准的50%扣除 |
| 卖房后重新购房 | 再次购买住房发生的贷款，如符合首套房条件，可继续享受 |

### 1.5 住房租金专项附加扣除

#### 1.5.1 扣除条件

| 条件项 | 具体要求 |
|--------|----------|
| 工作城市 | 纳税人及配偶在主要工作城市无自有住房 |
| 租赁状态 | 处于租赁期间 |

#### 1.5.2 扣除标准

| 城市 type | 扣除标准 |
|-----------|----------|
| 直辖市、省会（首府）城市、计划单列市以及国务院确定的其他城市 | 1,500元/月 |
| 市辖区户籍人口超过100万的城市 | 1,100元/月 |
| 市辖区户籍人口不超过100万的城市 | 800元/月 |

#### 1.5.3 主要城市扣除标准参考

| 城市 | 扣除标准 |
|------|----------|
| 北京、上海、广州、深圳 | 1,500元/月 |
| 杭州、南京、成都、武汉等省会城市 | 1,500元/月 |
| 苏州、宁波、青岛等地级市 | 1,100元/月或1,500元/月（视具体城市） |

#### 1.5.4 注意事项

| 事项 | 说明 |
|------|------|
| 夫妻双方 | 主要工作城市相同的，只能由一方扣除 |
| 不同城市 | 主要工作城市不同的，可分别扣除 |
| 与房贷利息 | 不得与住房贷款利息同时享受 |

### 1.6 赡养老人专项附加扣除

#### 1.6.1 扣除条件

| 条件项 | 具体要求 |
|--------|----------|
| 老人年龄 | 年满60周岁 |
| 赡养对象 | 父母；子女均已去世的祖父母、外祖父母 |
| 赡养人 | 负有赡养义务的子女（或其他法定赡养人） |

#### 1.6.2 扣除标准

| 情形 | 扣除标准 |
|------|----------|
| 独生子女 | 3,000元/月（全额） |
| 非独生子女 | 每人最高1,500元/月，兄弟姐妹合计不超过3,000元/月 |

#### 1.6.3 分摊方式

非独生子女可通过以下方式分摊：
1. 平均分摊
2. 约定分摊
3. 指定分摊（由被赡养人指定）

#### 1.6.4 计算示例

**例**：三兄妹赡养父母，约定分摊比例为50%、30%、20%
```
大哥扣除金额 = 3,000 × 50% = 1,500元/月（达上限）
二哥扣除金额 = 3,000 × 30% = 900元/月
三弟扣除金额 = 3,000 × 20% = 600元/月
```

### 1.7 3岁以下婴幼儿照护专项附加扣除

#### 1.7.1 扣除条件

| 条件项 | 具体要求 |
|--------|----------|
| 婴幼儿年龄 | 0-3岁（不含3周岁） |
| 扣除主体 | 父母（监护人） |

#### 1.7.2 扣除标准

| 项目 | 标准 |
|------|------|
| 扣除金额 | 2,000元/月/婴 |
| 扣除方式 | 父母各50%或一方100% |
| 起止时间 | 婴幼儿出生的当月至满3周岁的前一月 |

#### 1.7.3 与子女教育衔接

- 婴幼儿满3周岁当月，3岁以下婴幼儿照护扣除停止
- 婴幼儿满3周岁当月，可开始享受子女教育扣除
- 两个扣除项目无缝衔接

## 二、全年一次性奖金计税

### 2.1 政策背景

居民个人取得全年一次性奖金，在2027年12月31日前，可以选择单独计税或并入综合所得计税。

### 2.2 单独计税方法

#### 2.2.1 计算公式

```
应纳税额 = 全年一次性奖金收入 × 适用税率 - 速算扣除数
```

#### 2.2.2 税率表（按月换算）

| 级数 | 月应纳税所得额 | 税率 | 速算扣除数 |
|------|----------------|------|------------|
| 1 | 不超过3,000元 | 3% | 0 |
| 2 | 超过3,000元至12,000元 | 10% | 210 |
| 3 | 超过12,000元至25,000元 | 20% | 1,410 |
| 4 | 超过25,000元至35,000元 | 25% | 2,660 |
| 5 | 超过35,000元至55,000元 | 30% | 4,410 |
| 6 | 超过55,000元至80,000元 | 35% | 7,160 |
| 7 | 超过80,000元 | 45% | 15,160 |

#### 2.2.3 计算步骤

1. 将全年一次性奖金除以12个月
2. 按其商数参照按月换算后的综合所得税率表确定适用税率和速算扣除数
3. 计算应纳税额

#### 2.2.4 计算示例

**例**：全年一次性奖金60,000元
```
60,000 ÷ 12 = 5,000元
对应税率10%，速算扣除数210
应纳税额 = 60,000 × 10% - 210 = 5,790元
```

### 2.3 并入综合所得计税

将全年一次性奖金并入当年综合所得，一并计算个人所得税。

### 2.4 两种方式比较

#### 2.4.1 比较示例

**基本情况**：
- 年工资收入：200,000元
- 年终奖：60,000元
- 专项扣除：30,000元
- 专项附加扣除：36,000元

**方式一：单独计税**
```
综合所得应纳税所得额 = 200,000 - 60,000 - 30,000 - 36,000 = 74,000元
综合所得税额 = 74,000 × 10% - 2,520 = 4,880元

年终奖税额 = 60,000 × 10% - 210 = 5,790元

合计税额 = 4,880 + 5,790 = 10,670元
```

**方式二：并入综合所得**
```
综合所得收入额 = 200,000 + 60,000 = 260,000元
应纳税所得额 = 260,000 - 60,000 - 30,000 - 36,000 = 134,000元
税额 = 134,000 × 10% - 2,520 = 10,880元
```

**结论**：本例中单独计税更优，可节省210元。

#### 2.4.2 选择建议

| 情形 | 建议选择 |
|------|----------|
| 综合所得适用低税率，年终奖较高 | 单独计税 |
| 综合所得扣除后较低，年终奖较低 | 并入综合所得 |
| 综合所得已适用高税率 | 单独计税 |
| 综合所得为负（扣除后） | 并入综合所得 |

### 2.5 年终奖临界点

| 年终奖金额 | 单独计税税额 | 税后实际收入 | 与上一档差额 |
|------------|--------------|--------------|--------------|
| 36,000元 | 1,080 | 34,920 | - |
| 36,001元 | 1,080.1 | 34,920.9 | +0.9 |
| 38,566元 | 3,646 | 34,920 | -0（临界点） |
| 144,000元 | 14,190 | 129,810 | - |
| 144,001元 | 14,190.1 | 129,810.9 | +0.9 |
| 160,500元 | 31,050 | 129,450 | -360（临界点） |

**注意**：年终奖存在"多发一元，少拿千元"的临界点问题，发年终奖时应避开这些区间。

### 2.6 年终奖临界区间

| 临界区间 | 说明 |
|----------|------|
| 36,001 - 38,566元 | 多发的钱不如交的税多 |
| 144,001 - 160,500元 | 多发的钱不如交的税多 |
| 300,001 - 318,333元 | 多发的钱不如交的税多 |
| 420,001 - 447,500元 | 多发的钱不如交的税多 |
| 660,001 - 706,538元 | 多发的钱不如交的税多 |
| 960,001 - 1,120,000元 | 多发的钱不如交的税多 |

## 三、其他纳税调整

### 3.1 公益慈善捐赠扣除

#### 3.1.1 扣除条件

| 条件项 | 具体要求 |
|--------|----------|
| 捐赠对象 | 符合规定的公益性社会组织、县级以上人民政府及其部门 |
| 扣除凭证 | 符合规定的公益捐赠票据 |
| 扣除限额 | 不超过应纳税所得额的30% |

#### 3.1.2 计算公式

```
可扣除捐赠额 = MIN(实际捐赠额, 应纳税所得额 × 30%)
```

#### 3.1.3 计算示例

**例**：应纳税所得额100,000元，捐赠20,000元
```
扣除限额 = 100,000 × 30% = 30,000元
实际捐赠20,000元 < 30,000元，可全额扣除
调整后应纳税所得额 = 100,000 - 20,000 = 80,000元
```

### 3.2 全年一次性奖金与其他奖金

| 奖金类型 | 计税方式 |
|----------|----------|
| 全年一次性奖金 | 可选单独计税或并入综合所得 |
| 半年奖、季度奖 | 并入当月工资薪金计税 |
| 加班奖、先进奖 | 并入当月工资薪金计税 |

### 3.3 多处取得工资的处理

居民个人从两处以上取得工资薪金所得，应将各处所得合并计算，年度汇算清缴时多退少补。

## 四、汇算清缴实务

### 4.1 需要办理汇算清缴的情形

| 序号 | 情形 |
|------|------|
| 1 | 综合所得年收入超过12万元且需要补税金额超过400元 |
| 2 | 综合所得年收入不超过12万元但已预缴税额大于应纳税额（可申请退税） |
| 3 | 取得多项综合所得，预扣预缴税率低于年度适用税率 |
| 4 | 扣除项目未在预扣预缴时充分扣除 |

### 4.2 无需办理汇算清缴的情形

| 序号 | 情形 |
|------|------|
| 1 | 综合所得年收入不超过12万元 |
| 2 | 汇算需补税金额不超过400元 |
| 3 | 已预缴税额与应纳税额一致 |

### 4.3 汇算清缴流程

1. **准备阶段**（次年1-2月）
   - 收集年度收入证明
   - 核对专项附加扣除信息
   - 确认已预缴税款

2. **申报阶段**（次年3月1日-6月30日）
   - 登录个人所得税APP或网页版
   - 进入"年度汇算"功能
   - 核对收入和扣除信息
   - 计算应补/退税额
   - 提交申报

3. **补税/退税阶段**
   - 补税：通过APP或网上银行缴款
   - 退税：确认银行账户，等待退税到账
