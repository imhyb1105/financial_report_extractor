# 企业所得税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向企业财务人员的企业所得税计算器网页应用，帮助用户快速计算企业所得税及进行纳税调整。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、会计师、税务代理人员

**核心痛点**：
- 企业所得税涉及大量纳税调整项目
- 各项扣除限额计算复杂
- 优惠政策适用条件多样

## 3. 功能需求

### 3.1 核心功能

#### 税率类型
- 法定税率（25%）
- 小型微利企业优惠税率
- 高新技术企业优惠税率（15%）

#### 纳税调整项目

| 扣除项目 | 扣除限额 |
|---------|---------|
| 职工福利费 | 工资薪金总额14% |
| 工会经费 | 工资薪金总额2% |
| 职工教育经费 | 工资薪金总额8% |
| 广告费和业务宣传费 | 销售(营业)收入15% |
| 业务招待费 | 发生额60%，且不超过收入5‰ |
| 公益性捐赠 | 年度利润总额12% |

### 3.2 税率表

| 企业类型 | 应纳税所得额 | 税率 |
|---------|-------------|------|
| 一般企业 | - | 25% |
| 高新技术企业 | - | 15% |
| 小型微利企业 | ≤100万元 | 2.5% |
| 小型微利企业 | 100-300万元 | 5% |

### 3.3 数据输入/输出

**输入数据**：
- 企业类型（必选）
- 会计利润（必填）
- 各项纳税调整项目（选填）

**输出数据**：
- 应纳税所得额
- 适用税率
- 应纳税额
- 纳税调整明细

## 5. 计算公式

### 应纳税所得额
```
应纳税所得额 = 会计利润 + 纳税调增项目 - 纳税调减项目
```

### 应纳税额
```
应纳税额 = 应纳税所得额 × 适用税率
```

### 小型微利企业
```
应纳税额 = 应纳税所得额 × 25% × 20% (≤100万元部分)
应纳税额 = 应纳税所得额 × 25% × 20% (100-300万元部分)
```
