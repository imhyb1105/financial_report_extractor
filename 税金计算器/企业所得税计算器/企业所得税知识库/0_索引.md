## 核心计算流程

企业所得税计算的核心是**应纳税所得额的计算**：

```
应纳税所得额 = 会计利润总额
            + 纳税调整增加额（限额扣除超限、不得扣除项目）
            - 纳税调整减少额（免税收入、加计扣除、亏损弥补）
```

```
应纳税额 = 应纳税所得额 x 适用税率 - 减免税额 - 抵免税额
```
