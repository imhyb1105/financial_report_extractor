# 企业所得税计算方法

**版本号：2026.03-001**

---

## 一、基本计算公式

### 1. 应纳税所得额

**直接计算法**（《企业所得税法》第五条）：

```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 允许弥补的以前年度亏损
```

**间接计算法**（常用）：

```
应纳税所得额 = 会计利润总额 + 纳税调整增加额 - 纳税调整减少额
```

### 2. 应纳税额

**基本公式**（《企业所得税法》第三条）：

```
应纳税额 = 应纳税所得额 x 适用税率 - 减免税额 - 抵免税额
```

### 3. 实际税负率

```
实际税负率 = 应纳税额 / 收入总额 x 100%
```

---

## 二、纳税调整项目

### 纳税调整增加额

| 调整项目 | 说明 |
|----------|------|
| 限额扣除超限部分 | 业务招待费、广告费等超过扣除限额的部分 |
| 不得扣除项目 | 罚款、滞纳金、赞助支出等 |

### 纳税调整减少额

| 调整项目 | 说明 |
|----------|------|
| 免税收入 | 国债利息、股息红利等 |
| 不征税收入 | 财政拨款等 |
| 加计扣除 | 研发费用、残疾人工资等 |
| 亏损弥补 | 以前年度亏损弥补 |

## 三、各类企业计算示例

### 1. 一般企业（25%税率）

**例1**：某一般企业2026年度数据

| 项目 | 金额（万元） |
|------|--------------|
| 收入总额 | 1,000 |
| 成本费用 | 700 |
| 纳税调整增加 | 50 |

**计算**：

```
应纳税所得额 = 1,000 - 700 + 50 = 350万元
应纳税额 = 350 x 25% = 87.5万元
实际税负率 = 87.5 / 1,000 = 8.75%
```

---

### 2. 小型微利企业

**例2**：年应纳税所得额80万元（不超过100万）

```
应纳税额 = 80 x 25% x 20% x 50% = 2万元
实际税负率 = 2 / 80 = 2.5%
```

**例3**：年应纳税所得额200万元（100万-300万区间）

```
不超过100万部分：100 x 25% x 20% x 50% = 2.5万元
超过100万部分：(200-100) x 25% x 20% = 5万元
应纳税额合计 = 2.5 + 5 = 7.5万元
实际税负率 = 7.5 / 200 = 3.75%
```

**例4**：年应纳税所得额320万元（超过300万临界点）

```
警告：超过300万元，全部所得按一般企业25%税率计算
应纳税额 = 320 x 25% = 80万元
实际税负率 = 80 / 320 = 25%
```

**临界点对比**：

| 应纳税所得额 | 应纳税额 | 实际税负 |
|--------------|----------|----------|
| 300万元 | 12.5万元 | 4.17% |
| 301万元 | 75.25万元 | 25% |

**结论**：超过1万元，多缴税约62.75万元！

---

### 3. 高新技术企业（15%税率）

**例5**：某高新技术企业

| 项目 | 金额（万元） |
|------|--------------|
| 年应纳税所得额 | 500 |

```
应纳税额 = 500 x 15% = 75万元
实际税负率 = 75 / 500 = 15%

相比一般企业节税 = 500 x (25% - 15%) = 50万元
```

---

### 4. 研发费用加计扣除

**例6**：某企业

| 项目 | 金额（万元） |
|------|--------------|
| 会计利润 | 300 |
| 研发费用 | 100（已计入成本费用） |

```
研发费用加计扣除 = 100 x 100% = 100万元
应纳税所得额 = 300 - 100 = 200万元

按一般企业25%计算：
应纳税额 = 200 x 25% = 50万元

按高新技术企业15%计算：
应纳税额 = 200 x 15% = 30万元
```

---

### 5. 公益性捐赠扣除

**例7**：某企业

| 项目 | 金额（万元） |
|------|--------------|
| 年度利润总额 | 500 |
| 公益性捐赠 | 80 |

```
扣除限额 = 500 x 12% = 60万元
本年扣除 = 60万元（限额内）
结转以后年度扣除 = 80 - 60 = 20万元（可在以后三年内扣除）
纳税调整增加 = 20万元
```

---

### 6. 亏损弥补

**例8**：某企业盈亏情况

| 年度 | 盈亏（万元） | 说明 |
|------|--------------|------|
| 2023年 | -100 | 亏损 |
| 2024年 | 30 | 弥补2023年亏损 |
| 2025年 | 50 | 弥补2023年亏损 |
| 2026年 | 80 | 部分弥补2023年亏损 |

```
2024年弥补2023年亏损：30万元，剩余亏损70万元
2025年弥补2023年亏损：50万元，剩余亏损20万元
2026年弥补2023年亏损：20万元，剩余应纳税所得额60万元
2026年应纳税额 = 60 x 25% = 15万元
```

## 四、综合计算示例

**例9**：某高新技术企业2026年度完整计算

### 基础数据

| 项目 | 金额（万元） |
|------|--------------|
| 收入总额 | 2,000 |
| 成本费用 | 1,500（含研发费用200） |
| 公益性捐赠 | 30 |
| 2025年亏损 | 50 |
| 业务招待费 | 20（实际发生） |
| 广告费 | 350（实际发生） |

### 计算步骤

**第一步：计算会计利润**

```
会计利润 = 2,000 - 1,500 = 500万元
```

**第二步：计算纳税调整**

```
1. 业务招待费调整：
   60%计算 = 20 x 60% = 12万元
   5‰限额 = 2,000 x 5‰ = 10万元
   扣除限额 = 10万元（取低）
   调增 = 20 - 10 = 10万元

2. 广告费调整：
   扣除限额 = 2,000 x 15% = 300万元
   调增 = 350 - 300 = 50万元（可结转）

3. 公益性捐赠：
   扣除限额 = 500 x 12% = 60万元
   实际30万元，未超限，无需调整

4. 研发费用加计扣除：
   加计扣除 = 200 x 100% = 200万元
   调减 = 200万元
```

**第三步：计算应纳税所得额**

```
应纳税所得额 = 500 + 10 + 50 - 200 = 360万元
```

**第四步：弥补亏损**

```
弥补2025年亏损后 = 360 - 50 = 310万元
```

**第五步：计算应纳税额**

```
应纳税额 = 310 x 15% = 46.5万元
```

**第六步：计算节税效果**

```
- 研发加计扣除节税 = 200 x 15% = 30万元
- 高新技术优惠节税 = 310 x (25% - 15%) = 31万元
- 合计节税 = 61万元
```

## 五、税负率参考

### 计算公式

```
实际税负率 = 应纳税额 / 收入总额 x 100%
```

### 行业参考税负率

| 行业 | 参考税负率 |
|------|------------|
| 制造业 | 2% - 4% |
| 商贸企业 | 1% - 2% |
| 服务业 | 3% - 5% |
| 房地产 | 5% - 8% |
| 高新技术企业 | 2% - 4% |

**注意**：税负率仅供参考，实际税负受多种因素影响，如享受的优惠政策、成本结构等。税负率过低可能引起税务机关关注。

            |         |

## 七、常见问题

### Q1：小微企业超过300万怎么办？

**答**：年应纳税所得额超过300万元，则**全部所得**按一般企业25%税率计算，不是仅对超限部分按25%计算。

### Q2：研发费用加计扣除需要备案吗？

**答**：目前研发费用加计扣除采用"自行判别、申报享受、相关资料留存备查"的方式，无需事前备案，但需留存相关资料备查。

### Q3：亏损弥补年限从哪年算起？

**答**：亏损弥补年限从亏损年度的次年开始计算，连续计算，不论期间是否有盈利或亏损。

### Q4：高新技术企业认定后有效期多久？

**答**：高新技术企业资格自颁发证书之日起有效期为三年，期满前需重新认定。

### Q5：可以同时享受小微企业和高新技术企业优惠吗？

**答**：不可以。两种优惠只能选择其一。企业应根据自身情况选择最有利的优惠方式。
