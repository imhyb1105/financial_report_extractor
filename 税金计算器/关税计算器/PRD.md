# 关税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向进出口企业的关税计算器网页应用，帮助用户快速计算进出口货物应缴纳的关税。

## 2. 用户画像 (Persona)

**主要用户**：进出口企业关务人员、外贸财务人员、报关行工作人员

**核心痛点**：
- 关税税率随商品编码变化
- 存在从价、从量、复合三种计征方式
- 完税价格确定方法多样
- 自贸协定优惠税率适用复杂

## 3. 功能需求

### 3.1 核心功能

#### 功能模块（Tab切换）
- 进口关税计算
- 出口关税计算
- 完税价格计算

### 3.2 主要税率参考

| 商品类别 | 计征方式 | 税率范围 |
|---------|---------|---------|
| 电子产品 | 从价 | 0-20% |
| 汽车 | 从价 | 15-25% |
| 奢侈品 | 从价 | 10-30% |
| 原材料 | 从价 | 0-10% |
| 稀土(出口) | 从价 | 20% |

### 3.3 数据输入/输出

**输入数据**：
- 进出口类型（必选）
- 计征方式（必选）
- 完税价格/货物数量（必填）
- 关税税率/单位税额（必填）

**输出数据**：
- 关税税额
- 完税价格
- 计算明细

## 5. 计算公式

### 从价计征
```
关税 = 完税价格 × 关税税率
```

### 从量计征
```
关税 = 货物数量 × 单位税额
```

### 复合计征
```
关税 = 完税价格 × 关税税率 + 货物数量 × 单位税额
```

### 完税价格
```
完税价格 = CIF价格（进口）
完税价格 = FOB价格 - 出口关税（出口）
```
