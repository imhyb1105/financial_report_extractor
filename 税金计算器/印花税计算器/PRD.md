# 印花税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向财务人员和办税人员的印花税计算器网页应用，帮助用户快速计算各类合同、产权转移书据、营业账簿等应税凭证的印花税税额。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、办税人员、合同管理员、会计师

**核心痛点**：
- 印花税税目繁多，税率容易混淆
- 不同类型合同适用不同税率
- 计算增值税是否包含容易出错

## 3. 功能需求

### 3.1 核心功能

#### 税目选择
- 借款合同（0.05‰）
- 买卖合同（0.3‰）
- 技术合同（0.3‰）
- 租赁合同（1‰）
- 承揽合同（0.3‰）
- 运输合同（0.3‰）
- 仓储合同（1‰）
- 保管合同（1‰）
- 融资租赁合同（0.05‰）
- 产权转移书据（0.5‰）
- 营业账簿（0.25‰）
- 定额贴花（5元/件）

### 3.2 数据输入/输出

**输入数据**：
- 应税凭证类型（必选）
- 金额/件数（必填）
- 是否含增值税（可选）

**输出数据**：
- 应纳税额
- 计算明细
- 适用税率说明

## 5. 计算公式

### 基本计算公式
```
应纳税额 = 计税依据 × 适用税率
```

### 增值税处理
```
如金额含增值税：计税依据 = 含税金额 ÷ (1 + 增值税率)
如金额不含增值税：计税依据 = 金额
```
