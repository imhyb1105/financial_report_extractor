# 印花税基础知识

**版本：2026.03-001**
**更新日期：2026年3月**

---

## 概念定义

### 什么是印花税

印花税是对经济活动和经济交往中书立、领受、使用的应税经济凭证所征收的一种税。它是一种行为税，以凭证为征税对象。

### 印花税的特点

| 特点 | 说明 |
|------|------|
| **凭证税** | 以各类经济凭证为征税对象 |
| **行为税** | 对书立、领受凭证的行为征税 |
| **税率低** | 税负相对较轻，税率通常在万分之几到千分之几 |
| **自行贴花** | 纳税人自行计算、自行购买、自行贴花注销 |
| **征收面广** | 涉及经济活动的多个领域 |

## 纳税人

### 纳税人定义

在中华人民共和国境内书立、领受、使用应税凭证的单位和个人，为印花税的纳税人。

### 纳税人类型

| 类型 | 说明 |
|------|------|
| **立合同人** | 书立各类应税合同的当事人（双方或多方均为纳税人） |
| **立据人** | 书立产权转移书据的当事人 |
| **立账簿人** | 设立并使用营业账簿的单位和个人 |
| **领受人** | 领取权利许可证照的单位和个人 |
| **使用人** | 在国外书立、领受，但在国内使用的应税凭证使用人 |

### 纳税人身份

- **单位**：企业、行政单位、事业单位、军事单位、社会团体及其他单位
- **个人**：个体工商户和其他个人

### 特殊情况

| 情形 | 纳税人确定 |
|------|------------|
| 合同双方当事人 | 双方均为纳税人，各自就所持合同金额纳税 |
| 代理人签订合同 | 代理人有代理纳税义务 |
| 保人、证人、鉴定人 | 不属于纳税人 |
| 电子凭证 | 以电子形式签订的各类应税凭证，用户为纳税人 |

## 征税范围

### 应税凭证分类

根据《中华人民共和国印花税法》，应税凭证分为以下几类：

#### 一、合同类

| 序号 | 合同类型 | 说明 |
|------|----------|------|
| 1 | 买卖合同 | 即购销合同，包括供应、预购、采购、购销结合及协作、调剂等合同 |
| 2 | 承揽合同 | 即加工承揽合同，包括加工、定作、修缮、修理、印刷、广告等合同 |
| 3 | 建设工程合同 | 包括工程勘察、设计、施工合同 |
| 4 | 运输合同 | 包括货运合同和多式联运合同（不含管道运输） |
| 5 | 技术合同 | 包括技术开发、转让、咨询、服务等合同 |
| 6 | 租赁合同 | 即财产租赁合同 |
| 7 | 保管合同 | 即仓储保管合同 |
| 8 | 仓储合同 | 即仓储保管合同 |
| 9 | 借款合同 | 包括银行及其他金融组织和借款人之间的借款合同 |
| 10 | 融资租赁合同 | 融资租赁业务中订立的合同 |
| 11 | 保险合同 | 即财产保险合同 |

#### 二、产权转移书据

| 序号 | 书据类型 | 说明 |
|------|----------|------|
| 1 | 土地使用权出让书据 | 土地使用权出让合同 |
| 2 | 土地使用权转让书据 | 土地使用权出售、赠与、互换合同 |
| 3 | 房屋等建筑物转让书据 | 房屋买卖、赠与、互换合同 |
| 4 | 股权转让书据 | 包括上市和非上市公司股权转让 |
| 5 | 商标专用权转让书据 | 商标权转让合同 |
| 6 | 著作权转让书据 | 著作权转让合同 |
| 7 | 专利权转让书据 | 专利权转让合同 |
| 8 | 专有技术使用权转让书据 | 非专利技术转让合同 |

#### 三、营业账簿

| 类型 | 说明 |
|------|------|
| 资金账簿 | 记载实收资本（股本）、资本公积合计金额的账簿 |
| 其他账簿 | 除资金账簿以外的其他营业账簿 |

#### 四、证券交易

在证券交易所、国务院批准的其他全国性证券交易场所交易的股票和以股票为基础的存托凭证。

#### 五、权利许可证照

| 类型 | 说明 |
|------|------|
| 房屋产权证 | 房屋所有权证 |
| 工商营业执照 | 企业法人营业执照等 |
| 商标注册证 | 商标注册证 |
| 专利证 | 专利证书 |
| 土地使用证 | 国有土地使用证 |

## 不征税范围

### 不属于印花税征税范围的凭证

| 类型 | 说明 |
|------|------|
| 非应税合同 | 如劳动合同、行政合同等 |
| 非应税书据 | 如赠与合同（非产权转移）、继承文书等 |
| 非营业账簿 | 如备查账、临时账簿等 |
| 非权利许可证照 | 如卫生许可证、安全生产许可证等 |

### 具体不征税项目

1. **既无金额又无数量的凭证**：如空白凭证、作废凭证
2. **副本或抄本**：凭证的副本或抄本，不作为正本使用
3. **内部凭证**：单位内部使用的凭证，不对外发生效力
4. **行政管理合同**：如行政管理、军事、外交等方面的合同

## 纳税义务发生时间

| 凭证类型 | 纳税义务发生时间 |
|----------|------------------|
| 应税合同 | 合同书立之日 |
| 产权转移书据 | 书据书立之日 |
| 营业账簿 | 账簿启用之日 |
| 权利许可证照 | 证照领受之日 |
| 证券交易 | 交易完成之日 |

---

## 纳税期限

| 纳税方式 | 期限要求 |
|----------|----------|
| 按期汇总缴纳 | 最长不超过1个月 |
| 按次缴纳 | 纳税义务发生之日起15日内 |

---

## 纳税地点

| 情形 | 纳税地点 |
|------|----------|
| 单位 | 机构所在地 |
| 个人 | 凭证书立地或领受地 |
| 跨地区合同 | 各自所在地 |
| 证券交易 | 证券登记结算机构所在地 |
