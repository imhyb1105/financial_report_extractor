# 印花税计算方法

**版本：2026.03-001**
**更新日期：2026年3月**

---

## 基本计算公式

### 比例税率计算公式

```
应纳税额 = 计税金额 × 税率
```

**说明**：
- 计税金额通常为合同所载金额（不含增值税）
- 税率以百分比形式计算

### 定额税率计算公式

```
应纳税额 = 应税凭证件数 × 定额税率
```

**说明**：
- 定额税率为每件5元
- 适用于权利许可证照

---

## 计算步骤

### 步骤一：确定凭证类型

首先判断凭证属于哪一类型：
1. 合同类（买卖、承揽、建设工程、运输、技术、租赁、保管、仓储、借款、融资租赁、保险）
2. 产权转移书据
3. 营业账簿
4. 权利许可证照

### 步骤二：确定适用税率

根据凭证类型，查找对应的适用税率（详见[印花税税率表](./2_印花税税率.md)）

### 步骤三：确定计税依据

| 凭证类型 | 计税依据 |
|----------|----------|
| 合同 | 合同金额（不含增值税） |
| 产权转移书据 | 成交金额（不含增值税） |
| 营业账簿 | 实收资本 + 资本公积 |
| 权利许可证照 | 件数 |

### 步骤四：计算应纳税额

```
应纳税额 = 计税依据 × 税率
```

### 步骤五：考虑优惠政策

检查是否适用减免优惠政策，如有则按优惠后金额计算

## 计算实例

### 实例1：买卖合同（购销合同）

**场景**：A公司与B公司签订一份货物买卖合同，合同金额为1,000,000元（不含税）

**计算**：
- 凭证类型：买卖合同
- 适用税率：0.03%
- 计税依据：1,000,000元
- 应纳税额 = 1,000,000 × 0.03% = 300元

**结果**：A公司应缴纳印花税300元，B公司应缴纳印花税300元

---

### 实例2：借款合同

**场景**：某企业向银行借款5,000,000元，期限一年

**计算**：
- 凭证类型：借款合同
- 适用税率：0.005%
- 计税依据：5,000,000元
- 应纳税额 = 5,000,000 × 0.005% = 250元

**结果**：企业应缴纳印花税250元

---

### 实例3：财产租赁合同

**场景**：某公司租赁办公用房，年租金为600,000元，租期3年

**计算**：
- 凭证类型：租赁合同
- 适用税率：0.1%
- 计税依据：600,000 × 3 = 1,800,000元

**结果**：应缴纳印花税1,800元

**注意**：租赁合同以租金总额为计税依据

## 特殊情况计算

### 同一合同涉及多个事项

**场景**：某合同同时包含技术服务和设备销售，技术服务费300,000元，设备价款700,000元，金额分别列明

**计算**：
- 技术服务部分：300,000 × 0.03% = 90元
- 设备销售部分：700,000 × 0.03% = 210元
- 合计应纳税额 = 90 + 210 = 300元

**注意**：如果金额未分别列明，则按较高税率（0.05%）计税

---

### 合同金额含增值税

**场景**：某买卖合同金额为1,130,000元（含税），增值税税率13%

**计算方法一**（合同分别列明不含税价和税额）：
- 不含税价：1,130,000 ÷ 1.13 = 1,000,000元
- 应纳税额 = 1,000,000 × 0.03% = 300元

**计算方法二**（合同未分别列明）：
- 按全额计税：1,130,000 × 0.03% = 339元

**建议**：签订合同时分别列明不含税价和税额，可降低印花税负担

---

### 合同金额为外币

**场景**：某进出口合同金额为100,000美元，汇率1:7.2

**计算**：
- 折合人民币：100,000 × 7.2 = 720,000元
- 应纳税额 = 720,000 × 0.03% = 216元

**注意**：以外币计价的合同，应按照书立合同当日国家外汇管理局公布的外汇牌价折合人民币计税

---

### 小微企业优惠计算

**场景**：某增值税小规模纳税人签订买卖合同，合同金额500,000元，当地规定可享受50%减免

**计算**：
- 正常应纳税额 = 500,000 × 0.03% = 150元
- 减免后应纳税额 = 150 × 50% = 75元

**结果**：实际应缴纳印花税75元

## 计算注意事项

### 计税依据注意事项

1. **增值税处理**：合同所列金额不包含增值税的，以不含税金额为计税依据；未分别列明的，以全额为计税依据

2. **尾数处理**：应纳税额不足1角的，免纳印花税；应纳税额在1角以上的，其税额尾数不满5分的不计，满5分的按1角计算

3. **外币折算**：以外币计价的，按凭证书立当日外汇牌价折合人民币计税

### 合同签订注意事项

1. **双方纳税**：合同双方均为纳税人，各自按合同金额全额计税

2. **合同变更**：合同金额增加的，增加部分补贴印花；减少的，已贴印花不退

3. **合同解除**：合同签订后解除的，已贴印花不予退还

### 优惠政策注意事项

1. **优惠资格**：确认是否符合优惠条件

2. **优惠备案**：部分优惠需要事先备案或留存相关资料

3. **政策时效**：注意优惠政策的有效期限
