# 城镇土地使用税计算方法

## 计算公式

### 年应纳税额
```
年应纳税额 = 计税土地面积（㎡）× 适用税额（元/㎡）
```

### 月应纳税额
```
月应纳税额 = 年应纳税额 ÷ 12
```

## 计算实例

### 实例1
某企业位于大城市市区，实际占用土地面积5000平方米，当地适用税额15元/㎡。

```
年应纳税额 = 5,000 × 15 = 75,000元
月应纳税额 = 75,000 ÷ 12 = 6,250元
```

### 实例2
某工厂位于县城，实际占用土地面积10000平方米，当地适用税额5元/㎡。

```
年应纳税额 = 10,000 × 5 = 50,000元
月应纳税额 = 50,000 ÷ 12 = 4,166.67元
```
