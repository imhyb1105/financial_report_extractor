# 契税税率

**版本：2026.03-001**
**更新日期：2026年3月**

---

## 一、法定税率范围

根据《中华人民共和国契税法》规定，契税的法定税率为**3% - 5%**。

各省、自治区、直辖市可在上述税率幅度内，提出本地区具体适用税率，报同级人民代表大会常务委员会决定，并报全国人民代表大会常务委员会和国务院备案。

## 二、首套房与二套房认定标准

### 2.1 首套房认定标准

| 条件 | 说明 |
|------|------|
| 无房记录 | 家庭成员（包括借款人、配偶及未成年子女）名下无住房 |
| 无贷款记录 | 无住房贷款记录（部分地区以不动产登记查询结果为准） |

### 2.2 二套房认定标准

符合以下任一条件即认定为二套房：

| 情形 | 说明 |
|------|------|
| 已有一套房 | 家庭成员名下已有一套住房 |
| 有贷款记录 | 曾有住房贷款记录（即使已还清） |
| 异地购房 | 在不同城市拥有住房（部分城市按属地原则认定） |

### 2.3 三套及以上

家庭成员名下已有两套及以上住房，再购房属于三套及以上。

## 五、税率适用注意事项

### 5.1 面积认定标准

| 项目 | 说明 |
|------|------|
| 面积来源 | 以不动产权证书或房屋买卖合同记载的建筑面积为准 |
| 建筑面积 | 包括套内建筑面积和分摊的共有建筑面积 |
| 车库/储藏室 | 单独计算，不计入住房面积 |

### 5.2 税率适用顺序

```
判断顺序：
1. 是否属于法定免税情形 → 是：免税
                        → 否：继续判断
2. 是否为法定继承/夫妻转移 → 是：免税
                           → 否：继续判断
3. 购房套数认定 → 首套房/二套房/三套及以上
4. 面积判断 → ≤90㎡ 或 ＞90㎡
5. 确定适用税率
```

### 5.3 常见问题

**Q1：离婚后购房是否算首套房？**

A：需查询个人名下房产情况，如名下无房，可按首套房计算。

**Q2：出售原有住房后再购买，是否算首套房？**

A：如已办理完出售手续且名下无房，可按首套房计算（具体以当地政策为准）。

**Q3：商业贷款已还清，再次购房契税如何计算？**

A：契税按名下房产套数认定，与贷款是否还清无关。

**Q4：不同城市的房产如何认定套数？**

A：部分城市实行属地认定（只查本市），部分城市联网查询，以当地政策为准。

## 六、税率历史变化

| 时间 | 政策变化 |
|------|----------|
| 2021年9月1日 | 《契税法》正式实施，原《契税暂行条例》废止 |
| 2016年2月 | 调整个人购买住房契税优惠政策（现行标准） |
| 2010年10月 | 首套房90㎡以下减按1%征收 |
| 2008年11月 | 首套房契税税率下调至1% |
