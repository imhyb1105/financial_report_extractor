# 房产税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向企业财务人员和房产投资者的房产税计算器网页应用，帮助用户快速计算从价计征和从租计征的房产税金额。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、房产投资者、会计师

**核心痛点**：
- 房产税有两种计征方式，容易混淆
- 扣除比例各地不同
- 自用与出租的计算方式不同

## 3. 功能需求

### 3.1 核心功能

#### 计征方式选择
- 从价计征（自用房产）
- 从租计征（出租房产）

### 3.2 税率表

| 计征方式 | 税率 | 计税依据 |
|---------|------|----------|
| 从价计征 | 1.2% | 房产原值 × (1 - 扣除比例) |
| 从租计征 | 12% | 租金收入 |

### 3.3 数据输入/输出

**输入数据**：
- 计征方式（必选）
- 房产原值/租金收入（必填）
- 扣除比例（必填，从价计征时）
- 计税期限（月/年）

**输出数据**：
- 应纳税额
- 计算明细
- 适用税率说明

## 5. 计算公式

### 从价计征
```
年应纳税额 = 应税房产原值 × (1 - 扣除比例) × 1.2%
```

### 从租计征
```
年应纳税额 = 租金收入 × 12%
```

## 6. 政策依据

- 《中华人民共和国房产税暂行条例》
- 各省、自治区、直辖市房产税实施细则
