# 房产税基础知识

**版本：2026.03-001**
**更新日期：2026年3月**

## 一、房产税定义

房产税是以房屋为征税对象，按房屋的计税余值或租金收入为计税依据，向产权所有人征收的一种财产税。

### 1.1 房产税的特点

| 特点 | 说明 |
|------|------|
| **财产税性质** | 对纳税人拥有的财产征税 |
| **地方税种** | 收入归地方政府所有 |
| **征税对象特定** | 仅对房屋征税 |
| **征税范围限定** | 仅在城市、县城、建制镇和工矿区征收 |

### 1.2 立法目的

- 筹集地方财政收入
- 调节财富分配
- 促进房产资源合理配置

## 二、纳税人

### 2.1 纳税义务人

房产税由产权所有人缴纳。具体规定如下：

| 情形 | 纳税人 |
|------|--------|
| 产权属于国家所有 | 经营管理单位 |
| 产权属于集体和个人所有 | 集体单位和个人 |
| 产权出典 | 承典人 |
| 产权所有人、承典人不在房产所在地 | 房产代管人或使用人 |
| 产权未确定及租典纠纷未解决 | 房产代管人或使用人 |
| 纳税单位无租使用免税单位房产 | 使用人代缴 |

### 2.2 纳税人分类

```
纳税人
├── 产权所有人（主要纳税人）
├── 经营管理单位（国有房产）
├── 承典人（出典房产）
└── 代管人/使用人（特殊情形）
```

## 三、征税范围

### 3.1 征税区域

房产税的征税范围限于以下区域：

| 区域类型 | 说明 |
|----------|------|
| **城市** | 经国务院批准设立的市 |
| **县城** | 县人民政府所在地 |
| **建制镇** | 经省人民政府批准设立的建制镇 |
| **工矿区** | 工商业比较发达、人口比较集中、符合国务院规定的建制镇标准，但尚未设立建制镇的大中型工矿企业所在地 |

### 3.2 不在征税范围的区域

- 农村地区
- 不符合上述条件的其他地区

### 3.3 征税对象

**征税对象**：房屋

**房屋的定义**：
- 有屋面和围护结构（有墙或两边有柱）
- 能够遮风避雨
- 可供人们在其中生产、工作、学习、娱乐、居住或储藏物资的场所

### 3.4 独立于房屋之外的建筑物

以下建筑物不属于房产税征税对象：

- 围墙
- 烟囱
- 水塔
- 变电塔
- 油池油柜
- 酒窖菜窖
- 酒精池
- 糖蜜池
- 室外游泳池
- 玻璃暖房
- 砖瓦石灰窑
- 各种油气罐
- 栈桥

## 四、计税依据

### 4.1 两种计税方式

| 计税方式 | 适用情形 | 计税依据 |
|----------|----------|----------|
| **从价计征** | 经营自用的房产 | 房产原值减除一定比例后的余值 |
| **从租计征** | 出租的房产 | 房产租金收入 |

### 4.2 房产原值的确定

房产原值是指纳税人按照会计制度规定，在账簿"固定资产"科目中记载的房屋原价。

**房产原值应包括：**

1. 房屋本身的价值
2. 与房屋不可分割的各种附属设备或一般不单独计算价值的配套设施
3. 附属设备包括：
   - 暖气、卫生、通风、照明、煤气等设备
   - 各种管线、石油压送、蒸汽、压缩空气、石油、给水排水等管道及电力、电讯、电缆导线
   - 电梯、升降机、过道、晒台等

### 4.3 房产余值的计算

**房产余值 = 房产原值 x (1 - 减除比例)**

- 减除比例由省、自治区、直辖市人民政府规定
- 一般为10%-30%
- 常见比例为30%

### 4.4 租金收入

租金收入是指房屋产权所有人出租房产使用权所得的报酬，包括：

- 货币收入
- 实物收入
- 其他经济利益

## 五、纳税义务发生时间

| 情形 | 纳税义务发生时间 |
|------|------------------|
| 纳税人将原有房产用于生产经营 | 从生产经营之月起 |
| 纳税人自行新建房屋用于生产经营 | 从建成之次月起 |
| 纳税人委托施工企业建设的房屋 | 从办理验收手续之次月起 |
| 纳税人购置新建商品房 | 自房屋交付使用之次月起 |
| 纳税人购置存量房 | 自办理房屋权属转移、变更登记手续，房地产权属登记机关签发房屋权属证书之次月起 |
| 纳税人出租、出借房产 | 自交付出租、出借房产之次月起 |
| 房地产开发企业自用、出租、出借本企业建造的商品房 | 自房屋使用或交付之次月起 |

## 六、常见问题

### Q1：房屋装修费用是否计入房产原值？

A：需要区分情况：
- 新建房屋的装修费用，凡属于房屋不可分割的附属设备，应计入房产原值
- 旧房装修费用，一般不计入房产原值

### Q2：地下建筑是否征收房产税？

A：凡在房产税征收范围内的具备房屋功能的地下建筑，包括与地上房屋相连的地下建筑以及完全建在地面以下的建筑、地下人防设施等，均应当征收房产税。

### Q3：房产原值如何调整？

A：房产原值发生增减变化时，应按规定及时调整房产原值，如：
- 对原有房屋进行改建、扩建
- 更换房屋附属设备和配套设施
- 房屋部分拆除
