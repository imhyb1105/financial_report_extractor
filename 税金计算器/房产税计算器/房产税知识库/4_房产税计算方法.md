# 房产税计算方法

**版本：2026.03-001**
**更新日期：2026年3月**

---

## 一、计算公式

### 1.1 从价计征计算公式

**年应纳税额 = 房产原值 x (1 - 减除比例) x 1.2%**

| 要素 | 说明 |
|------|------|
| 房产原值 | 会计账簿"固定资产"科目记载的房屋原价 |
| 减除比例 | 由各省规定，一般为10%-30% |
| 税率 | 年税率1.2% |

**其他期间计算：**
- 季度应纳税额 = 年应纳税额 / 4
- 月应纳税额 = 年应纳税额 / 12
- 半年应纳税额 = 年应纳税额 / 2

### 1.2 从租计征计算公式

**应纳税额 = 租金收入 x 12%**

| 要素 | 说明 |
|------|------|
| 租金收入 | 出租房产取得的全部收入 |
| 税率 | 12%（个人出租住房4%） |

### 1.3 公式对比

| 计税方式 | 计算公式 | 适用情形 |
|----------|----------|----------|
| 从价计征 | 房产原值 x (1-减除比例) x 1.2% | 自用房产 |
| 从租计征 | 租金收入 x 12% | 出租房产 |

## 二、计算实例

### 2.1 从价计征实例

#### 【实例1】企业经营自用房产

**基本情况：**
- 某公司拥有办公用房，房产原值800万元
- 当地减除比例为30%
- 房产全部自用

**计算过程：**

```
步骤1：确定房产原值
房产原值 = 8,000,000 元

步骤2：计算房产余值
房产余值 = 8,000,000 x (1 - 30%) = 5,600,000 元

步骤3：计算年应纳税额
年应纳税额 = 5,600,000 x 1.2% = 67,200 元

步骤4：计算各期间应纳税额
月应纳税额 = 67,200 / 12 = 5,600 元
季应纳税额 = 67,200 / 4 = 16,800 元
```

**结果汇总：**

| 期间 | 应纳税额 |
|------|----------|
| 年 | 67,200 元 |
| 半年 | 33,600 元 |
| 季 | 16,800 元 |
| 月 | 5,600 元 |

---

#### 【实例2】部分自用部分出租

**基本情况：**
- 某企业房产原值1200万元
- 当地减除比例为30%
- 60%自用，40%出租
- 年租金收入50万元

**计算过程：**

```
一、自用部分（从价计征）
自用部分房产原值 = 12,000,000 x 60% = 7,200,000 元
自用部分房产余值 = 7,200,000 x (1 - 30%) = 5,040,000 元
自用部分年应纳税额 = 5,040,000 x 1.2% = 60,480 元

二、出租部分（从租计征）
出租部分年应纳税额 = 500,000 x 12% = 60,000 元

三、合计年应纳税额
年应纳税额 = 60,480 + 60,000 = 120,480 元
```

**结果汇总：**

| 部分 | 计税方式 | 计税依据 | 税率 | 应纳税额 |
|------|----------|----------|------|----------|
| 自用部分 | 从价计征 | 5,040,000元 | 1.2% | 60,480元 |
| 出租部分 | 从租计征 | 500,000元 | 12% | 60,000元 |
| **合计** | - | - | - | **120,480元** |

---

### 2.2 从租计征实例

#### 【实例3】企业出租房产

**基本情况：**
- 某企业将闲置厂房出租
- 月租金收入10万元
- 租期1年

**计算过程：**

```
年租金收入 = 100,000 x 12 = 1,200,000 元
年应纳税额 = 1,200,000 x 12% = 144,000 元
月应纳税额 = 144,000 / 12 = 12,000 元
```

---

#### 【实例4】个人出租住房

**基本情况：**
- 张先生将个人住房出租
- 月租金收入5,000元

**计算过程：**

```
年租金收入 = 5,000 x 12 = 60,000 元
年应纳税额 = 60,000 x 4% = 2,400 元
月应纳税额 = 2,400 / 12 = 200 元
```

---

### 2.3 特殊情形实例

#### 【实例5】地下建筑

**基本情况：**
- 某企业地下车库，原值200万元
- 用于商业用途
- 当地规定地下商业建筑按原值70%计算

**计算过程：**

```
应税房产原值 = 2,000,000 x 70% = 1,400,000 元
房产余值 = 1,400,000 x (1 - 30%) = 980,000 元
年应纳税额 = 980,000 x 1.2% = 11,760 元
```

---

#### 【实例6】融资租赁房产

**基本情况：**
- 某企业以融资租赁方式租入厂房
- 房产原值500万元
- 融资租赁合同约定开始日为2026年1月1日
- 当地减除比例30%

**计算过程：**

```
房产余值 = 5,000,000 x (1 - 30%) = 3,500,000 元
年应纳税额 = 3,500,000 x 1.2% = 42,000 元
```

**注意：** 从合同约定开始日的次月起纳税，即2026年2月起纳税。

## 三、计算步骤详解

### 3.1 从价计征计算步骤

```
第一步：确定房产原值
    ↓
第二步：确定减除比例
    ↓
第三步：计算房产余值
    房产余值 = 房产原值 x (1 - 减除比例)
    ↓
第四步：计算年应纳税额
    年应纳税额 = 房产余值 x 1.2%
    ↓
第五步：计算各期间应纳税额
    根据当地纳税期限要求计算
```

### 3.2 从租计征计算步骤

```
第一步：确定租金收入
    ↓
第二步：确定适用税率
    一般房产：12%
    个人出租住房：4%
    ↓
第三步：计算应纳税额
    应纳税额 = 租金收入 x 税率
```

## 四、常见计算问题

### 4.1 房产原值变化如何计算？

**情形：** 房产原值在年度中间发生变化

**处理方法：**
- 按月计算，发生变化的当月起按新原值计算
- 或按实际使用月份加权平均计算

**计算公式：**
```
年应纳税额 = sum(各月房产余值 x 月税率)
```

### 4.2 租金收入含税还是不含税？

**规定：** 房产税计税依据为不含增值税的租金收入

**换算公式：**
```
不含税租金 = 含税租金 / (1 + 增值税率)
```

### 4.3 免租期如何处理？

**规定：** 纳税人出租房产，约定免收租金期限的，免收租金期间由产权所有人按照房产原值缴纳房产税。

### 4.4 部分月份出租如何计算？

**情形：** 房产部分月份自用，部分月份出租

**处理方法：**
- 自用月份：从价计征
- 出租月份：从租计征

| 季应纳税额 | 季度应缴纳的房产税 |

## 七、注意事项

### 7.1 计算注意事项

1. **核实房产原值**：确保原值准确完整
2. **确认减除比例**：查询当地规定
3. **区分计税方式**：自用与出租分别计算
4. **注意优惠适用**：符合条件的及时享受优惠

### 7.2 常见错误

| 错误 | 正确做法 |
|------|----------|
| 直接用原值乘税率 | 应先计算房产余值 |
| 混淆两种计税方式 | 区分自用和出租 |
| 忽略减除比例差异 | 查询当地具体规定 |
| 租金收入未剔税 | 按不含税收入计算 |
