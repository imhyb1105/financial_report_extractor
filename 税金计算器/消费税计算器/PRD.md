# 消费税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向企业财务人员和办税人员的消费税计算器网页应用，帮助用户快速计算各类应税消费品的消费税金额。

## 2. 用户画像 (Persona)

**主要用户**：生产企业财务人员、进口商、会计师

**核心痛点**：
- 消费税税目繁多，税率差异大
- 有从价、从量、复合三种计征方式
- 部分税目在生产、进口、批发多环节征税

## 3. 功能需求

### 3.1 核心功能

#### 税目分类（Tab切换）
- 烟类
- 酒类
- 高档化妆品
- 成品油
- 小汽车
- 其他

### 3.2 主要税率表

| 税目 | 计征方式 | 税率/税额 |
|------|---------|----------|
| 甲类卷烟 | 复合计征 | 56% + 0.003元/支 |
| 乙类卷烟 | 复合计征 | 36% + 0.003元/支 |
| 白酒 | 复合计征 | 20% + 0.5元/500克 |
| 高档化妆品 | 从价计征 | 15% |
| 汽油 | 从量计征 | 1.52元/升 |
| 小汽车 | 从价计征 | 1%-40%（按排量） |

### 3.3 数据输入/输出

**输入数据**：
- 应税消费品类型（必选）
- 销售额/销售数量（必填）
- 包装物数量（如适用）

**输出数据**：
- 应纳税额
- 适用税率/税额
- 计算明细

## 5. 计算公式

### 从价计征
```
应纳税额 = 销售额 × 税率
```

### 从量计征
```
应纳税额 = 销售数量 × 单位税额
```

### 复合计征（如卷烟、白酒）
```
应纳税额 = 销售额 × 税率 + 销售数量 × 单位税额
```
