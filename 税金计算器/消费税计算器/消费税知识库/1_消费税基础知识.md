# 消费税基础知识

## 一、消费税概述

### 1. 定义
消费税是对在我国境内从事生产、委托加工和进口应税消费品的单位和个人，就其销售额或销售数量征收的一种税。

### 2. 纳税人
- 生产应税消费品的单位和个人
- 委托加工应税消费品的单位和个人
- 进口应税消费品的单位和个人
- 批发卷烟的单位
- 零售超豪华小汽车的单位和个人

### 3. 征税范围
- 烟
- 酒
- 高档化妆品
- 贵重首饰和珠宝玉石
- 鞭炮焰火
- 成品油
- 摩托车
- 小汽车
- 高尔夫球及球具
- 高档手表
- 游艇
- 木制一次性筷子
- 实木地板
- 电池
- 涂料

## 二、计征方式

### 1. 从价计征
```
应纳税额 = 销售额 × 税率
```

### 2. 从量计征
```
应纳税额 = 销售数量 × 单位税额
```

### 3. 复合计征
```
应纳税额 = 销售额 × 税率 + 销售数量 × 单位税额
```

## 三、征收环节

### 1. 生产环节
大多数应税消费品在生产环节征收

### 2. 进口环节
进口应税消费品在报关进口时征收

### 3. 批发环节
卷烟在批发环节加征一道消费税

### 4. 零售环节
超豪华小汽车在零售环节加征消费税
