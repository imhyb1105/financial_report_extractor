# 耕地占用税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向建设单位和用地企业的耕地占用税计算器网页应用，帮助用户快速计算占用耕地应缴纳的耕地占用税。

## 2. 用户画像 (Persona)

**主要用户**：建设单位财务人员、房地产开发企业、基础设施建设企业

**核心痛点**：
- 耕地占用税按占用面积和适用税额计算
- 不同地区税额标准差异大
- 需要区分耕地类型和占用情形

## 3. 功能需求

### 3.1 核心功能

#### 税额标准（按地区经济发展水平分档）

| 地区类型 | 税额范围（元/平方米） |
|---------|---------------------|
| 人均耕地≤1亩 | 10-50 |
| 人均耕地1-2亩 | 8-40 |
| 人均耕地2-3亩 | 6-30 |
| 人均耕地＞3亩 | 5-25 |

### 3.2 数据输入/输出

**输入数据**：
- 占用耕地面积（必填，平方米）
- 适用税额（必填，元/平方米）
- 占用类型（一般占用/临时占用）
- 减免情形（如有）

**输出数据**：
- 应纳耕地占用税
- 计算明细
- 减免税额（如适用）

## 5. 计算公式

### 基本公式
```
应纳税额 = 占用耕地面积 × 适用税额
```

### 临时占用
```
应纳税额 = 占用耕地面积 × 适用税额（可退还）
```
