# 耕地占用税税率

## 一、税率体系概述

耕地占用税采用**定额税率**，以纳税人实际占用的耕地面积为计税依据，按照规定的适用税额一次性征收。

### 税额确定因素

1. **人均耕地面积**：根据县级行政区域人均耕地面积确定
2. **经济发展水平**：结合当地经济发展情况
3. **地区差异**：各省可在规定范围内确定具体税额

## 二、税额标准

### 全国统一标准

根据《耕地占用税法》规定，耕地占用税的税额标准如下：

| 人均耕地面积 | 税额范围（元/平方米） |
|-------------|---------------------|
| ≤1亩/人 | 10-50 |
| 1-2亩/人 | 8-40 |
| 2-3亩/人 | 6-30 |
| ＞3亩/人 | 5-25 |

### 各地区税额确定

各省、自治区、直辖市人民政府在上述范围内，根据下列因素确定具体适用税额：

1. **人均耕地面积**
2. **经济发展水平**
3. **耕地保护需要**

### 经济发达地区

经济发达且人均耕地特别少的地区，适用税额可以提高**不超过50%**。

## 三、基本农田加征

### 加征规定

占用**基本农田**的，应当按照当地适用税额的**150%**征收。

### 基本农田的范围

基本农田是指按照一定时期人口和社会经济发展对农产品的需求，依据土地利用总体规划确定的不得占用的耕地，包括：

1. 经国务院有关主管部门或者县级以上地方人民政府批准确定的粮、棉、油生产基地内的耕地
2. 有良好的水利与水土保持设施的耕地
3. 蔬菜生产基地
4. 农业科研、教学试验田
5. 国务院规定应当划入基本农田保护区的其他耕地

### 加征计算示例

假设当地适用税额为20元/平方米：

```
一般耕地：20元/平方米
基本农田：20 × 150% = 30元/平方米
```

## 四、减免规定

### 法定免税

下列占用耕地免征耕地占用税：

#### 1. 军事设施

- 军事禁区、军事管理区
- 军事指挥工程
- 军用机场、港口、码头
- 军用仓库、营区

#### 2. 学校

- 幼儿园
- 各类学校（小学、中学、职业学校、高等院校）

#### 3. 医疗机构

- 医院
- 诊所
- 其他医疗机构

#### 4. 社会福利机构

- 养老服务机构
- 殡葬服务机构

#### 5. 交通设施

- 铁路线路
- 公路线路
- 飞机场跑道、停机坪
- 港口、航道

#### 6. 水利设施

- 水利工程占用耕地

### 法定减税

#### 农村居民建房

农村居民在户口所在地占用耕地新建自用住宅，按照当地适用税额**减半征收**。

**条件**：
1. 必须是农村居民
2. 在户口所在地
3. 新建自用住宅
4. 在规定用地标准以内

#### 困难减免

农村烈士家属、残疾军人、鳏寡孤独以及革命老根据地、少数民族聚居区和边远贫困山区生活困难的农村居民，在规定用地标准以内新建住宅缴纳耕地占用税确有困难的，可以免征或者减征。

## 五、税额速查

### 典型地区参考

以下为部分地区参考税额（具体以当地规定为准）：

| 地区 | 税额（元/平方米） |
|-----|------------------|
| 北京 | 40 |
| 上海 | 45 |
| 广东（广州） | 30 |
| 江苏（南京） | 25 |
| 浙江（杭州） | 25 |

> **注意**：以上税额仅供参考，实际适用税额请咨询当地税务机关。
