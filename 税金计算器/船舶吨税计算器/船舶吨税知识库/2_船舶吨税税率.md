# 船舶吨税税率

## 一、税率体系概述

船舶吨税采用**定额税率**，根据以下因素确定适用税率：

1. **船舶净吨位**：决定适用哪一档税率
2. **执照期限**：30天或90天（1年期按90天标准×4）
3. **税率类型**：普通税率或优惠税率

## 二、税率表

### 完整税率表

| 净吨位 | 普通税率(30天) | 普通税率(90天) | 优惠税率(30天) | 优惠税率(90天) |
|--------|---------------|---------------|---------------|---------------|
| ≤2000吨 | 2.9元/吨 | 8.7元/吨 | 1.5元/吨 | 4.5元/吨 |
| 2001-10000吨 | 3.4元/吨 | 10.2元/吨 | 1.7元/吨 | 5.1元/吨 |
| 10001-50000吨 | 3.9元/吨 | 11.7元/吨 | 2.0元/吨 | 6.0元/吨 |
| ＞50000吨 | 4.6元/吨 | 13.8元/吨 | 2.3元/吨 | 6.9元/吨 |

### 1年期执照计算

1年期执照按**90天税额标准×4**计算，不单独设定税率。

## 三、普通税率

### 适用条件

普通税率适用于**不适用优惠税率的船舶**，主要包括：

1. 未与我国签订最惠国待遇条款的国家或地区的船舶
2. 无法提供相关证明文件的船舶
3. 其他不符合优惠条件的船舶

### 税率档次

#### 30天执照期限

| 净吨位区间 | 税额（元/吨） |
|-----------|-------------|
| ≤2000 | 2.9 |
| 2001-10000 | 3.4 |
| 10001-50000 | 3.9 |
| ＞50000 | 4.6 |

#### 90天执照期限

| 净吨位区间 | 税额（元/吨） |
|-----------|-------------|
| ≤2000 | 8.7 |
| 2001-10000 | 10.2 |
| 10001-50000 | 11.7 |
| ＞50000 | 13.8 |

## 四、优惠税率

### 适用条件

优惠税率适用于**与中华人民共和国签订含有相互给予船舶税费最惠国待遇条款的国家或地区的船舶**。

### 认定标准

1. **双边协定**：该国与我国签订了相关贸易协定或航运协定
2. **最惠国条款**：协定中包含相互给予船舶税费最惠国待遇的条款
3. **国籍证明**：船舶持有该国或地区的有效国籍证书

### 优惠税率档次

#### 30天执照期限

| 净吨位区间 | 税额（元/吨） |
|-----------|-------------|
| ≤2000 | 1.5 |
| 2001-10000 | 1.7 |
| 10001-50000 | 2.0 |
| ＞50000 | 2.3 |

#### 90天执照期限

| 净吨位区间 | 税额（元/吨） |
|-----------|-------------|
| ≤2000 | 4.5 |
| 2001-10000 | 5.1 |
| 10001-50000 | 6.0 |
| ＞50000 | 6.9 |

### 优惠税率与普通税率对比

优惠税率约为普通税率的**50%-55%**，体现了互惠原则。

| 净吨位 | 30天优惠/普通比例 | 90天优惠/普通比例 |
|--------|------------------|------------------|
| ≤2000 | 51.7% | 51.7% |
| 2001-10000 | 50.0% | 50.0% |
| 10001-50000 | 51.3% | 51.3% |
| ＞50000 | 50.0% | 50.0% |

## 五、净吨位的确定

### 净吨位的概念

净吨位是指船舶有效容积折算成的吨位数，反映船舶可用于载货、载客的空间。

### 确定方法

1. **依据吨位丈量证书**：以船舶吨位丈量证书载明的净吨位为准
2. **国际标准**：按照《国际船舶吨位丈量公约》计算
3. **国内标准**：国内航行船舶按照国内相关标准丈量

### 分档标准

| 档次 | 净吨位范围 | 说明 |
|-----|-----------|-----|
| 第一档 | ≤2000 | 小型船舶 |
| 第二档 | 2001-10000 | 中型船舶 |
| 第三档 | 10001-50000 | 大型船舶 |
| 第四档 | ＞50000 | 超大型船舶 |
