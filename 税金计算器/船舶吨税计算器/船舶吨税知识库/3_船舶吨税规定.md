# 船舶吨税规定

## 一、纳税期限

### 申报期限

应税船舶负责人应当**自海关填发吨税缴款书之日起15日内**缴清税款。

### 缴款流程

```
1. 船舶进境申报
2. 海关审核计算
3. 填发吨税缴款书
4. 15日内缴纳税款
5. 领取吨税执照
```

### 逾期处理

未按规定期限缴纳税款的，从滞纳税款之日起，按日加收滞纳税款**万分之五**的滞纳金。

## 二、执照期限

### 期限类型

吨税执照期限分为三种：

| 期限 | 适用情形 | 特点 |
|-----|---------|-----|
| 30天 | 短期停靠 | 适合临时进境船舶 |
| 90天 | 常规航行 | 最常用的期限类型 |
| 1年 | 长期运营 | 适合频繁往来船舶 |

### 期限计算

1. **30天执照**：自签发之日起30日内有效
2. **90天执照**：自签发之日起90日内有效
3. **1年执照**：自签发之日起1年内有效

### 多次进出境

在吨税执照有效期内，船舶可**多次进出境**，无需重复缴纳吨税。

> **注意**：执照期满后24小时内再次进境的，需重新缴纳吨税。

## 三、免税规定

### 法定免税情形

根据《船舶吨税法》规定，以下船舶免征吨税：

#### 1. 应纳税额在人民币50元以下的船舶

- 适用于小型船舶
- 按普通税率计算应纳税额不足50元的

#### 2. 自境外取得所有权的初次进口船舶

- 以购买方式取得
- 以受赠方式取得
- 以继承方式取得
- 仅限初次进口

#### 3. 吨税执照期满后24小时内不上下客货的船舶

- 执照期满后短期停留
- 不进行客货作业
- 24小时内离开

#### 4. 非机动船舶（不包括非机动驳船）

- 无动力装置的船舶
- 非机动驳船除外（需纳税）

#### 5. 捕捞、养殖渔船

- 从事捕捞作业
- 从事养殖作业
- 需持有相关证书

#### 6. 避难、防疫隔离、修理、改造、终止运营的船舶

| 情形 | 条件 |
|-----|-----|
| 避难 | 因不可抗力或意外事故 |
| 防疫隔离 | 因疫情防控需要 |
| 修理/改造 | 在船厂进行维修改造 |
| 终止运营 | 不再从事商业运营 |

#### 7. 其他法定免税情形

- 军事船舶
- 外交船舶
- 法律法规规定的其他免税船舶

### 免税申请

1. 提交免税申请及相关证明材料
2. 海关审核确认
3. 办理免税手续

## 四、征收管理

### 海关职责

1. **税款征收**：依法征收吨税
2. **执照管理**：签发和管理吨税执照
3. **监督检查**：检查船舶吨税缴纳情况
4. **违规处理**：对违规行为依法处理

### 申报材料

申报吨税时需提交以下材料：

1. 《船舶吨税申报单》
2. 船舶国籍证书
3. 船舶吨位证书
4. 上一航次的吨税执照（如有）
5. 其他相关证明文件

### 电子申报

目前各地海关普遍支持电子申报：

1. 通过"单一窗口"平台申报
2. 在线缴纳税款
3. 电子执照下载

## 五、违规处理

### 未按规定缴纳吨税

1. **不予放行**：海关有权不予放行未缴纳吨税的船舶
2. **追缴税款**：责令补缴应纳税款
3. **滞纳金**：按日加收万分之五滞纳金
4. **行政处罚**：情节严重的可处以罚款

### 虚假申报

1. 补缴税款
2. 加收滞纳金
3. 处以欠缴税款50%以上5倍以下罚款
4. 构成犯罪的，依法追究刑事责任

### 执照管理违规

1. 伪造、变造吨税执照的，依法追究刑事责任
2. 出借、转让吨税执照的，处以罚款

## 六、特殊情况处理

### 执照丢失

1. 及时向签发海关报告
2. 申请补发执照
3. 提供相关证明材料

### 信息变更

船舶信息发生变更时：

1. 及时向海关申报变更
2. 更新相关证明文件
3. 必要时重新办理执照
