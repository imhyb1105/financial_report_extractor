# 资源税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向矿山企业和资源开采单位的资源税计算器网页应用，帮助用户快速计算各类资源开采应缴纳的资源税。

## 2. 用户画像 (Persona)

**主要用户**：矿山企业财务人员、资源开采单位会计、税务代理人员

**核心痛点**：
- 资源税税目众多，不同资源税率差异大
- 从价定率和从量定额两种计征方式
- 各省可在幅度内自行确定具体税额

## 3. 功能需求

### 3.1 核心功能

#### 主要税目分类
- 能源矿产（原油、天然气、煤炭）
- 金属矿产（铁矿、金矿、铜矿等）
- 非金属矿产（石灰石、粘土、砂石等）
- 盐类

### 3.2 主要税率表

| 税目 | 计征方式 | 税额幅度 |
|------|---------|---------|
| 原油 | 从量定额 | 6-30元/吨 |
| 天然气 | 从量定额 | 2-15元/千立方米 |
| 煤炭 | 从量定额 | 2-20元/吨 |
| 铁矿 | 从量定额 | 5-25元/吨 |
| 稀土 | 从量定额 | 7.5-30元/吨 |

### 3.3 数据输入/输出

**输入数据**：
- 资源类型（必选）
- 开采/销售数量（必填）
- 适用税额（必填）
- 计税单位

**输出数据**：
- 应纳税额
- 适用税额
- 计算明细

## 5. 计算公式

### 从量定额计征
```
应纳税额 = 销售数量 × 单位税额
```

### 从价定率计征
```
应纳税额 = 销售额 × 税率
```

## 6. 政策依据

- 《中华人民共和国资源税法》
