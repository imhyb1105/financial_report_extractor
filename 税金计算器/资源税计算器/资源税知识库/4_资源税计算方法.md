# 资源税计算方法

**版本：2026.03-001**

---

## 一、基本计算公式

### 1.1 从价计征公式

```
应纳税额 = 销售额 × 适用税率
```

**参数说明**：

| 参数 | 说明 |
|------|------|
| 销售额 | 应税产品的全部价款和价外费用（不含增值税） |
| 适用税率 | 各省确定的具体税率 |

### 1.2 从量计征公式

```
应纳税额 = 销售数量 × 适用税额标准
```

**参数说明**：

| 参数 | 说明 |
|------|------|
| 销售数量 | 应税产品的实际销售数量（吨、立方米等） |
| 适用税额标准 | 各省确定的具体税额（元/吨、元/立方米） |

---

## 二、从价计征计算详解

### 2.1 销售额的确定

**基本公式**：
```
销售额 = 销售价款 + 价外费用
```

**含税销售额的换算**：
```
不含税销售额 = 含税销售额 ÷ (1 + 增值税税率或征收率)
```

### 2.2 计算步骤

```
第一步：确定应税产品类别
    ↓
第二步：确定计征方式（从价/从量）
    ↓
第三步：确定适用税率/税额
    ↓
第四步：计算销售额/销售数量
    ↓
第五步：计算应纳税额
    ↓
第六步：确认是否有优惠减免
    ↓
第七步：计算最终应纳税额
```

## 三、从价计征计算实例

### 实例1：原油销售

**基本情况**：
- 某油田销售原油10000吨
- 不含税销售价格：4000元/吨
- 适用税率：6%

**计算过程**：
```
销售额 = 10000吨 × 4000元/吨 = 40,000,000元

应纳税额 = 40,000,000元 × 6% = 2,400,000元
```

**结果**：应纳资源税2,400,000元

## 五、税收优惠计算实例

### 实例13：衰竭期矿山优惠

**基本情况**：
- 某衰竭期铁矿销售铁矿石10000吨
- 不含税销售价格：700元/吨
- 适用税率：4%
- 减征幅度：30%

**计算过程**：
```
销售额 = 10,000吨 × 700元/吨 = 7,000,000元

应纳税额 = 7,000,000元 × 4% = 280,000元

优惠后应纳税额 = 280,000元 × (1 - 30%) = 196,000元
```

**结果**：应纳资源税196,000元（减征84,000元）

---

### 实例14：充填开采优惠

**基本情况**：
- 某煤矿采用充填开采方式销售原煤20000吨
- 不含税销售价格：480元/吨
- 适用税率：5%
- 减征幅度：50%

**计算过程**：
```
销售额 = 20,000吨 × 480元/吨 = 9,600,000元

应纳税额 = 9,600,000元 × 5% = 480,000元

优惠后应纳税额 = 480,000元 × (1 - 50%) = 240,000元
```

**结果**：应纳资源税240,000元（减征240,000元）

## 六、混合销售计算实例

### 实例16：原煤和洗选煤分别销售

**基本情况**：
- 某煤矿销售原煤30000吨，不含税价格450元/吨
- 同时销售洗选煤20000吨，不含税价格600元/吨
- 原煤税率：5%，洗选煤折算率：70%

**计算过程**：
```
原煤：
销售额 = 30,000吨 × 450元/吨 = 13,500,000元
应纳税额 = 13,500,000元 × 5% = 675,000元

洗选煤：
销售额 = 20,000吨 × 600元/吨 = 12,000,000元
计税销售额 = 12,000,000元 × 70% = 8,400,000元
应纳税额 = 8,400,000元 × 5% = 420,000元

合计应纳税额 = 675,000元 + 420,000元 = 1,095,000元
```

**结果**：应纳资源税1,095,000元

## 七、自产自用计算实例

### 实例17：自用原煤

**基本情况**：
- 某煤矿开采原煤10000吨
- 销售6000吨，不含税价格500元/吨
- 自用于发电4000吨
- 适用税率：5%

**计算过程**：
```
销售部分：
销售额 = 6,000吨 × 500元/吨 = 3,000,000元
应纳税额 = 3,000,000元 × 5% = 150,000元

自用部分（视同销售）：
销售额 = 4,000吨 × 500元/吨 = 2,000,000元
应纳税额 = 2,000,000元 × 5% = 100,000元

合计应纳税额 = 150,000元 + 100,000元 = 250,000元
```

**结果**：应纳资源税250,000元

## 八、计算要点总结

### 8.1 从价计征要点

| 要点 | 说明 |
|------|------|
| 销售额确定 | 不含增值税的全部价款和价外费用 |
| 税率选择 | 按开采地所在省份规定的税率 |
| 洗选煤处理 | 需按折算率折算销售额 |
| 自用处理 | 视同销售，按同类产品价格计税 |

### 8.2 从量计征要点

| 要点 | 说明 |
|------|------|
| 数量确定 | 实际销售数量或移送使用数量 |
| 税额选择 | 按开采地所在省份规定的税额标准 |
| 单位换算 | 注意计量单位（吨、立方米等） |

### 8.3 优惠减免要点

| 要点 | 说明 |
|------|------|
| 条件判断 | 严格按政策规定判断是否符合优惠条件 |
| 减征计算 | 应纳税额 × (1 - 减征比例) |
| 备案要求 | 部分优惠需提前向税务机关备案 |

## 九、常见计算错误

### 9.1 销售额错误

| 错误类型 | 正确做法 |
|----------|----------|
| 未剔除增值税 | 销售额应不含增值税 |
| 漏计价外费用 | 价外费用应计入销售额 |
| 折扣处理不当 | 同一笔交易的折扣可扣除 |

### 9.2 税率适用错误

| 错误类型 | 正确做法 |
|----------|----------|
| 使用国家幅度税率 | 应使用省级确定的具体税率 |
| 混合销售未分别核算 | 分别核算，分别适用税率 |
| 未分别核算从高 | 未分别核算时从高适用税率 |

### 9.3 优惠计算错误

| 错误类型 | 正确做法 |
|----------|----------|
| 重复享受优惠 | 同一项目不得重复享受优惠 |
| 未按规定备案 | 按规定程序申请和备案 |
| 超出优惠范围 | 严格按优惠条件适用 |
