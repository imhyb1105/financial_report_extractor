# 车船税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向车主和财务人员的车船税计算器网页应用，帮助用户快速计算乘用车、商用车等的车船税金额。

## 2. 用户画像 (Persona)

**主要用户**：车主、企业车队管理员、财务人员

**核心痛点**：
- 车船税按排量分档，税率档次多
- 不同类型车辆计算方式不同
- 新能源汽车优惠政策变化

## 3. 功能需求

### 3.1 核心功能

#### 乘用车税率表（按排量）

| 排量区间 | 基准税额（年） |
|---------|--------------|
| 1.0L（含）以下 | 60-180元 |
| 1.0L-1.6L（含） | 300-360元 |
| 1.6L-2.0L（含） | 360-660元 |
| 2.0L-2.5L（含） | 660-1200元 |
| 2.5L-3.0L（含） | 1200-2400元 |
| 3.0L-4.0L（含） | 2400-3600元 |
| 4.0L以上 | 3600-5400元 |

### 3.2 数据输入/输出

**输入数据**：
- 车辆类型（必选）
- 排量/载重（必填）
- 车辆数量（必填）

**输出数据**：
- 年应纳税额
- 适用税率说明

## 5. 计算公式

### 乘用车计算公式
```
年应纳税额 = 计税单位 × 适用年基准税额
```

### 货车计算公式
```
年应纳税额 = 整备质量（吨）× 适用年基准税额（元/吨）
```

## 6. 政策依据

- 《中华人民共和国车船税法》
