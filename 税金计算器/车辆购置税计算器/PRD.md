# 车辆购置税计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向购车者的车辆购置税计算器网页应用，帮助用户快速计算燃油汽车和新能源汽车的购置税金额。

## 2. 用户画像 (Persona)

**主要用户**：购车者、汽车销售人员、财务人员

**核心痛点**：
- 新能源汽车优惠政策有时间限制和金额上限
- 计税价格与发票价格不同
- 2026年起新能源汽车优惠"退坡"

## 3. 功能需求

### 3.1 核心功能

#### 车辆类型选择
- 燃油汽车（10%）
- 新能源汽车（2024-2025免征）
- 新能源汽车（2026-2027减半）

### 3.2 政策时间表

| 时间段 | 政策内容 | 实际税率 | 减免上限 |
|--------|----------|----------|----------|
| 2024.1.1-2025.12.31 | 全额免征 | 0% | 每辆乘用车免税额不超过3万元 |
| 2026.1.1-2027.12.31 | 减半征收 | 5% | 每辆乘用车减税额不超过1.5万元 |

### 3.3 数据输入/输出

**输入数据**：
- 车辆价格（必填）
- 车辆类型（必选）
- 购买日期（必填）

**输出数据**：
- 购置税金额
- 优惠金额
- 计算明细

## 5. 计算公式

### 基本计算公式
```
计税价格 = 含税价格 ÷ (1 + 增值税率13%)
应纳税额 = 计税价格 × 税率
```

### 新能源汽车（2026-2027）
```
实际减税额 = Min(计税价格 × 5%, 15000)
实际应纳税额 = 计税价格 × 5% - 实际减税额
```
