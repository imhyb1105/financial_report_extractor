# 车辆购置税税率

**版本：2026.03-001**
**更新日期：2026年3月16日**

---

## 一、基本税率

### 1.1 标准税率

根据《中华人民共和国车辆购置税法》规定，车辆购置税的税率为**10%**。

### 1.2 税率适用范围

| 车辆类型 | 适用税率 |
|----------|----------|
| 燃油汽车 | 10% |
| 柴油汽车 | 10% |
| 混合动力汽车（非插电式） | 10% |
| 普通摩托车（排气量>150ml） | 10% |
| 汽车挂车 | 10% |
| 有轨电车 | 10% |

## 二、新能源汽车税率政策

### 2.1 政策概述

为支持新能源汽车产业发展，促进汽车消费，国家对新能源汽车实施车辆购置税减免政策。

### 2.2 新能源汽车减免税率表

| 时间段 | 政策内容 | 适用税率 |
|--------|----------|----------|
| 2024年1月1日 - 2025年12月31日 | 免征车辆购置税 | 0% |
| 2026年1月1日 - 2027年12月31日 | 减半征收车辆购置税 | 5% |

### 2.3 新能源汽车认定标准

符合以下条件的新能源汽车可享受减免政策：

#### 纯电动汽车

| 项目 | 要求 |
|------|------|
| 纯电续航里程 | 达到规定标准（不同车型要求不同） |
| 动力电池能量密度 | 符合国家标准 |
| 车辆类型 | 纳入《免征车辆购置税的新能源汽车车型目录》 |

#### 插电式混合动力汽车

| 项目 | 要求 |
|------|------|
| 纯电续航里程 | ≥50公里（NEDC工况）或≥43公里（WLTC工况） |
| 专用号牌 | 需悬挂新能源汽车专用号牌 |
| 车辆类型 | 纳入《免征车辆购置税的新能源汽车车型目录》 |

#### 燃料电池汽车

| 项目 | 要求 |
|------|------|
| 车辆类型 | 纳入《免征车辆购置税的新能源汽车车型目录》 |
| 技术标准 | 符合国家标准 |

### 2.4 减免税限额规定

每辆新能源乘用车减免税额实行限额管理：

| 时间段 | 减免税限额 |
|--------|------------|
| 2024年1月1日 - 2025年12月31日 | 每辆不超过30,000元 |
| 2026年1月1日 - 2027年12月31日 | 每辆不超过15,000元 |

**计算说明**：
- 2024-2025年：若按10%计算的税额超过30,000元，则最高免税30,000元，超出部分需缴纳
- 2026-2027年：若按10%计算的税额超过30,000元，则减税额最高为15,000元（即实际税额 = 原税额 - 15,000元）

## 四、增值税税率说明

### 4.1 汽车销售增值税

汽车销售适用的增值税税率为**13%**。

### 4.2 计税价格换算

```
计税价格 = 含增值税价格 / (1 + 增值税率)
         = 含增值税价格 / 1.13
```

### 4.3 换算示例

| 含税价格 | 计税价格 | 计算过程 |
|----------|----------|----------|
| 113,000元 | 100,000元 | 113,000 / 1.13 = 100,000 |
| 226,000元 | 200,000元 | 226,000 / 1.13 = 200,000 |
