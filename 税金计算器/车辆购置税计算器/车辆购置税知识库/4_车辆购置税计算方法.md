**版本：2026.03-001**
**更新日期：2026年3月16日**

## 一、基本计算公式

### 1.1 标准计算公式

```
应纳税额 = 计税价格 × 税率
```

### 1.2 计税价格确定

#### 购买自用车辆

```
计税价格 = 含增值税的全部价款 / (1 + 增值税率)
         = 含增值税的全部价款 / 1.13
```

**说明**：
- 增值税率：汽车销售适用13%的增值税率
- 全部价款：实际支付给销售者的全部金额（不包括增值税税款）

## 二、燃油车计算示例

### 2.1 计算步骤

1. 确定车辆含税价格（发票金额）
2. 计算计税价格：含税价格 / 1.13
3. 计算应纳税额：计税价格 × 10%

### 2.2 计算实例

#### 示例一：10万元燃油车

| 项目 | 金额 |
|------|------|
| 车辆含税价格 | 100,000元 |
| 计税价格 | 100,000 / 1.13 = 88,495.57元 |
| 应纳税额 | 88,495.57 × 10% = 8,849.56元 |

#### 示例二：20万元燃油车

| 项目 | 金额 |
|------|------|
| 车辆含税价格 | 200,000元 |
| 计税价格 | 200,000 / 1.13 = 176,991.15元 |
| 应纳税额 | 176,991.15 × 10% = 17,699.12元 |

#### 示例三：50万元燃油车

| 项目 | 金额 |
|------|------|
| 车辆含税价格 | 500,000元 |
| 计税价格 | 500,000 / 1.13 = 442,477.88元 |
| 应纳税额 | 442,477.88 × 10% = 44,247.79元 |

## 三、新能源汽车计算示例（2026年）

### 3.1 2026年政策说明

- 税率：减半征收，即5%
- 减税限额：每辆不超过15,000元

### 3.2 计算步骤

1. 计算计税价格：含税价格 / 1.13
2. 计算原税额：计税价格 × 10%
3. 计算减税额：min(原税额 / 2, 15,000)
4. 计算应纳税额：原税额 - 减税额

### 3.3 计算实例

#### 示例一：15万元新能源车（未超限额）

| 项目 | 金额 |
|------|------|
| 车辆含税价格 | 150,000元 |
| 计税价格 | 150,000 / 1.13 = 132,743.36元 |
| 原税额 | 132,743.36 × 10% = 13,274.34元 |
| 减税额 | min(13,274.34 / 2, 15,000) = 6,637.17元 |
| 应纳税额 | 13,274.34 - 6,637.17 = 6,637.17元 |

#### 示例二：30万元新能源车（刚好达限额）

| 项目 | 金额 |
|------|------|
| 车辆含税价格 | 300,000元 |
| 计税价格 | 300,000 / 1.13 = 265,486.73元 |
| 原税额 | 265,486.73 × 10% = 26,548.67元 |
| 减税额 | min(26,548.67 / 2, 15,000) = 13,274.34元 |
| 应纳税额 | 26,548.67 - 13,274.34 = 13,274.34元 |

#### 示例三：50万元新能源车（超限额）

| 项目 | 金额 |
|------|------|
| 车辆含税价格 | 500,000元 |
| 计税价格 | 500,000 / 1.13 = 442,477.88元 |
| 原税额 | 442,477.88 × 10% = 44,247.79元 |
| 减税额 | min(44,247.79 / 2, 15,000) = 15,000元 |
| 应纳税额 | 44,247.79 - 15,000 = 29,247.79元 |

## 四、进口车辆计算示例

### 4.1 计算公式

```
计税价格 = 关税完税价格 + 关税 + 消费税
应纳税额 = 计税价格 × 10%
```

### 4.2 计算实例

#### 示例：进口小汽车

| 项目 | 金额 |
|------|------|
| 关税完税价格（CIF） | 300,000元 |
| 关税（25%） | 75,000元 |
| 消费税（排量2.5L，9%） | 30,000元 |
| 计税价格 | 300,000 + 75,000 + 30,000 = 405,000元 |
| 应纳税额 | 405,000 × 10% = 40,500元 |

## 五、免税条件消失补税计算

### 5.1 计算公式

```
补税额 = 初次申报计税价格 × (1 - 使用年限 × 10%) × 10%

说明：
- 使用年限按年计算，不足一年的按一年计算
- 最高不超过10年，即使用10年以上的不需要补税
```

### 5.2 计算实例

#### 示例一：免税车辆使用3年后转让

| 项目 | 金额 |
|------|------|
| 原计税价格 | 200,000元 |
| 使用年限 | 3年 |
| 使用系数 | 1 - 3 × 10% = 0.7 |
| 补税额 | 200,000 × 0.7 × 10% = 14,000元 |

#### 示例二：免税车辆使用7年后转让

| 项目 | 金额 |
|------|------|
| 原计税价格 | 200,000元 |
| 使用年限 | 7年 |
| 使用系数 | 1 - 7 × 10% = 0.3 |
| 补税额 | 200,000 × 0.3 × 10% = 6,000元 |

| 信息项 | 说明 |

| 应纳税额 | 最终需要缴纳的税额 |

## 八、常见问题

### Q1：为什么计税价格比发票价格低？

因为发票价格包含增值税，而车辆购置税的计税价格不含增值税，需要先剔除13%的增值税。

### Q2：新能源汽车的减税额如何确定？

2026-2027年期间，减税额为原税额的一半，但最高不超过15,000元。

### Q3：最低计税价格是什么？

税务机关对部分车型核定最低计税价格，当实际购车价格（剔除增值税后）低于最低计税价格时，按最低计税价格计算税额。

### Q4：二手车需要缴纳车辆购置税吗？

不需要。车辆购置税在车辆首次购置时一次性征收，二手车交易不再征收。
