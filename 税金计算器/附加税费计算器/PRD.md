# 附加税费计算器产品需求文档 (PRD)

## 1. 核心目标 (Mission)

开发一款面向企业财务人员的附加税费计算器网页应用，帮助用户快速计算与增值税、消费税相关的城市维护建设税及教育费附加。

## 2. 用户画像 (Persona)

**主要用户**：企业财务人员、会计师、税务代理人员

**核心痛点**：
- 附加税费以增值税、消费税为计税依据
- 不同地区城市维护建设税率不同
- 需同时计算多项附加税费

## 3. 功能需求

### 3.1 核心功能

#### 附加税费构成

| 税费名称 | 计税依据 | 税率 |
|---------|---------|------|
| 城市维护建设税 | 增值税+消费税 | 1%/5%/7% |
| 教育费附加 | 增值税+消费税 | 3% |
| 地方教育附加 | 增值税+消费税 | 2% |

### 3.2 城市维护建设税率

| 地区类型 | 税率 |
|---------|------|
| 市区 | 7% |
| 县城、镇 | 5% |
| 其他地区 | 1% |

### 3.3 数据输入/输出

**输入数据**：
- 地区类型（必选）
- 增值税税额（必填）
- 消费税税额（选填）

**输出数据**：
- 城市维护建设税
- 教育费附加
- 地方教育附加
- 附加税费合计
- 计算明细

## 5. 计算公式

### 计税依据
```
计税依据 = 实际缴纳的增值税 + 实际缴纳的消费税
```

### 城市维护建设税
```
城市维护建设税 = 计税依据 × 适用税率(7%/5%/1%)
```

### 教育费附加
```
教育费附加 = 计税依据 × 3%
```

### 地方教育附加
```
地方教育附加 = 计税依据 × 2%
```

### 附加税费合计
```
附加税费合计 = 城市维护建设税 + 教育费附加 + 地方教育附加
```

## 6. 政策依据

- 《中华人民共和国城市维护建设税法》
- 《征收教育费附加的暂行规定》
- 各省、自治区、直辖市地方教育附加征收规定
