# 附加税费计算方法

**版本：2026.03-001**

---

## 一、基本公式

### 1.1 城市维护建设税

```
城建税 = (实际缴纳的增值税 + 实际缴纳的消费税) × 城建税率
```

| 纳税人所在地 | 城建税率 |
|--------------|----------|
| 市区 | 7% |
| 县城、建制镇 | 5% |
| 其他地区 | 1% |

### 1.2 教育费附加

```
教育费附加 = (实际缴纳的增值税 + 实际缴纳的消费税) × 3%
```

### 1.3 地方教育附加

```
地方教育附加 = (实际缴纳的增值税 + 实际缴纳的消费税) × 2%
```

### 1.4 合计附加税费

```
合计附加税费 = 城建税 + 教育费附加 + 地方教育附加
```

或简化为：

```
合计附加税费 = (增值税 + 消费税) × (城建税率 + 3% + 2%)
```

## 二、计算示例

### 示例1：市区企业（仅缴纳增值税）

**已知条件：**
- 实际缴纳增值税：10,000.00 元
- 实际缴纳消费税：0 元
- 纳税地点：市区（城建税率7%）

**计算过程：**

| 税费名称 | 计算式 | 金额（元） |
|----------|--------|------------|
| 城建税 | 10,000 × 7% | 700.00 |
| 教育费附加 | 10,000 × 3% | 300.00 |
| 地方教育附加 | 10,000 × 2% | 200.00 |
| **合计** | 700 + 300 + 200 | **1,200.00** |

**验证：** 10,000 × 12% = 1,200.00 元

---

### 示例2：县城企业（缴纳增值税和消费税）

**已知条件：**
- 实际缴纳增值税：50,000.00 元
- 实际缴纳消费税：20,000.00 元
- 纳税地点：县城（城建税率5%）

**计算过程：**

计税依据 = 50,000 + 20,000 = 70,000 元

| 税费名称 | 计算式 | 金额（元） |
|----------|--------|------------|
| 城建税 | 70,000 × 5% | 3,500.00 |
| 教育费附加 | 70,000 × 3% | 2,100.00 |
| 地方教育附加 | 70,000 × 2% | 1,400.00 |
| **合计** | 3,500 + 2,100 + 1,400 | **7,000.00** |

**验证：** 70,000 × 10% = 7,000.00 元

---

### 示例3：其他地区企业（仅缴纳增值税）

**已知条件：**
- 实际缴纳增值税：8,000.00 元
- 实际缴纳消费税：0 元
- 纳税地点：乡镇（城建税率1%）

**计算过程：**

| 税费名称 | 计算式 | 金额（元） |
|----------|--------|------------|
| 城建税 | 8,000 × 1% | 80.00 |
| 教育费附加 | 8,000 × 3% | 240.00 |
| 地方教育附加 | 8,000 × 2% | 160.00 |
| **合计** | 80 + 240 + 160 | **480.00** |

**验证：** 8,000 × 6% = 480.00 元

## 四、特殊情形计算

### 4.1 享受减税情形

**示例：** 某企业享受增值税减半征收优惠

- 原应纳增值税：10,000 元
- 实际缴纳增值税：5,000 元（减半）
- 纳税地点：市区

附加税费按**实际缴纳**金额计算：

| 税费名称 | 计算式 | 金额（元） |
|----------|--------|------------|
| 城建税 | 5,000 × 7% | 350.00 |
| 教育费附加 | 5,000 × 3% | 150.00 |
| 地方教育附加 | 5,000 × 2% | 100.00 |
| **合计** | - | **600.00** |

### 4.2 补缴税款情形

**示例：** 某企业当期正常缴纳增值税10,000元，另补缴以前期间增值税2,000元

- 计税依据 = 10,000 + 2,000 = 12,000 元
- 纳税地点：县城

| 税费名称 | 计算式 | 金额（元） |
|----------|--------|------------|
| 城建税 | 12,000 × 5% | 600.00 |
| 教育费附加 | 12,000 × 3% | 360.00 |
| 地方教育附加 | 12,000 × 2% | 240.00 |
| **合计** | - | **1,200.00** |

## 五、计算规范

### 5.1 金额精度

- 计算结果**保留两位小数**
- 采用**四舍五入**法

### 5.2 结果展示

完整的附加税费计算结果应包含：

1. 计税依据（增值税、消费税金额）
2. 适用税率（城建税按地区）
3. 各项附加税费金额
4. 合计附加税费金额

---

## 六、注意事项

1. **实际缴纳原则**：以实际缴纳的增值税、消费税为计税依据，不是应纳税额
2. **进口不征**：进口环节缴纳的增值税、消费税不征收附加税费
3. **减免联动**：增值税、消费税减免时，附加税费同步减免
4. **税率确认**：城建税税率按纳税地点确定，需确认正确的地区档次
