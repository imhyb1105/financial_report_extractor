# 耕地占用税规定

## 一、纳税期限

### 申报期限

纳税人应当**自收到农用地转用批准文件之日起30日内**申报缴纳耕地占用税。

### 不同情形的期限

| 情形 | 纳税期限 |
|-----|---------|
| 经批准占用 | 收到批准文件之日起30日内 |
| 未经批准占用 | 实际占用之日起30日内 |
| 临时占用 | 批准占用之日起30日内 |

### 逾期处理

未按规定期限缴纳税款的，从滞纳税款之日起，按日加收滞纳税款**万分之五**的滞纳金。

## 二、临时占用

### 临时占用的定义

临时占用耕地是指因建设项目施工、地质勘查等需要，在经批准的期限内临时使用耕地的行为。

### 临时占用规定

1. **需缴纳耕地占用税**：临时占用耕地也应当缴纳耕地占用税
2. **可退还税款**：在批准临时占用耕地期限内恢复耕地原状的，全额退还已缴纳的耕地占用税
3. **逾期不退**：超过批准期限未恢复原状的，已缴纳的税款不予退还

### 退还条件

```
条件：在批准期限内恢复耕地原状
退还金额：全额退还已缴纳的耕地占用税
```

## 三、免税规定

### 完整免税清单

根据《耕地占用税法》规定，以下情形免征耕地占用税：

#### 1. 军事设施占用耕地

- 军事禁区、军事管理区的办公、营房、训练设施
- 军用机场、港口、码头
- 军用仓库、输油管道
- 军用通信、侦察、导航设施

#### 2. 学校、幼儿园占用耕地

- 教学楼、实验楼
- 图书馆、体育馆
- 宿舍、食堂
- 操场、绿化用地

#### 3. 社会福利机构占用耕地

- 养老院、敬老院
- 儿童福利院
- 殡仪馆、公墓

#### 4. 医疗机构占用耕地

- 综合医院、专科医院
- 社区卫生服务中心
- 疾病预防控制中心

#### 5. 交通设施占用耕地

| 类型 | 免税范围 |
|-----|---------|
| 铁路 | 线路、车站、调度中心 |
| 公路 | 高速公路、国道、省道 |
| 机场 | 跑道、停机坪、航站楼 |
| 港口 | 码头、泊位、堆场 |

#### 6. 水利工程占用耕地

- 水库
- 水闸
- 灌溉渠道
- 防洪设施

### 免税申请

1. 提交免税申请及相关证明材料
2. 税务机关审核确认
3. 办理免税手续

## 四、减税规定

### 农村居民建房减税

**减税对象**：农村居民在户口所在地占用耕地新建自用住宅

**减税幅度**：按照当地适用税额**减半征收**

**条件限制**：
1. 必须是农村居民身份
2. 必须在户口所在地建房
3. 必须是自用住宅
4. 必须在规定用地标准以内

### 困难减免

**适用对象**：
- 农村烈士家属
- 残疾军人
- 鳏寡孤独人员
- 革命老根据地的困难农村居民
- 少数民族聚居区的困难农村居民
- 边远贫困山区的困难农村居民

**减免条件**：在规定用地标准以内新建住宅缴纳耕地占用税确有困难

**减免方式**：可以免征或者减征

## 五、征收管理

### 税务机关职责

1. **税款征收**：依法征收耕地占用税
2. **信息共享**：与自然资源部门建立信息共享机制
3. **监督检查**：检查耕地占用税缴纳情况
4. **违规处理**：对违规行为依法处理

### 申报材料

申报耕地占用税时需提交以下材料：

1. 《耕地占用税纳税申报表》
2. 农用地转用批准文件（如有）
3. 土地权属证明
4. 身份证明文件
5. 其他相关证明文件

### 信息共享机制

税务机关与自然资源部门应当建立耕地占用税涉税信息共享机制：

1. **信息传递**：自然资源部门定期传递农用地转用信息
2. **数据比对**：税务机关定期比对纳税情况
3. **协同监管**：联合开展耕地保护工作

## 六、违规处理

### 未按规定缴纳

1. **追缴税款**：责令补缴应纳税款
2. **滞纳金**：按日加收万分之五滞纳金
3. **行政处罚**：情节严重的可处以罚款

### 虚假申报

1. 补缴税款
2. 加收滞纳金
3. 处以欠缴税款50%以上5倍以下罚款
