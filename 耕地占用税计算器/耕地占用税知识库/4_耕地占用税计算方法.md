# 耕地占用税计算方法

## 一、计算公式

### 基本公式

```
应纳税额 = 占用耕地面积 × 适用税额
```

### 基本农田加征公式

```
应纳税额 = 占用耕地面积 × 适用税额 × 150%
```

### 农村居民建房减税公式

```
应纳税额 = 占用耕地面积 × 适用税额 × 50%
```

## 二、计算步骤

### 第一步：确定占用耕地面积

以实际占用的耕地面积为准，单位为平方米。

### 第二步：确定适用税额

根据当地税务机关公布的标准确定适用税额。

### 第三步：判断是否涉及加征或减免

- 占用基本农田：加征50%
- 农村居民建房：减半征收
- 其他减免情形：按规定执行

### 第四步：计算应纳税额

代入相应公式计算。

## 三、计算示例

### 示例1：一般占用

**条件：**
- 占用耕地面积：10000平方米
- 当地适用税额：20元/平方米
- 占用类型：永久占用

**计算：**
```
应纳税额 = 10000 × 20 = 200000元
```

### 示例2：占用基本农田

**条件：**
- 占用耕地面积：5000平方米
- 当地适用税额：25元/平方米
- 占用类型：基本农田

**计算：**
```
应纳税额 = 5000 × 25 × 150% = 187500元
```

### 示例3：农村居民建房

**条件：**
- 占用耕地面积：200平方米
- 当地适用税额：15元/平方米
- 占用类型：农村居民新建自用住宅

**计算：**
```
应纳税额 = 200 × 15 × 50% = 1500元
```

### 示例4：临时占用（可退还）

**条件：**
- 占用耕地面积：3000平方米
- 当地适用税额：20元/平方米
- 占用类型：临时占用（期限2年）

**计算：**
```
应纳税额 = 3000 × 20 = 60000元

注：在批准期限内恢复耕地原状的，全额退还
```

### 示例5：大型项目

**条件：**
- 占用耕地面积：100000平方米（约150亩）
- 当地适用税额：30元/平方米
- 占用类型：工业用地

**计算：**
```
应纳税额 = 100000 × 30 = 3000000元
```

## 四、面积换算

### 亩与平方米换算

```
1亩 ≈ 666.67平方米
1平方米 ≈ 0.0015亩
```

### 换算示例

| 亩数 | 平方米 |
|-----|--------|
| 1亩 | 666.67㎡ |
| 5亩 | 3333.33㎡ |
| 10亩 | 6666.67㎡ |
| 100亩 | 66666.67㎡ |

## 五、特殊情形处理

### 1. 部分免税项目

**条件**：免税项目中的经营性部分

**处理**：
- 免税部分：按规定免税
- 经营性部分：按规定征税

**示例**：
学校用地中，教学楼免税，但配套的商业设施需要征税。

### 2. 混合用途

**条件**：同一项目占用耕地用于不同用途

**处理**：
- 按各用途分别计算
- 适用不同税率的，分别适用

### 3. 分期占用

**条件**：分期占用耕地

**处理**：
- 每次占用时单独计算
- 分别申报缴纳

### 4. 退还情形

**条件**：临时占用在期限内恢复原状

**处理**：
- 全额退还已缴纳税款
- 需提供恢复原状的证明材料

## 六、常见问题解答

### Q1：如何确定当地适用税额？

**A**：当地适用税额由各省、自治区、直辖市人民政府在法定范围内确定，请咨询当地税务机关或查询当地政府公告。

### Q2：占用基本农田如何计税？

**A**：占用基本农田的，按照当地适用税额的150%征收。例如，当地适用税额为20元/平方米，占用基本农田则按30元/平方米计算。

### Q3：农村居民建房有什么优惠？

**A**：农村居民在户口所在地占用耕地新建自用住宅，按照当地适用税额减半征收。

### Q4：临时占用耕地可以退税吗？

**A**：可以。临时占用耕地经批准的，在批准临时占用耕地期限内恢复耕地原状的，全额退还已缴纳的耕地占用税。

### Q5：超过期限缴税有什么后果？

**A**：未按规定期限缴纳税款的，从滞纳税款之日起，按日加收滞纳税款万分之五的滞纳金。

### Q6：学校用地是否免税？

**A**：是的。幼儿园、学校占用耕地免征耕地占用税。

### Q7：如何计算亩数和平方米的换算？

**A**：1亩约等于666.67平方米。例如，1000平方米约等于1.5亩。

### Q8：未经批准占用耕地如何处理？

**A**：未经批准占用耕地的，应当自实际占用之日起30日内申报缴纳耕地占用税，并可能面临行政处罚。
