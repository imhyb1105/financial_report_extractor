# 船舶吨税基础知识

## 一、船舶吨税的定义

船舶吨税（简称"吨税"）是海关对自中国境外港口进入中国境内港口的船舶征收的一种税。吨税起源于航海时代的"灯塔费"，是国际航运中普遍征收的税费。

### 基本特征

1. **征收目的**：用于海上航标的维护和建设，保障船舶航行安全
2. **征税对象**：自境外港口进入境内港口的船舶
3. **计税依据**：船舶净吨位
4. **征收机关**：海关

## 二、纳税义务人

### 纳税义务人的确定

**吨税的纳税义务人**为应税船舶的负责人，包括：

1. **船舶所有人**：拥有船舶所有权的单位或个人
2. **船舶经营人**：实际经营船舶的单位或个人
3. **船舶管理人**：受委托管理船舶的单位或个人
4. **船舶承租人**：光船租赁或定期租船的承租人

### 注意事项

- 吨税执照上载明的负责人为法定纳税人
- 多方共同负责时，由主要负责人承担纳税义务
- 委托代理办理的，纳税义务仍由负责人承担

## 三、征税对象

### 征税范围

吨税的征税对象为**自中国境外港口进入中国境内港口的船舶**，包括：

1. **国际航行船舶**：从事国际运输的各类船舶
2. **港澳台船舶**：来自香港、澳门、台湾地区的船舶

### 不征税范围

以下情形不征收吨税：

1. **境内港口间航行**：仅在中国境内港口间航行的船舶
2. **非商业用途**：军事、外交等非商业用途的船舶
3. **其他法定免税情形**

## 四、征收机关

### 海关的职责

船舶吨税由**海关**负责征收，具体职责包括：

1. **税款征收**：计算、征收吨税
2. **执照签发**：核发《船舶吨税执照》
3. **监督检查**：检查船舶吨税缴纳情况
4. **违规处理**：对未按规定缴纳吨税的行为进行处理

### 办理流程

```
船舶进境 → 申报吨税 → 海关审核 → 缴纳税款 → 领取执照
```

## 五、吨税执照

### 执照的作用

《船舶吨税执照》是船舶已缴纳吨税的凭证，具有以下作用：

1. **完税证明**：证明船舶已依法缴纳吨税
2. **通行凭证**：海关放行船舶的必要条件
3. **期限证明**：明确吨税的有效期限

### 执照期限

吨税执照的有效期限分为三种：

| 期限类型 | 适用情形 |
|---------|---------|
| 30天 | 短期停靠、临时进境 |
| 90天 | 常规航行、一般贸易 |
| 1年 | 长期运营、频繁往来 |

### 注意事项

- 吨税执照期限内，船舶可多次进出境，无需重复缴税
- 执照期满后需重新申报缴纳吨税
- 执照应当随船携带，以备查验
