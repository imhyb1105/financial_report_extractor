# 船舶吨税计算方法

## 一、计算公式

### 基本公式

```
应纳税额 = 净吨位 × 适用税额标准
```

### 1年期执照计算

```
应纳税额 = 净吨位 × 90天税额标准 × 4
```

## 二、计算步骤

### 第一步：确定净吨位

以船舶吨位丈量证书载明的净吨位为准。

### 第二步：选择执照期限

根据船舶运营需要选择：
- 30天
- 90天
- 1年

### 第三步：确定适用税率类型

判断是否适用优惠税率：
- 适用优惠税率：与我国签订最惠国待遇条款的国家或地区
- 适用普通税率：其他情况

### 第四步：确定适用税额

根据净吨位区间确定适用税额。

### 第五步：计算应纳税额

代入公式计算。

## 三、计算示例

### 示例1：小型船舶（普通税率，30天）

**条件：**
- 净吨位：1500吨
- 执照期限：30天
- 税率类型：普通税率

**计算：**
```
应纳税额 = 1500 × 2.9 = 4350元
```

### 示例2：中型船舶（优惠税率，90天）

**条件：**
- 净吨位：8000吨
- 执照期限：90天
- 税率类型：优惠税率

**计算：**
```
应纳税额 = 8000 × 5.1 = 40800元
```

### 示例3：大型船舶（普通税率，1年）

**条件：**
- 净吨位：35000吨
- 执照期限：1年
- 税率类型：普通税率

**计算：**
```
应纳税额 = 35000 × 11.7 × 4 = 1638000元
```

### 示例4：超大型船舶（优惠税率，1年）

**条件：**
- 净吨位：80000吨
- 执照期限：1年
- 税率类型：优惠税率

**计算：**
```
应纳税额 = 80000 × 6.9 × 4 = 2208000元
```

## 五、特殊情形处理

### 1. 小额免税

**条件**：应纳税额在人民币50元以下

**计算**：
```
净吨位 × 税额标准 ≤ 50元
```

**示例**：
- 净吨位10吨，30天普通税率：10 × 2.9 = 29元 < 50元，免税

### 2. 初次进口船舶

**条件**：自境外以购买、受赠、继承等方式取得船舶所有权的初次进口

**处理**：
- 首次进口时免税
- 后续进口需正常缴纳吨税

### 3. 多次进出境

**条件**：在执照有效期内多次进出境

**处理**：
- 只需缴纳一次吨税
- 无需重复申报

### 4. 执照延期

**条件**：执照期满后需继续停留

**处理**：
- 期满前申请延期
- 或重新申请新执照

## 六、常见问题解答

### Q1：如何判断是否适用优惠税率？

**A**：查看船舶国籍，确认该国或地区是否与我国签订含有相互给予船舶税费最惠国待遇条款的协定。如有，则适用优惠税率。

### Q2：1年期执照如何计算税额？

**A**：1年期执照按90天税额标准乘以4计算。例如：10000吨船舶，普通税率1年期为10000 × 10.2 × 4 = 408000元。

### Q3：执照期限内多次进出境是否需要重复缴税？

**A**：不需要。在吨税执照有效期内，船舶可多次进出境，无需重复缴税。

### Q4：应纳税额不足50元是否需要缴税？

**A**：不需要。应纳税额在人民币50元以下的船舶免征吨税。

### Q5：执照丢失怎么办？

**A**：应及时向签发海关报告并申请补发，提供相关证明材料。

### Q6：逾期缴税有什么后果？

**A**：从滞纳税款之日起，按日加收滞纳税款万分之五的滞纳金。

### Q7：非机动驳船是否需要缴纳吨税？

**A**：需要。非机动船舶免税规定不包括非机动驳船。

### Q8：渔船是否免税？

**A**：捕捞、养殖渔船免征吨税，但需持有相关证书。
