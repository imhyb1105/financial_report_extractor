# 智能财务报表数据提取工具

> 从PDF年报中快速、准确地提取核心财务数据，三模型AI验证，准确率95%+

## 项目概述

一款面向会计师、审计师、财务分析师、投资研究员等专业人士的智能财务报表数据提取工具。通过三模型AI验证机制，从PDF年报中快速、准确地提取核心财务数据和关键非财务信息。

### 核心价值

- **效率提升**: 从手工10+分钟缩短到AI处理5分钟
- **准确可靠**: 三模型验证机制，准确率95%+
- **智能核对**: 自动勾稽关系核对，错误自动重试
- **单位统一**: 支持元/万元/亿元自由切换
- **可追溯**: 每项数据标注来源页码和位置
- **灵活配置**: 用户自带API Key，支持多种AI模型

┌─────────────┐         ┌─────────────┐

       │                       │
       └───────────┬───────────┘

                   │
       ┌───────────┼───────────┐

## MVP功能清单

### 财务指标（21项）

| 类别 | 指标 |
|------|------|
| 收入利润 | 营业收入、营业成本、毛利润、净利润、归母净利润、扣非净利润 |
| 资产负债 | 总资产、总负债、净资产、流动资产、流动负债、应收账款、存货 |
| 现金流 | 经营活动现金流净额、投资活动现金流净额、筹资活动现金流净额 |
| 计算指标 | 毛利率、净利率、资产负债率、流动比率、ROE |

### 非财务信息（5类）

| 类型 | 内容 |
|------|------|
| 公司概况 | 主营业务、所属行业、成立时间 |
| 风险提示 | 前3-5条重大风险因素 |
| 重大事项 | 诉讼、担保、关联交易等 |
| 未来规划 | 投资计划、战略方向 |
| 分红方案 | 分红金额、比例、日期 |
