# 资源税基础知识

**版本：2026.03-001**

---

## 一、资源税概述

### 1.1 定义

资源税是对在我国领域和管辖海域开采应税资源的单位和个人，就其应税资源的销售额或者销售数量征收的一种税。

### 1.2 特点

- **特定性**：仅对特定自然资源征税
- **调节性**：调节资源级差收入
- **有偿性**：体现国有资源有偿使用原则
- **普遍性**：对开发国有资源普遍征收

### 1.3 立法目的

1. 促进资源节约集约利用
2. 保护生态环境
3. 调节资源级差收入
4. 增加财政收入

---

## 二、纳税义务人

### 2.1 纳税人

在中华人民共和国领域和管辖海域开采应税资源的单位和个人，为资源税的纳税人。

**具体包括：**

| 类型 | 说明 |
|------|------|
| 企业 | 国有企业、集体企业、私营企业、外商投资企业等 |
| 事业单位 | 开采应税资源的事业单位 |
| 其他单位 | 社会团体、部队等 |
| 个体工商户 | 从事应税资源开采的个体经营者 |
| 个人 | 从事应税资源开采的自然人 |

### 2.2 扣缴义务人

收购未税矿产品的单位为资源税的扣缴义务人。

**主要包括：**
- 独立矿山
- 联合企业
- 其他收购未税矿产品的单位

## 三、征税范围

### 3.1 应税资源分类

资源税的征税范围包括以下五大类：

#### （一）能源矿产

| 税目 | 说明 |
|------|------|
| 原油 | 天然原油，不含人造石油 |
| 天然气 | 专门开采或与原油同时开采的天然气 |
| 煤炭 | 原煤和以未税原煤加工的洗选煤 |
| 煤层气 | 煤层瓦斯 |
| 页岩气 | 页岩层中开采的天然气 |
| 天然卤水 | 含盐天然卤水 |
| 地热 | 地下热水资源 |
| 砂石 | 建筑用砂石 |
| 石煤、油页岩 | 其他能源矿产 |

#### （二）金属矿产

| 税目 | 说明 |
|------|------|
| 黑色金属 | 铁矿、锰矿、铬矿、钒矿、钛矿 |
| 有色金属 | 铜矿、铅锌矿、铝土矿、钨矿、锡矿、锑矿、钼矿、镍矿、钴矿等 |
| 贵金属 | 金矿、银矿、铂族金属 |
| 稀有金属 | 稀土、铌、钽、锆、锂、锶等 |

#### （三）非金属矿产

| 税目 | 说明 |
|------|------|
| 矿物类 | 高岭土、石灰岩、磷矿、硫铁矿、萤石等 |
| 岩石类 | 花岗岩、大理岩、石英岩、砂岩等 |
| 宝玉石类 | 宝石、玉石、玛瑙等 |

#### （四）水气矿产

| 税目 | 说明 |
|------|------|
| 矿泉水 | 饮用天然矿泉水 |
| 二氧化碳气 | 天然二氧化碳气 |
| 硫化氢气 | 天然硫化氢气 |
| 氦气 | 天然氦气 |

#### （五）盐

| 税目 | 说明 |
|------|------|
| 海盐 | 海水晒制的盐 |
| 湖盐 | 盐湖开采的盐 |
| 井矿盐 | 地下盐矿开采的盐 |

### 3.2 不征税项目

以下项目不属于资源税征税范围：

1. 人造石油
2. 洗煤、选煤（以未税原煤加工的除外）
3. 进口应税资源产品
4. 出口应税资源产品（在国内已纳税的）

## 四、计税依据

### 4.1 计征方式

资源税实行从价计征和从量计征两种方式：

| 计征方式 | 计税依据 | 适用范围 |
|----------|----------|----------|
| 从价计征 | 销售额 | 大部分矿产品 |
| 从量计征 | 销售数量 | 部分特定矿产品 |

### 4.2 销售额的确定

**销售额是指纳税人销售应税产品向购买方收取的全部价款和价外费用。**

包括：
- 销售货物价款
- 价外费用（手续费、补贴、基金、集资费、返还利润、奖励费、违约金、滞纳金、延期付款利息、赔偿金、代收款项、代垫款项、包装费、包装物租金、储备费、优质费、运输装卸费等）

不包括：
- 增值税销项税额
- 符合条件的代垫运输费用
- 代为收取的政府性基金或行政事业性收费

### 4.3 销售数量的确定

**销售数量是指纳税人销售应税产品的实际数量。**

## 五、纳税义务发生时间

| 销售方式 | 纳税义务发生时间 |
|----------|------------------|
| 采取分期收款 | 销售合同规定的收款日期当天 |
| 采取预收货款 | 发出应税产品当天 |
| 采取其他结算方式 | 收讫销售款或取得索取销售款凭据当天 |
| 自产自用 | 移送使用应税产品当天 |
| 扣缴义务人代扣代缴 | 支付首笔货款或首次开具支付货款凭据当天 |

---

## 六、纳税期限与地点

### 6.1 纳税期限

- 按月或按季申报纳税
- 按月申报：次月15日内
- 按季申报：季度终了次月15日内

### 6.2 纳税地点

向应税产品开采地或生产地税务机关申报纳税。
