# 资源税规定

**版本：2026.03-001**

---

## 一、计征方式

### 1.1 从价计征

**定义**：以应税产品的销售额为计税依据，按照适用比例税率计算应纳税额。

**适用范围**：大部分应税矿产品

**计算公式**：
```
应纳税额 = 销售额 × 适用税率
```

**销售额的组成**：

| 包括项目 | 不包括项目 |
|----------|------------|
| 销售价款 | 增值税销项税额 |
| 价外费用 | 符合条件的代垫运费 |
| 补贴收入 | 代收政府性基金 |
| 违约金 | 行政事业性收费 |
| 滞纳金 | - |
| 延期付款利息 | - |
| 包装费 | - |

### 1.2 从量计征

**定义**：以应税产品的销售数量为计税依据，按照适用定额税率计算应纳税额。

**适用范围**：
- 地热
- 砂石
- 矿泉水
- 天然卤水
- 盐类等

**计算公式**：
```
应纳税额 = 销售数量 × 适用税额
```

**销售数量的确定**：

| 情形 | 销售数量确定 |
|------|--------------|
| 销售应税产品 | 实际销售数量 |
| 自用应税产品 | 移送使用数量 |
| 收购未税矿产品 | 收购数量 |

## 二、销售额的特殊规定

### 2.1 销售额的核定

纳税人申报的应税产品销售额明显偏低且无正当理由的，税务机关按下列顺序核定销售额：

1. **按纳税人当月同类产品平均销售价格确定**
2. **按纳税人最近时期同类产品平均销售价格确定**
3. **按组成计税价格确定**：
   ```
   组成计税价格 = 成本 × (1 + 成本利润率) ÷ (1 - 税率)
   ```

### 2.2 关联交易

纳税人开采或生产应税产品自用于连续生产应税产品的，不缴纳资源税；自用于其他方面的，视同销售，按规定缴纳资源税。

**关联企业间销售额偏低时的处理**：
- 按照独立企业之间相同或类似业务价格核定
- 按照成本加合理费用和利润的方法核定
- 按照其他合理方法核定

### 2.3 混合销售

纳税人开采或生产应税产品同时销售原矿和选矿的：

| 情形 | 处理方式 |
|------|----------|
| 分别核算 | 分别适用税率计税 |
| 未分别核算 | 从高适用税率 |

### 2.4 洗选煤销售额

洗选煤销售额按下列公式折算：
```
洗选煤销售额 = 洗选煤销售额 × 折算率
```

**折算率**由省级财税部门确定，通常在60%-80%之间。

## 三、税收优惠

### 3.1 法定减免

以下项目免征资源税：

| 减免项目 | 减免幅度 | 条件 |
|----------|----------|------|
| 开采原油用于加热、修井 | 免征 | 用于原油生产 |
| 油田运输过程中加热原油 | 免征 | 自用 |
| 煤层气抽采 | 免征 | - |
| 低品位矿开采 | 减征30% | 符合条件 |
| 尾矿再利用 | 减征50% | 利用尾矿 |
| 充填开采 | 减征50% | 充填开采方式 |
| 衰竭期矿山开采 | 减征30% | 剩余可采储量下降 |
| 共伴生矿综合利用 | 免征 | 主矿产品外收入 |
| 深水油气田开采 | 减征30% | 水深超过300米 |

### 3.2 地方减免

省、自治区、直辖市可决定对以下情形给予减免：

1. 纳税人开采或生产应税产品过程中，因意外事故或自然灾害等原因遭受重大损失
2. 鼓励低品位、难选冶矿产资源开发利用
3. 促进尾矿资源综合利用
4. 支持老矿山接替资源开发利用

**减免权限**：
- 减征幅度：由省级政府确定
- 减免期限：原则上不超过3年
- 需报财政部、税务总局备案

### 3.3 优惠申请流程

```
1. 纳税人自行判断是否符合优惠条件
       ↓
2. 向主管税务机关提交优惠申请
       ↓
3. 税务机关审核确认
       ↓
4. 纳税申报时享受优惠
       ↓
5. 税务机关后续管理核查
```

## 四、特定行业规定

### 4.1 石油天然气行业

**原油天然气销售额的特殊规定**：

| 项目 | 规定 |
|------|------|
| 原油 | 按原油的销售额计税 |
| 天然气 | 按天然气的销售额计税 |
| 稠油、高凝油 | 减征40%资源税 |
| 三次采油 | 减征30%资源税 |
| 深水油气 | 减征30%资源税 |

**三次采油定义**：指油田在二次采油后，采用聚合物驱、化学驱、热采、气驱等提高采收率技术进行的采油。

### 4.2 煤炭行业

**煤炭销售额的规定**：

| 煤炭类型 | 计税依据 |
|----------|----------|
| 原煤 | 原煤销售额 |
| 洗选煤 | 洗选煤销售额 × 折算率 |
| 煤矸石 | 免征资源税 |

**折算率计算**：
```
折算率 = 原煤平均销售价格 ÷ 洗选煤平均销售价格 × 100%
```

### 4.3 稀土行业

**稀土资源税的特殊规定**：

| 项目 | 规定 |
|------|------|
| 税率 | 7%-12%（较高税率） |
| 计征方式 | 从价计征 |
| 轻稀土 | 内蒙古、四川、山东适用税率 |
| 中重稀土 | 江西等南方省份适用税率 |

### 4.4 盐业

**盐资源税的规定**：

| 盐类 | 计征方式 | 说明 |
|------|----------|------|
| 海盐 | 从量计征 | 海水晒制 |
| 湖盐 | 从量计征 | 盐湖开采 |
| 井矿盐 | 从量计征 | 地下盐矿 |
| 液体盐 | 从量计征 | 卤水 |

## 五、纳税申报规定

### 5.1 纳税期限

| 纳税方式 | 申报期限 | 缴款期限 |
|----------|----------|----------|
| 按月申报 | 次月15日内 | 次月15日内 |
| 按季申报 | 季度终了次月15日内 | 季度终了次月15日内 |

### 5.2 申报地点

```
纳税地点 = 应税产品开采地或生产地
```

**跨区域开采的处理**：
- 分别向各开采地税务机关申报纳税
- 各开采地销售额或销售数量分别计算

### 5.3 申报资料

纳税人申报资源税时应提供：

1. 资源税纳税申报表
2. 应税产品销售额、销售数量明细
3. 税收优惠相关证明材料（如适用）
4. 其他税务机关要求的资料

## 六、法律责任

### 6.1 违法行为及处罚

| 违法行为 | 处罚标准 |
|----------|----------|
| 未按规定期限申报纳税 | 按日加收滞纳金0.05% |
| 偷税 | 补税+滞纳金+0.5-5倍罚款 |
| 抗税 | 补税+滞纳金+1-5倍罚款 |
| 骗税 | 追回税款+1-5倍罚款 |
| 虚假申报 | 责令限期改正+罚款 |

### 6.2 滞纳金计算

```
滞纳金 = 应纳税额 × 滞纳天数 × 0.05%
```

## 七、争议处理

### 7.1 纳税争议解决途径

```
1. 与税务机关沟通协商
       ↓
2. 税务行政复议
       ↓
3. 税务行政诉讼
```

### 7.2 复议前置

纳税人同税务机关在纳税上发生争议时，必须先依照税务机关的纳税决定缴纳或者解缴税款及滞纳金或者提供相应的担保，然后可以依法申请行政复议。
