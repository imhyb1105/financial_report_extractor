# 车船税基础知识

**版本：** 2026.03-001
**更新日期：** 2026年3月

---

## 一、车船税概述

### 1.1 什么是车船税

车船税是指在中华人民共和国境内的车辆、船舶的所有人或者管理人，依照《中华人民共和国车船税法》应缴纳的一种财产税。

车船税属于财产税类，是对纳税人拥有的车船这一特定财产征收的税种。

### 1.2 车船税的特点

| 特点 | 说明 |
|------|------|
| 财产税性质 | 对纳税人拥有的车船财产征税 |
| 征税对象特定 | 仅对车辆和船舶征税 |
| 按年征收 | 每年缴纳一次 |
| 定额税率 | 按车船的种类、吨位等实行固定税额 |
| 代收代缴 | 保险机构代收代缴 |

---

## 二、纳税人

### 2.1 纳税人定义

在中华人民共和国境内，车辆、船舶（以下简称车船）的所有人或者管理人为车船税的纳税人。

### 2.2 纳税人类型

| 纳税人类型 | 说明 |
|------------|------|
| 所有人 | 车船的产权所有人 |
| 管理人 | 对车船具有管理权和使用权的人 |

### 2.3 特殊情况

- **所有权与管理权分离时**：由管理人缴纳车船税
- **车船租赁时**：原则上由所有人缴纳，但可约定由使用人缴纳
- **车船管理人与所有人不一致时**：由管理人缴纳

---

## 三、征税范围

### 3.1 征税对象

车船税的征税对象是依法应当在车船登记管理部门登记的机动车辆和船舶，以及依法不需要在车船登记管理部门登记的在单位内部场所行驶或者作业的机动车辆和船舶。

### 3.2 车辆范围

#### 3.2.1 乘用车

乘用车是指在设计和技术特性上主要用于载运乘客及随身行李，核定载客人数包括驾驶员在内不超过9人的汽车。

**按排气量分类：**

| 排气量（升） | 类别 |
|--------------|------|
| 1.0及以下 | 小排量乘用车 |
| 1.0-1.6（含） | 普通乘用车 |
| 1.6-2.0（含） | 中等排量乘用车 |
| 2.0-2.5（含） | 中大排量乘用车 |
| 2.5-3.0（含） | 大排量乘用车 |
| 3.0-4.0（含） | 高排量乘用车 |
| 4.0以上 | 豪华乘用车 |

#### 3.2.2 商用车

商用车是指除乘用车外，在设计和技术特性上用于载运乘客、货物及特殊用途的汽车。

| 商用车类型 | 说明 |
|------------|------|
| 客车 | 核定载客人数9人以上的载客汽车 |
| 货车 | 载货汽车、半挂牵引车、挂车等 |
| 专用作业车 | 具有特殊用途的车辆（如消防车、救护车等） |

#### 3.2.3 摩托车

摩托车是指无论采用何种驱动方式，最高设计车速大于每小时50公里，或者使用内燃机，其排量大于50毫升的两轮或者三轮车辆。

#### 3.2.4 其他车辆

| 车辆类型 | 说明 |
|----------|------|
| 三轮汽车 | 最高设计车速不超过每小时50公里的三轮车辆 |
| 低速载货汽车 | 以柴油机为动力，最高设计车速不超过每小时70公里的四轮货车 |
| 挂车 | 无动力驱动的载货车辆 |

### 3.3 船舶范围

#### 3.3.1 机动船舶

机动船舶是指依靠燃料等能源为动力运行的船舶，包括客船、货船、气垫船等。

#### 3.3.2 非机动驳船

非机动驳船是指无动力驱动的驳运船舶。

#### 3.3.3 游艇

游艇是指长度大于8米小于90米，船体为玻璃钢、钢、铝合金、钛合金、碳纤维、聚酯纤维等材质，用于游览观光、休闲娱乐等活动的船舶。

## 四、纳税义务发生时间

### 4.1 一般规定

车船税纳税义务发生时间为取得车船所有权或者管理权的当月。

### 4.2 具体情形

| 情形 | 纳税义务发生时间 |
|------|------------------|
| 新购置车船 | 购置发票所载日期的当月 |
| 取得车船所有权 | 所有权转移的当月 |
| 取得车船管理权 | 管理权取得的当月 |

### 4.3 新购置车船

纳税人新购置车船，购置当年的应纳税额自纳税义务发生的当月起按月计算。计算公式为：

**应纳税额 = 年应纳税额 / 12 * 应纳税月份数**

---

## 五、纳税地点

### 5.1 车辆

纳税人自行申报缴纳车船税的，纳税地点为车船的登记地；由保险机构代收代缴车船税的，纳税地点为保险机构所在地。

### 5.2 船舶

船舶的纳税地点为船籍港所在地。

---

## 六、纳税期限

| 纳税方式 | 纳税期限 |
|----------|----------|
| 自行申报 | 车船税按年申报，分月计算，一次性缴纳 |
| 代收代缴 | 购买机动车交通事故责任强制保险时一并缴纳 |

---

## 七、征收管理

### 7.1 代收代缴制度

从事机动车第三者责任强制保险业务的保险机构为机动车车船税的扣缴义务人，应当在收取保险费时依法代收车船税，并出具代收税款凭证。

### 7.2 代收代缴义务

| 义务 | 内容 |
|------|------|
| 代收义务 | 在销售交强险时代收车船税 |
| 代缴义务 | 按规定期限向税务机关解缴税款 |
| 出具凭证 | 向纳税人出具代收税款凭证 |
| 信息报送 | 向税务机关报送代收代缴税款信息 |

---

## 法律依据

1. 《中华人民共和国车船税法》第一条至第六条
2. 《中华人民共和国车船税法实施条例》第二条至第八条
