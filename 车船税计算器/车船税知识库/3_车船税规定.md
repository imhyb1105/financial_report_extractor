# 车船税规定

**版本：** 2026.03-001
**更新日期：** 2026年3月

---

## 第一节 法定减免

### 1.1 法定免征车船税的车船

根据《车船税法》规定，下列车船免征车船税：

| 序号 | 免征车船类型 | 说明 |
|------|--------------|------|
| 1 | 捕捞、养殖渔船 | 在渔业船舶登记管理部门登记为捕捞船或者养殖船的船舶 |
| 2 | 军队、武装警察部队专用的车船 | 按照规定在军队、武装警察部队车船登记管理部门登记 |
| 3 | 警用车船 | 公安机关、国家安全机关、监狱、劳动教养管理机关和法院、检察院领取警用牌照的车辆和执行警务的专用船舶 |
| 4 | 依照法律规定应当予以免税的外国驻华使领馆、国际组织驻华代表机构及其有关人员的车船 | 依照有关法律规定 |
| 5 | 对新能源车船，免征或者减半征收车船税 | 见本文件第三节 |

### 1.2 法定减征车船税

| 减征对象 | 减征幅度 | 条件 |
|----------|----------|------|
| 挂车 | 50% | 货车税额的50% |
| 非机动驳船 | 50% | 机动船舶税额的50% |
| 拖船 | 50% | 机动船舶税额的50% |

## 第二节 特定减免

### 2.1 省、自治区、直辖市人民政府减免

省、自治区、直辖市人民政府根据当地实际情况，可以对公共交通车船，农村居民拥有并主要在农村地区使用的摩托车、三轮汽车和低速载货汽车定期减征或者免征车船税。

### 2.2 常见地方减免政策

| 减免对象 | 减免类型 | 说明 |
|----------|----------|------|
| 公交车辆 | 减免 | 各地政策不同 |
| 农村摩托车 | 减免 | 主要在农村使用 |
| 三轮汽车 | 减免 | 主要在农村使用 |
| 低速载货汽车 | 减免 | 主要在农村使用 |

## 第三节 新能源汽车优惠政策

### 3.1 新能源汽车减免政策

根据《关于延续和优化新能源汽车车辆购置税减免政策的公告》及相关规定：

#### 3.1.1 免征车船税的新能源汽车

| 车型类型 | 优惠政策 | 条件 |
|----------|----------|------|
| 纯电动汽车 | 免征车船税 | 符合新能源汽车标准 |
| 燃料电池汽车 | 免征车船税 | 符合新能源汽车标准 |
| 插电式混合动力汽车 | 免征车船税 | 符合新能源汽车标准 |

#### 3.1.2 新能源汽车认定标准

| 指标 | 要求 |
|------|------|
| 纯电动续驶里程 | 符合国家标准 |
| 电池能量密度 | 符合国家标准 |
| 整车能耗 | 符合国家标准 |
| 免征目录 | 列入《免征车辆购置税的新能源汽车车型目录》 |

### 3.2 新能源汽车类型说明

| 类型 | 特点 | 充电方式 |
|------|------|----------|
| 纯电动汽车（BEV） | 仅由电动机驱动 | 外部充电 |
| 插电式混合动力汽车（PHEV） | 电动机+内燃机 | 外部充电+燃油 |
| 燃料电池汽车（FCEV） | 氢燃料电池驱动 | 加氢 |

### 3.3 节能汽车优惠政策

#### 3.3.1 减半征收车船税的节能汽车

| 车型类型 | 优惠政策 | 条件 |
|----------|----------|------|
| 节能乘用车 | 减半征收 | 符合节能汽车标准 |
| 节能商用车 | 减半征收 | 符合节能汽车标准 |

#### 3.3.2 节能乘用车认定标准

| 指标 | 要求 |
|------|------|
| 发动机排量 | 1.6升及以下 |
| 综合工况油耗 | 符合节能标准 |
| 节能目录 | 列入节能汽车目录 |

## 第四节 退税规定

### 4.1 可申请退税的情形

| 情形 | 说明 |
|------|------|
| 车船被盗抢 | 经县级以上公安机关证明 |
| 车船报废 | 提供相关证明 |
| 车船灭失 | 提供相关证明 |
| 多缴税款 | 提供完税证明 |

### 4.2 退税计算

**计算公式：**

```
退税额 = 已缴纳税额 / 12 * 未使用月份数
```

### 4.3 退税流程

| 步骤 | 内容 |
|------|------|
| 1 | 准备退税申请材料 |
| 2 | 向税务机关提交申请 |
| 3 | 税务机关审核 |
| 4 | 办理退税手续 |

### 4.4 退税所需材料

| 材料 | 说明 |
|------|------|
| 退税申请表 | 税务机关提供 |
| 完税证明 | 原件及复印件 |
| 身份证明 | 原件及复印件 |
| 相关证明材料 | 盗抢证明、报废证明等 |

## 第五节 违章处理

### 5.1 未按规定缴纳车船税

| 迱章行为 | 处罚标准 |
|----------|----------|
| 未按规定期限缴纳税款 | 从滞纳税款之日起，按日加收滞纳税款万分之五的滞纳金 |
| 拒不缴纳 | 税务机关可采取强制执行措施 |

### 5.2 滞纳金计算

**计算公式：**

```
滞纳金 = 滞纳税额 * 滞纳天数 * 0.05%
```

### 5.3 保险机构违规处理

| 迱章行为 | 处罚标准 |
|----------|----------|
| 未按规定代收车船税 | 处应代收代缴税款50%以上3倍以下罚款 |
| 未按规定解缴税款 | 处未解缴税款50%以上5倍以下罚款 |

## 第六节 其他规定

### 6.1 已缴纳车船税的车船被盗抢

已缴纳车船税的车船被盗抢、报废、灭失的，纳税人可以凭有关管理机关出具的证明和完税凭证，向纳税所在地的主管税务机关申请退还自被盗抢、报废、灭失月份起至该纳税年度终了期间的税款。

### 6.2 车船转让

已缴纳车船税的车船在同一纳税年度内转让过户的，不另纳税，也不退税。

### 6.3 临时入境车船

临时入境的外国车船和香港特别行政区、澳门特别行政区、台湾地区的车船，不征收车船税。

### 6.4 外商投资企业和外国企业

外商投资企业、外国企业、港澳台投资企业以及外籍个人，按照《车船税法》规定缴纳车船税。

## 第七节 常见问题解答

### Q1：购买二手车是否需要缴纳车船税？

答：车船税按年缴纳，二手车过户时不需要额外缴纳当年车船税，但购买时应核实原车主是否已缴纳当年车船税。

### Q2：车辆报废后车船税可以退吗？

答：可以。车辆报废后，纳税人可凭报废证明向税务机关申请退还自报废月份起至该纳税年度终了期间的税款。

### Q3：忘记缴纳车船税怎么办？

答：应尽快补缴，并需缴纳滞纳金（按日加收滞纳税款万分之五）。可通过保险公司补缴或前往税务机关办理。

### Q4：车辆在外地使用，在哪里缴纳车船税？

答：车辆车船税在车辆登记地缴纳。通过交强险代收代缴的，在办理交强险时一并缴纳。

### Q5：新能源车如何办理车船税减免？

答：购买新能源车时，保险公司会自动识别并免征车船税。需确保车辆列入《免征车辆购置税的新能源汽车车型目录》。
