# 车船税计算方法

**版本：** 2026.03-001
**更新日期：** 2026年3月

---

## 第一节 基本计算公式

### 1.1 按辆计征

适用于乘用车、客车、摩托车。

**计算公式：**

```
应纳税额 = 适用年税额 * 辆数
```

### 1.2 按整备质量计征

适用于货车、挂车、专用作业车、轮式专用机械车。

**计算公式：**

```
应纳税额 = 整备质量（吨） * 适用年税额（元/吨）
```

### 1.3 按净吨位计征

适用于机动船舶、非机动驳船。

**计算公式：**

```
应纳税额 = 净吨位（吨） * 适用年税额（元/吨）
```

### 1.4 按艇身长度计征

适用于游艇。

**计算公式：**

```
应纳税额 = 艇身长度（米） * 适用年税额（元/米）
```

## 第二节 年度应纳税额计算

### 2.1 乘用车年应纳税额计算

**示例1：** 计算1.6升排量乘用车的年应纳税额（以北京市标准为例）

| 项目 | 数据 |
|------|------|
| 排气量 | 1.6升 |
| 适用档次 | 1.0升-1.6升（含） |
| 北京市年税额 | 420元/辆 |
| 应纳税额 | **420元** |

**示例2：** 计算2.5升排量乘用车的年应纳税额（以北京市标准为例）

| 项目 | 数据 |
|------|------|
| 排气量 | 2.5升 |
| 适用档次 | 2.0升-2.5升（含） |
| 北京市年税额 | 720元/辆 |
| 应纳税额 | **720元** |

### 2.2 货车年应纳税额计算

**示例：** 计算整备质量8吨货车的年应纳税额（以某省标准60元/吨为例）

| 项目 | 数据 |
|------|------|
| 整备质量 | 8吨 |
| 适用年税额 | 60元/吨 |
| 应纳税额 | 8 * 60 = **480元** |

### 2.3 挂车年应纳税额计算

挂车税额为货车税额的50%。

**示例：** 计算整备质量10吨挂车的年应纳税额（以货车税额60元/吨为例）

| 项目 | 数据 |
|------|------|
| 整备质量 | 10吨 |
| 货车年税额 | 60元/吨 |
| 挂车年税额 | 60 * 50% = 30元/吨 |
| 应纳税额 | 10 * 30 = **300元** |

### 2.4 船舶年应纳税额计算

**示例1：** 计算净吨位5000吨机动船舶的年应纳税额

| 项目 | 数据 |
|------|------|
| 净吨位 | 5000吨 |
| 适用档次 | 2001吨-10000吨 |
| 适用年税额 | 5元/吨 |
| 应纳税额 | 5000 * 5 = **25000元** |

**示例2：** 计算艇身长度15米游艇的年应纳税额

| 项目 | 数据 |
|------|------|
| 艇身长度 | 15米 |
| 适用档次 | 10.01米-18米 |
| 适用年税额 | 900元/米 |
| 应纳税额 | 15 * 900 = **13500元** |

## 第三节 部分年度应纳税额计算

### 3.1 新购置车船计算

新购置车船，购置当年的应纳税额自纳税义务发生的当月起按月计算。

**计算公式：**

```
应纳税额 = 年应纳税额 / 12 * 应纳税月份数
```

**示例：** 2026年5月新购置一辆1.6升乘用车（年税额420元）

| 项目 | 数据 |
|------|------|
| 年应纳税额 | 420元 |
| 购置月份 | 5月 |
| 应纳税月份 | 5月-12月（8个月） |
| 应纳税额 | 420 / 12 * 8 = **280元** |

### 3.2 年度中途转让过户

已缴纳车船税的车船在同一纳税年度内转让过户的，不另纳税，也不退税。

**示例：** 2026年3月转让一辆已缴纳全年车船税的乘用车

| 情况 | 处理 |
|------|------|
| 原车主 | 不退税 |
| 新车主 | 不需额外缴纳当年车船税 |

### 3.3 年度中途报废、被盗抢

可申请退还自报废、被盗抢月份起至该纳税年度终了期间的税款。

**计算公式：**

```
退税额 = 年应纳税额 / 12 * 剩余月份数
```

**示例：** 2026年4月报废一辆年税额420元的乘用车

| 项目 | 数据 |
|------|------|
| 年应纳税额 | 420元 |
| 报废月份 | 4月 |
| 剩余月份 | 5月-12月（8个月） |
| 可退税额 | 420 / 12 * 8 = **280元** |

|------|------|

## 第五节 滞纳金计算

### 5.1 滞纳金计算公式

```
滞纳金 = 滞纳税额 * 滞纳天数 * 0.05%
```

### 5.2 滞纳金计算示例

**示例：** 年税额420元，滞纳60天

| 项目 | 数据 |
|------|------|
| 滞纳税额 | 420元 |
| 滞纳天数 | 60天 |
| 日滞纳金率 | 0.05% |
| 滞纳金 | 420 * 60 * 0.05% = **12.6元** |
| 合计应缴 | 420 + 12.6 = **432.6元** |

## 第六节 综合计算实例

### 实例1：企业车队车船税计算

**某企业拥有以下车辆（假设某省税额）：**

| 车辆类型 | 数量 | 参数 | 年税额标准 |
|----------|------|------|------------|
| 乘用车 | 3辆 | 2.0升 | 480元/辆 |
| 客车 | 2辆 | 30座 | 960元/辆 |
| 货车 | 5辆 | 整备质量10吨 | 60元/吨 |
| 挂车 | 2辆 | 整备质量8吨 | 30元/吨 |

**计算过程：**

| 车辆类型 | 计算式 | 应纳税额 |
|----------|--------|----------|
| 乘用车 | 3 * 480 | 1440元 |
| 客车 | 2 * 960 | 1920元 |
| 货车 | 5 * 10 * 60 | 3000元 |
| 挂车 | 2 * 8 * 30 | 480元 |
| **合计** | | **6840元** |

### 实例2：个人多车辆计算

**张先生拥有以下车辆（以北京市标准）：**

| 车辆 | 排气量 | 年税额 |
|------|--------|--------|
| 私家车1 | 1.5升 | 420元 |
| 私家车2 | 2.5升 | 720元 |
| 摩托车 | - | 120元 |

**计算：**

| 项目 | 计算式 | 金额 |
|------|--------|------|
| 私家车1 | 1 * 420 | 420元 |
| 私家车2 | 1 * 720 | 720元 |
| 摩托车 | 1 * 120 | 120元 |
| **合计** | | **1260元** |

### 实例3：新购车+报废车综合计算

**情景：**
- 2026年3月报废一辆1.6升乘用车（已缴全年420元）
- 2026年6月新购置一辆2.0升乘用车（年税额480元）

**计算：**

| 项目 | 计算式 | 金额 |
|------|--------|------|
| 报废车可退税额 | 420 / 12 * 9（4-12月） | 315元 |
| 新购车应纳税额 | 480 / 12 * 7（6-12月） | 280元 |
| **实际应补缴/退税额** | 280 - 315 | **退35元** |

## 第七节 计算注意事项

### 7.1 计税单位换算

| 换算项目 | 换算关系 |
|----------|----------|
| 千瓦与马力 | 1千瓦 = 1.36马力 |
| 拖船功率换算 | 1千瓦 = 0.67净吨位 |

### 7.2 排气量确定

| 确定方式 | 说明 |
|----------|------|
| 车辆登记证书 | 以登记的排气量为准 |
| 车辆合格证 | 出厂标定的排气量 |
| 车辆行驶证 | 记载的排气量 |

### 7.3 整备质量确定

| 确定方式 | 说明 |
|----------|------|
| 车辆登记证书 | 以登记的整备质量为准 |
| 车辆合格证 | 出厂标定的整备质量 |
| 实际称重 | 有争议时以实际称重为准 |

### 7.4 计算精度

| 项目 | 精度要求 |
|------|----------|
| 排气量 | 精确到0.1升 |
| 整备质量 | 精确到0.5吨 |
| 净吨位 | 精确到0.5吨 |
| 艇身长度 | 精确到0.1米 |
| 税额 | 精确到元（四舍五入） |

## 第八节 常见问题解答

### Q1：车辆排气量处于临界点如何确定税额？

答：排气量以车辆登记证书登记为准，处于临界点时按实际登记排气量确定。如1.599升按1.6升（含）以下档次计算。

### Q2：货车整备质量有小数如何计算？

答：整备质量不足0.5吨的按0.5吨计算，超过0.5吨不足1吨的按1吨计算。

### Q3：年度中途购买保险如何计算车船税？

答：保险公司会自动计算从购车当月至年底的应纳税额。

### Q4：跨省转移车辆如何缴纳车船税？

答：车辆转移登记后，在新的登记地缴纳车船税。当年已缴纳的不需重复缴纳。

### Q5：单位内部使用不上路的车辆是否缴纳车船税？

答：依法不需要在车船登记管理部门登记的在单位内部场所行驶或者作业的机动车辆和船舶，也需要缴纳车船税。
