# 车辆购置税基础知识

**版本：2026.03-001**
**更新日期：2026年3月16日**

---

## 一、什么是车辆购置税

车辆购置税是对在中华人民共和国境内购置应税车辆的单位和个人征收的一种税。

### 1.1 法律依据

- 《中华人民共和国车辆购置税法》于2018年12月29日通过
- 自2019年7月1日起正式施行
- 原《车辆购置税暂行条例》同时废止

### 1.2 税种特点

| 特点 | 说明 |
|------|------|
| **征税对象** | 特定车辆（汽车、摩托车、电车等） |
| **征税环节** | 购置环节一次性征收 |
| **价外税** | 不包含在车辆价格内，需另行缴纳 |
| **中央税** | 收入归中央政府所有 |

---

## 二、纳税人

### 2.1 纳税人定义

在中华人民共和国境内购置应税车辆的单位和个人，为车辆购置税的纳税人。

### 2.2 "购置"的含义

购置包括以下行为：

| 行为类型 | 具体说明 |
|----------|----------|
| 购买 | 从销售方购买车辆 |
| 进口 | 从境外进口车辆 |
| 自产自用 | 自行生产并自用 |
| 受赠 | 接受赠与获得车辆 |
| 获奖 | 以获奖方式获得车辆 |
| 其他方式 | 其他取得并自用的方式 |

### 2.3 单位和个人的范围

- **单位**：包括企业、行政单位、事业单位、军事单位、社会团体等
- **个人**：包括个体工商户和其他个人（自然人）

---

## 三、征税范围

### 3.1 应税车辆类型

根据《车辆购置税法》，应税车辆包括：

| 车辆类型 | 说明 |
|----------|------|
| 汽车 | 各类乘用车、商用车 |
| 有轨电车 | 在轨道上行驶的电动车辆 |
| 汽车挂车 | 由汽车牵引的挂车 |
| 排气量超过150毫升的摩托车 | 气缸容量大于150cc的摩托车 |

### 3.2 不属于征税范围的车辆

以下车辆不属于车辆购置税征税范围：

- 排气量不超过150毫升的摩托车
- 电动自行车（非机动车）
- 农用三轮运输车
- 铁道机车车辆

### 3.3 特殊情形说明

| 情形 | 是否征税 | 备注 |
|------|----------|------|
| 二手车交易 | 否 | 车辆购置税一次性征收，二手车不重复征税 |
| 车辆置换 | 视情况 | 新购车辆需缴纳，与置换车辆无关 |
| 企业自用车 | 是 | 企业购买自用车辆需缴纳 |
| 个人自用车 | 是 | 个人购买自用车辆需缴纳 |

---

## 四、纳税义务发生时间

### 4.1 一般规定

纳税义务发生时间为纳税人购置应税车辆的当日。

### 4.2 不同购置方式的时间确定

| 购置方式 | 纳税义务发生时间 |
|----------|------------------|
| 购买 | 购车发票开具当日 |
| 进口 | 海关进口车辆放行当日 |
| 自产自用 | 车辆移送使用当日 |
| 受赠、获奖等 | 取得车辆当日 |

---

## 五、纳税期限和地点

### 5.1 纳税期限

- 纳税人应当自纳税义务发生之日起**60日内**申报缴纳车辆购置税

### 5.2 纳税地点

- 需要办理车辆登记的：向车辆登记地的主管税务机关申报纳税
- 不需要办理车辆登记的：向纳税人所在地的主管税务机关申报纳税

---

## 六、计税价格

### 6.1 计税价格的确定

| 购置方式 | 计税价格 |
|----------|----------|
| 购买自用 | 实际支付给销售者的全部价款（不含增值税） |
| 进口自用 | 关税完税价格 + 关税 + 消费税 |
| 自产自用 | 按照同类应税车辆的市场销售价格确定 |
| 受赠、获奖等 | 按照购置应税车辆时相关凭证载明的价格确定 |

### 6.2 最低计税价格

- 税务总局对部分车型核定最低计税价格
- 实际计税价格低于最低计税价格的，按最低计税价格计税
