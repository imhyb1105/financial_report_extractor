**更新日期：2026年3月16日**

## 一、法定减免税规定

### 1.1 法定免税车辆

根据《车辆购置税法》规定，下列车辆免征车辆购置税：

| 序号 | 免税车辆类型 | 说明 |
|------|--------------|------|
| 1 | 外国驻华使馆、领事馆和国际组织驻华机构及其外交人员自用的车辆 | 外交豁免 |
| 2 | 中国人民解放军和中国人民武装警察部队列入装备订货计划的车辆 | 军用车辆 |
| 3 | 悬挂应急救援专用号牌的国家综合性消防救援车辆 | 消防救援 |
| 4 | 设有固定装置的非运输专用作业车辆 | 工程作业车辆 |
| 5 | 城市公交企业购置的公共汽电车辆 | 公共交通 |

### 1.2 专用作业车辆说明

设有固定装置的非运输专用作业车辆包括：

| 车辆类型 | 典型示例 |
|----------|----------|
| 工程机械 | 挖掘机、推土机、装载机、起重机等 |
| 市政作业 | 扫路车、洒水车、垃圾车等 |
| 电力作业 | 电力工程车、带电作业车等 |
| 通信作业 | 通信工程车、应急通信车等 |
| 医疗救护 | 救护车、医疗检查车等 |

## 二、新能源汽车减免政策

### 2.1 政策背景

为支持新能源汽车产业发展，国家制定了新能源汽车车辆购置税减免政策。

### 2.2 政策时间表

| 时间段 | 政策内容 | 文号 |
|--------|----------|------|
| 2014年9月1日 - 2017年12月31日 | 免征 | 财税〔2014〕57号 |
| 2018年1月1日 - 2020年12月31日 | 免征 | 财税〔2017〕172号 |
| 2021年1月1日 - 2022年12月31日 | 免征 | 财政部公告2020年第21号 |
| 2023年1月1日 - 2023年12月31日 | 免征 | 财政部公告2022年第27号 |
| 2024年1月1日 - 2025年12月31日 | 免征 | 2023年第10号公告 |
| 2026年1月1日 - 2027年12月31日 | 减半征收 | 2023年第10号公告 |

### 2.3 减免税限额政策

根据2023年第10号公告，新能源乘用车减免税额实行限额：

| 时间段 | 减免方式 | 限额标准 |
|--------|----------|----------|
| 2024.1.1 - 2025.12.31 | 免征 | 每辆不超过30,000元 |
| 2026.1.1 - 2027.12.31 | 减半 | 每辆不超过15,000元 |

### 2.4 超限额计算方法

当车辆购置税税额超过限额时：

**2024-2025年（免征期）**：
```
实际应纳税额 = 计税价格 × 10% - 30,000（若结果为负则取0）
```

**2026-2027年（减半期）**：
```
减税额 = min(计税价格 × 10% ÷ 2, 15,000)
实际应纳税额 = 计税价格 × 10% - 减税额
```

## 四、退税规定

### 4.1 可申请退税的情形

| 情形 | 说明 |
|------|------|
| 车辆退回 | 已缴税车辆退回生产企业或经销商 |
| 免税条件消失 | 免税车辆因转让、改变用途等不再符合免税条件 |
| 公安机关不予办理登记 | 已缴税车辆无法办理车辆登记 |

### 4.2 退税计算

| 情形 | 退税金额计算 |
|------|--------------|
| 车辆退回 | 已缴纳税款全额退还 |
| 免税条件消失 | 按使用年限计算补缴，不涉及退税 |
| 不予登记 | 已缴纳税款全额退还 |

### 4.3 退税申请流程

1. 向原纳税地主管税务机关提出申请
2. 提供相关证明材料
3. 税务机关审核
4. 办理退税

## 五、补税规定

### 5.1 免税条件消失后的补税

当免税车辆发生以下情形时，需补缴车辆购置税：

| 情形 | 说明 |
|------|------|
| 转让 | 免税车辆转让给非免税对象 |
| 改变用途 | 车辆用途改变，不再符合免税条件 |
| 改装 | 车辆改装后不再符合免税条件 |

### 5.2 补税计算方法

```
补税额 = 初次办理纳税申报时确定的计税价格 × (1 - 使用年限 × 10%) × 10%

注：使用年限不超过10年，超过10年的按10年计算
```

### 5.3 补税计算示例

| 原计税价格 | 已使用年限 | 补税系数 | 补税额 |
|------------|------------|----------|--------|
| 200,000元 | 2年 | 0.8 | 16,000元 |
| 200,000元 | 5年 | 0.5 | 10,000元 |
| 200,000元 | 8年 | 0.2 | 4,000元 |

## 六、特殊情形规定

### 6.1 进口车辆

| 项目 | 规定 |
|------|------|
| 计税价格 | 关税完税价格 + 关税 + 消费税 |
| 纳税地点 | 进口地或车辆登记地 |
| 证明材料 | 海关进口货物证明书等 |

### 6.2 二手车

| 项目 | 规定 |
|------|------|
| 是否征税 | 不征收（车辆购置税一次性征收） |
| 完税证明 | 交易时需提供原完税证明 |
| 过户 | 办理车辆购置税档案转移 |

### 6.3 车辆改装

| 情形 | 税务处理 |
|------|----------|
| 改装后车辆类型不变 | 不重新征税 |
| 改装后车辆类型改变 | 可能需要补税或退税 |

## 七、违章处理

### 7.1 逾期未纳税

| 违章行为 | 处罚措施 |
|----------|----------|
| 逾期申报 | 从滞纳税款之日起，按日加收滞纳税款万分之五的滞纳金 |
| 未按规定期限缴税 | 同上 |

### 7.2 其他违章行为

| 违章行为 | 处罚措施 |
|----------|----------|
| 虚假申报 | 补缴税款，可能面临罚款 |
| 骗取减免税 | 补缴税款，追究法律责任 |
