# 附加税费计算器产品需求文档 (PRD) - 快速开发版

## 1. 核心目标 (Mission)

开发一款面向财务人员的附加税费计算器网页应用，基于“当期实际缴纳的增值税/消费税税额”与“纳税地点档位”，快速准确计算城市维护建设税（城建税）、教育费附加、地方教育附加，并提供可解释的结果与法规链接。

## 2. 用户画像 (Persona)

**主要用户**：出纳/会计/税务专员，需要在申报环节快速计算并核对附加税费，确保与增值税/消费税缴纳额勾稽一致。

## 3. 功能需求（一小时内可实现）

### 3.1 核心功能

1. **附加税费计算（城建税/教育费附加/地方教育附加）**
   - 输入“当期实际缴纳增值税”、“当期实际缴纳消费税（可为0）”、“纳税地点档位（城市/县城与建制镇/其他）”。
   - 按法规税率计算分项与合计；金额统一保留两位小数。
2. **进口环节提示**
   - 勾选“是否涉及进口环节”仅用于展示提示：进口环节税款由海关一并代征，计算公式不变。
3. **免税/零税率口径提醒**
   - 当“实际缴纳税额”为0或为空时，提示附加税费为0并给出口径解释（不计税基）。
4. **知识库联动**
   - 结果旁提供“公式与条文出处”链接到本工具的“附加税费知识库”。

### 3.2 数据输入/输出

**输入数据**：
- 当期实际缴纳增值税（必填，数字，≥0）
- 当期实际缴纳消费税（选填，数字，≥0，默认0）
- 纳税地点档位（必选：城市/县城与建制镇/其他）
- 是否涉及进口环节（选填，复选框，仅提示用途）

**输出数据**：
- 城市维护建设税（城建税）金额
- 教育费附加金额
- 地方教育附加金额
- 合计附加税费金额
- 简要计算依据与知识库链接

## 4. 技术实现

### 4.1 技术选择
- HTML + CSS + JavaScript
- 纯前端实现，无需后端
- 可复用增值税计算器的样式与组件风格

### 4.2 文件结构
- index.html：主页面
- styles.css：样式文件
- script.js：计算逻辑与校验
- rates.config.json：税率与版本信息（城建税档位、教育费附加、地方教育附加）
- 附加税费知识库：用于条文链接与口径解释

## 5. 界面设计

### 5.1 主要界面元素
- 数据输入表单（实际缴纳税额、纳税地点档位、进口环节提示）
- 计算结果展示区域（分项与合计，公式与条文锚点）
- 知识库链接区域（税率、征收规定、计算方法）

### 5.2 界面布局
- 简洁的单页面设计
- 表单与结果区域清晰分离，移动端友好
- 明确的校验与提示文案（负数/非法字符禁止，0值合规）

## 6. 一小时开发计划

1. **0-15分钟**：创建基本HTML结构和CSS样式；表单字段与结果区占位
2. **15-30分钟**：实现计算逻辑（公式/税率/档位分支），金额格式化
3. **30-45分钟**：加入知识库链接与提示文案（进口环节、免税/零税率口径）
4. **45-55分钟**：完善校验与轻度美化，准备 `rates.config.json`
5. **55-60分钟**：测试与修复（示例用例：增值税=10,000，县城档位）

## 7. 计算公式参考

- 合并税基：`税基 = 实际缴纳的增值税 + 实际缴纳的消费税`
- 城建税：`城建税 = 税基 × 城建税率（城市7% / 县城与建制镇5% / 其他1%）`
- 教育费附加：`教育费附加 = 税基 × 3%`
- 地方教育附加：`地方教育附加 = 税基 × 2%`
- 合计：`合计附加税费 = 城建税 + 教育费附加 + 地方教育附加`
- 金额保留两位小数，四舍五入

## 8. 合规与规则来源

- 税基口径：以“单位和个人实际缴纳的增值税、消费税税额”为计征依据；免税、零税率无应纳税额不计入税基；即征即退情形先行征收后退还或冲减
- 进口环节：增值税/消费税由海关征收时，附加税费按规定一并代征，计算口径与公式一致
- 法规依据：
  - 《中华人民共和国城市维护建设税法》（自2021年9月1日起施行）
  - 《教育费附加暂行条例》及相关修订
  - 《地方教育附加暂行条例》及相关修订
  - 财政部、税务总局相关公告与规范性文件（用于实施口径与征管细则）
- 知识库引用：本工具内的《附加税费知识库》各条目（税率、征收规定、计算方法）

## 9. 后续优化方向（一小时后）

1. 与增值税计算器联动：从增值税计算结果自动读取“实际缴纳增值税”，减少重复录入
2. 批量计算：支持CSV导入与逐行结果导出，标注异常行（非法数值/档位缺失）
3. 版本与政策期：在结果区显示税率版本号与生效日期，来源于 `rates.config.json`
4. 合规旁注：输出“计算底稿”视图（含税基、档位、公式与法规链接），便于留存备查
5. 多地纳税提醒：提示跨区域经营需确认纳税地点档位与适用税率
