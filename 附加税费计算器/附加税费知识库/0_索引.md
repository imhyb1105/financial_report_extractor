# 附加税费知识库索引

本知识库覆盖城市维护建设税（城建税）、教育费附加、地方教育附加的基础规则、税率与计征标准、征收管理、计算方法与示例，依据中国大陆现行法规编写（截至2024年10月）。

- 基础概览与税基：参见《[1_附加税费基础知识.md#一、附加税费概览](./1_附加税费基础知识.md)》《[1_附加税费基础知识.md#二、计税依据（税基）](./1_附加税费基础知识.md#二、计税依据（税基）)》
- 税率与计征标准：参见《[2_附加税费税率.md#一、附加税费税率](./2_附加税费税率.md#一、附加税费税率)》
- 征收与管理规定：参见《[3_附加税费征收规定.md#一、征收与管理](./3_附加税费征收规定.md#一、征收与管理)》
- 计算方法与示例：参见《[4_附加税费计算方法.md#一、基本公式](./4_附加税费计算方法.md#一、基本公式)》
- 法规依据汇总：参见《[1_附加税费基础知识.md#五、法规依据](./1_附加税费基础知识.md#五、法规依据)》

常用入口
- 城建税税率（7%/5%/1%）：《[2_附加税费税率.md#（一）城市维护建设税（城建税）税率](./2_附加税费税率.md#（一）城市维护建设税（城建税）税率)》
- 教育费附加（3%）：《[2_附加税费税率.md#（二）教育费附加税率](./2_附加税费税率.md#（二）教育费附加税率)》
- 地方教育附加（2%）：《[2_附加税费税率.md#（三）地方教育附加税率](./2_附加税费税率.md#（三）地方教育附加税率)》

提示
- 附加税费的计征依据为“单位和个人实际缴纳的增值税、消费税税额”，免税或未实际缴纳部分不计入附加税费税基。
- 进口环节不征收城市维护建设税、教育费附加和地方教育附加；附加税费计税依据与城建税计税依据一致，均以境内环节依法实际缴纳的增值税、消费税税额计算。
