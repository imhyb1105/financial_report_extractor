# 附加税费规定

**版本：2026.03-001**

---

## 一、征收管理规定

### 1.1 征收机关

| 税费名称 | 征收机关 |
|----------|----------|
| 城市维护建设税 | 税务机关 |
| 教育费附加 | 税务机关（随增值税、消费税一并征收） |
| 地方教育附加 | 税务机关（随增值税、消费税一并征收） |

### 1.2 申报缴纳

- **申报周期**：与增值税、消费税的申报周期一致，一般按月（或按期）申报
- **缴纳时限**：与增值税、消费税同时缴纳
- **纳税地点**：与增值税、消费税的纳税地点一致

### 1.3 进口环节规定

- 进口环节**不征收**城市维护建设税、教育费附加和地方教育附加
- 海关代征的增值税、消费税不作为附加税费计征依据

---

## 二、计税依据

### 2.1 基本规定

附加税费的计税依据为**纳税人实际缴纳的增值税、消费税税额**。

```
计税依据 = 实际缴纳的增值税 + 实际缴纳的消费税
```

### 2.2 计入计税依据的情形

- 当期实际缴纳的增值税税额
- 当期实际缴纳的消费税税额
- 补缴以前期间的增值税、消费税

### 2.3 不计入计税依据的情形

| 情形 | 说明 |
|------|------|
| 免税税额 | 依法享受免税的税额部分 |
| 零税率 | 适用零税率且无应纳税额的 |
| 留抵退税 | 留抵退税额 |
| 滞纳金 | 增值税、消费税的滞纳金 |
| 罚款 | 因违法被处的罚款 |

### 2.4 先征后退/即征即退

- 对"先征后退"或"即征即退"的情形，附加税费随同增值税、消费税先行征收
- 退税时按规定相应退还或冲减附加税费

---

## 三、减免优惠

### 3.1 随主税减免原则

附加税费随增值税、消费税的减免而**同步减免**：

| 主税情形 | 附加税费处理 |
|----------|--------------|
| 增值税免税 | 附加税费为零 |
| 增值税减征 | 附加税费相应减少 |
| 消费税免税 | 附加税费相应减少 |
| 出口零税率 | 附加税费为零 |

### 3.2 小规模纳税人优惠

- 小规模纳税人享受增值税减免的，附加税费随同减免
- 实际缴纳税额为零的，附加税费为零

### 3.3 起征点规定

- 增值税未达起征点的，不征收附加税费
- 实际缴纳税额为零的，不征收附加税费

### 3.4 其他减免情形

| 减免类型 | 说明 |
|----------|------|
| 疫情期间优惠 | 按国家临时性政策执行 |
| 特定行业优惠 | 按行业专项政策执行 |
| 区域性优惠 | 按地方政策执行 |

## 四、特殊情形处理

### 4.1 跨区域经营

- 按实际经营地确定城建税适用税率
- 总分机构汇总纳税的，按汇总纳税规定执行
- 建议向主管税务机关确认具体执行口径

### 4.2 代扣代缴

- 代扣代缴增值税、消费税的，同时代扣代缴附加税费
- 以实际代扣代缴的税额为计税依据

### 4.3 税务稽查补税

- 稽查补缴的增值税、消费税，应同时补缴附加税费
- 补缴税额计入当期计税依据

---

## 五、合规要点

### 5.1 申报资料留存

建议留存以下资料备查：

- 当期实际缴纳增值税、消费税税额明细
- 城建税、教育费附加计算底稿
- 税率适用依据（纳税地点证明材料）

### 5.2 勾稽关系检查

确保以下勾稽关系正确：

```
城建税 = (增值税 + 消费税) × 城建税率
教育费附加 = (增值税 + 消费税) × 3%
地方教育附加 = (增值税 + 消费税) × 2%
```

## 六、法规依据

- 《中华人民共和国城市维护建设税法》（自2021年9月1日起施行）
- 《教育费附加暂行条例》
- 《地方教育附加暂行条例》
- 财政部、税务总局关于贯彻实施《城市维护建设税法》若干事项的公告
- 各地方税务局征收管理公告
