# 项目测试报告（税法合规与数据准确性核查）

日期：2025-11-11  
范围：`附加税费计算器`、`增值税计算器`（d:\ai\ai编程\财税小工具）

## 概述
- 目标：检查两个计算器是否符合最新法律法规，数据与计算是否正确；识别合规风险并提出修复建议。
- 方法：代码与配置审查（HTML/CSS/JS/JSON）、知识库条文比对、政策核查与交叉验证、抽样计算复测。
- 结论摘要：
  - 增值税计算器整体税率架构与小规模优惠（3%减按1%至2027-12-31）一致，存在“出售不动产5%征收率未提供”的功能缺口；免税门槛（10万元/月）已在计算逻辑中实现，但季度累计（30万元/季）仅以说明提示呈现。
  - 附加税费计算器税率正确（城建税7%/5%/1%，教育费附加3%，地方教育附加2%），但“进口环节口径”与最新法规不一致：当前实现会将进口环节两税纳入税基并计算附加税费，且提示文案错误；按现行口径，进口环节不征收城市维护建设税，教育费附加与地方教育附加计征依据与城建税一致（不含进口环节两税）。

## 法规基线（核查与引用）
- 增值税（VAT）税率体系：13%（货物等）、9%（交通运输/建筑/租赁等）、6%（现代服务等）、0%（出口/零税率）。
- 小规模纳税人优惠与免税：
  - 对适用3%征收率的销售，减按1%征收至2027-12-31；适用3%预征率的项目，减按1%预缴（政策延续明确）。
  - 月销售额不超过10万元（或季度不超过30万元）免征增值税（按期纳税人适用）。
  - 引用：
    - 交通运输部政策指引（2023-08，延续至2027-12-31）：https://zizhan.mot.gov.cn/sj2019/caiwusjs/faguizd_css/202308/t20230825_3899102.html
    - 国家税务总局问答（2023-01-11）：https://www.gov.cn/zhengce/2023-01/11/content_5736338.htm
    - 地方税务12366热点问答（2024-02）：https://anhui.chinatax.gov.cn/art/2024/2/7/art_9438_1150127.html
- 城市维护建设税（城建税）与教育费附加、地方教育附加：
  - 城建税税率：市区7%、县城与建制镇5%、其他1%。
  - 计征依据：以依法实际缴纳的增值税、消费税为税基；教育费附加与地方教育附加计征依据与城建税一致。
  - 进口环节：对进口货物或境外单位/个人向境内销售劳务、服务、无形资产缴纳的两税税额，不征收城市维护建设税；教育费附加与地方教育附加遵循一致的计征依据（不含进口环节两税）。
  - 引用：
    - 国税总局征管公告解读（2021-09-02）：https://www.gov.cn/zhengce/2021-09/02/content_5634938.htm
    - 国税总局公告原文（2021-09-02）：https://www.gov.cn/zhengce/zhengceku/2021-09/02/content_5634919.htm
    - 财政部/税务总局公告2021年第28号（计税依据一致）：http://szs.mof.gov.cn/zhengcefabu/202108/t20210825_3747865.htm

## 问题清单与风险评级
### P0（必须修复，合规高风险）
1. 附加税费计算器的“进口环节口径与计算”错误。
   - 表现：勾选“进口环节”后仍将进口两税纳入税基并计算城建税、教育费附加与地方教育附加；提示文案亦错误。
   - 法规口径：进口环节不征收城市维护建设税；教育费附加与地方教育附加计征依据与城建税一致（不含进口两税）。
   - 影响：对进口业务用户会产生附加税费的错误计算与误导，存在较大合规风险。
   - 建议：
     - 计算层面：`isImport === true` 时，将进口两税（或用户标识为进口的两税）剔除出税基；或直接提示“进口环节不计征上述附加税费”，并不进行计算。
     - 文案修正：更正为“进口环节不计征城市维护建设税；教育费附加与地方教育附加计征依据与城建税一致（不含进口环节两税）”。

### P1（应修复，功能与口径一致性）
2. 增值税计算器未提供“小规模纳税人出售不动产 5%”征收率选项。
   - 表现：小规模征收率仅有 3% 与阶段性 1%。
   - 影响：涉及不动产出售的场景无法直接用工具精确计算。
   - 建议：在“小规模纳税人”征收率下拉中补充 `5%` 选项，并在说明中明确适用范围与凭证要求。

3. 小规模纳税人季度免税（30万元/季）未实现累计逻辑。
   - 表现：当前实现仅按单次输入判断 `≤10万元` 的免税；季度累计以提示说明呈现。
   - 建议：提供简易季度累计输入或在结果说明中提示“季度累计口径由用户手动校核”。

### P2（建议修复，文档/配置治理）
4. 附加税费知识库文案与进口环节口径矛盾。
   - 表现：索引与征收规定中写明“进口环节由海关一并代征城建税、教育费附加与地方教育附加”，与现行口径不符。
   - 建议：统一更正为“进口环节不征收城建税；教育费附加与地方教育附加计征依据与城建税一致（不含进口环节两税）”。

5. 城建税计税依据的细项（免抵税额、留抵退税扣除）未在附加税费计算器中体现。
   - 表现：仅提供“当期实际缴纳两税”输入；未体现免抵税额与留抵退税的口径差异。
   - 建议：在知识库与输入说明中提示“按申报口径的‘依法实际缴纳两税’填报，免抵税额与留抵退税按规定处理”。

6. 阶段性政策有效期未在增值税计算器中进行约束或提示（1%优惠至2027-12-31）。
   - 建议：在 UI 中显示有效期提示；或通过配置（如 `vat.config.json`）驱动有效期展示与默认值。

## 结论
- 增值税计算器：税率架构与优惠口径正确，功能上需补齐不动产5%征收率；免税门槛已实现，季度累计以提示方式处理。
- 附加税费计算器：税率正确；“进口环节”计算与文案不符合法规，需立即修正以消除较大的合规风险。
- 知识库：需统一进口环节的口径表述，确保与法规一致。
